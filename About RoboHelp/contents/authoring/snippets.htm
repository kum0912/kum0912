Snippets

Content Reuse for Images, Tables, Text and More

Snippets can contain text, variables, images and suchlike. They are especially useful where you need to repeat the same content in many topics as any change only has to be made once.

You can add a snippet within a snippet and you can add a snippet inline.

Creating Snippets

  1. Click the Snippet icon in the project toolbar on the left and any snippets already created will display alongside in the Snippet Panel or select some text, right click and select Create Snippet
  2. Hover over a snippet and click the Options icon to edit, delete or see the properties of a snippet.
  3. Click the + icon to add a snippet

  4. When you click Add, the New Snippet dialog displays.

In Classic versions of Adobe RoboHelp no CSS is applied to a snippet, it takes on the CSS of the topic in which it is used. In Adobe RoboHelp the default css set in Project Settings is applied. If you want Snippets to work the old way, when you create a Snippet go to Topic Properties and remove the link to the CSS.
It is important that you remove the link to your default css if you will be using the snippet inline, unless it is the same css as used in the topic.

  1. Set the fields as required and click Create to open a blank Snippet in the Design Editor.
  2. Set up the snippet as required. If you want to apply a specific CSS to the snippet rather than allowing it to inherit the CSS of the topics it is used in, right click and select Topic Properties.
  3.  Save your changes.

You can also select text in a topic, right click and select Snippet > Create to set up a new snippet from existing content.

Adding Snippets to a Topic

You can either drag a snippet from the Snippet Panel or press CTRL 2 and double click there.

You can insert the snippet within a paragraph or you can use it as a block snippet by inserting it in a new line.

 Snippets that contain block elements such as a table, cannot be used inline. They will get added as a block snippet.

Snippet Properties

While you are working on the Snippet, you will see the Properties panel alongside allowing you to modify the appearance.

Snippet Folders

When you create a Snippet it is automatically saved in Assets > Snippets. View that folder in the Contents Panel and you can then create folders and organize your snippets as you wish. The Snippets Panel is a flat view of all Snippets in the project, no matter where they are saved.

Snippets or Variables

In previous versions of Adobe RoboHelp snippets could only be added as a paragraph. That meant where you wanted to repeatedly add the same string inline, you had to create a variable and use that. Now using an inlne snippet would be the correct solution. Variables are now for where you want to use different values in different outputs such as $ and £ signs and values.
