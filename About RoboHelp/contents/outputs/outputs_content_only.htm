Content Only Output

Prior to Update 4 the Content Only Output covered Salesforce, Servicenow, Zendesk and XHTML. From Update 4 onward the Content Only output is for XHTML only. Salesforce, Servicenow and Zendesk have been moved to a new output type known as Knowledge Base which now also includes Zoho.
