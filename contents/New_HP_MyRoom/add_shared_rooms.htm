Add SharedRooms

HPE MyRoom SharedRooms are a special permission available as a purchased product. A product key is used to grant the ability to create shared rooms. SharedRooms are used when many people need continuous access to the same large group of rooms, for instance, on a support or crisis desk to keep track of events over a long period and with many shifts of people. In order to set up SharedRooms, you need to have a process mailbox already set up, a product key to create more than 10 rooms and a product key to share a group of rooms with others. Contact the HPE MyRoom Help Desk with questions about purchasing this option.

 

Once you have the prerequisites, start the process by creating an HPE MyRoom account for the process mailbox.

 

  • Go to the My account page.
  • Create an account
  • The email address will be the process mailbox address.
  • The telephone number will be the process mailbox owner's phone number.
  • The company name will be the process mailbox owner's name and email address.

 

  • Those who will have access to SharedRooms will already have received two emails from MyRoom@hpe.com:

 

    • Product key to create more than 10 rooms (Subject: Creating additional rooms).

    • Product key to share a group of rooms with others (Subject: Sharing a group of rooms).

 

  • Use both Product keys to apply both upgrades to this new account.

   

Create rooms associated with the process mailbox

 

  • If not already done, download and install HPE MyRoom from the MyRoom Download page.
  • Double click the desktop icon created to open the HPE MyRoom tray.
  • Log in with the Process Mailbox name and password.
  • Click on the Create room button.
  • Create all the rooms that need to be shared.

   

Grouping rooms

 

  • Create a group for rooms that will be shared.
  • Move the rooms to the group that will be shared by drag and drop.
  • If needed, create a different group for another set of shared rooms.
  • Move the 2nd group of rooms to the 2nd group.

 

A room can only belong to one group. More than one group can be created and shared with everyone or select users.
