See more/See less - event

The See more/See less icons function to expand or contract the list of options available in the tool list. See this page for the basic tools available in Chat: additional tools are discussed on the following pages.

 

 

Icon or Area

Name

Comments

Hand up

Click to raise a hand for attention in the room. See the Raise hand page for more details.

Quick key

Click to generate a quick 6-digit key to be used to access the room. The key is displayed to the right of the event name at the top of the room. See the Quick key page for more details.

Show recording toolbar

 

Click to open the recording toolbar. See the Record your meeting page for more details.

Join audio bridge

Click to set up and join an audio bridge. See the Phone Bridge page for more details.

Add or remove buttons

Adjust the buttons displayed on the See more options. See the Add or remove buttons page for more details.
