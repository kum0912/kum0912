Share an application

A user may share to the others in the room a region of their desktop, an entire desktop or, for Windows and Linux users, an application.

 

After clicking the Sharing icon (), the Sharing menu appears as shown at right.

 

 
Choosing An application gives you a list of all the open applications on your machine. Choose the application to share.  

   
The shared application shows for others in the room and they will see whatever is being done in that application.      

   
When you share an application, you can tick to include your computer audio from the shared application to others. This will enable others to hear audio from a video you are sharing.

   

If another application is brought into focus in front of the shared application, others will see a grayed out area saying Non-Shared Area.

 

NOTE: Start your PowerPoint slide show before sharing the PowerPoint application to avoid others seeing the Non-Shared Area instead of the slide show. PowerPoint Slide Show acts as a separate instance of PowerPoint and opens in another window, so if you start App Sharing before the slide show, users will see a Non-Shared Area.

 

 

 

Ubuntu and Sharing desktop

 

Ubuntu 18.04 (LTE) resolves a previous Application Sharing issue.

 

Ubuntu 17.10 users will need to switch Ubuntu to use Xorg desktop which is available on 17.10 as part of the installation package. You can switch to the Xorg desktop by clicking on the gear icon near the Linux sing in button and choosing the correct desktop. If you have both Unity and Xorg, the dialog will show three (3) options.
