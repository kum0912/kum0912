Moving to breakout rooms

For users with MyRoom University and those who have purchased the Breakout rooms option, you can use breakout rooms to facilitate smaller discussion groups, team collaboration and design. One presenter needs to have the Breakout rooms option.

 

You will see all the choices available in this menu if you have purchased MyRoom University. If you have the MyRoom Assist option along with MyRoom Premium or MyRoom University, you will see the Copy Clipboard, Get remote system information, Remote control, Tables and optionally, the Waiting room, options. Waiting room is only available in the MyRoom Assist option. Please go to the MyRoom Features page for a complete description of the features available in each option.

 

Note: Mobile and VRG attendees (iOS and Android) cannot enter breakout rooms.

 

If you have only a small number of participants, you can move each individually into breakout rooms.

 

  • Right click on the name in the People panel.
  • Choose To breakout...
  • Click the room to which you want to move the participant.

   

Your participant is placed in the breakout room. You can repeat this for each person.  

 

Everyone in a Breakout room is a presenter with some limitations.  

 

The following are not available:

  • Send invitation
  • Contacts
  • Configure room for participants
  • Eject from room
  • Eject all
  • Make Participant/Presenter
  • Request remote control
  • To breakout...
  • Quick keys

 

HPE MyRoom audio for each person is transferred from the main room into the breakout room so that anyone remaining in the main room does not hear any conversation from the breakouts.  

 

You can also your participants to navigate themselves by selecting the Allow participant room navigation box.

   
If you have a large number of people in the room and want them placed in breakout rooms in a faster fashion, you can use the Auto-Assign participants option. Right click in the Breakout rooms panel and choose Auto-assign participants.

   
   
This takes your participants and places them one by one in each of the breakout rooms until the last room is reached when the auto assign starts again with the first room until all participants have been placed.

   
If the presenter has chosen to allow participant navigation (see above), you as a participant can right click on a breakout room and choose Enter breakout to enter the breakout room.
   
If you have an old version of MyRoom and are dragged into a breakout room by a presenter, you would not know you are in a breakout room. The message in the example at right will appear to warn you to update your MyRoom version so that you can see and work in your breakout room. Click Update now to install the latest version of MyRoom.
