Breakout rooms

For users with MyRoom University and those who have purchased the Breakout rooms option, you can use breakout rooms to facilitate smaller discussion groups, team collaboration and design.

 

You will see all the choices available in this menu if you have purchased MyRoom University. If you have the MyRoom Assist option along with MyRoom Premium or MyRoom University, you will see the Copy Clipboard, Get remote system information, Remote control, Tables and optionally, the Waiting room, options. Waiting room is only available in the MyRoom Assist option. Please go to the MyRoom Features page for a complete description of the features available in each option.

 

Note: Mobile and VRG attendees (iOS and Android) cannot enter breakout rooms.

 

When a presenter clicks the Tools to manage people and room features button , the menu at right appears.  

 

Any presenter can click on Show breakout rooms.

   

The Breakout rooms panel then appears under the People panel, if that is open.  

 

It is good practice when training or with a scheduled large meeting to set up your Breakout rooms before the event starts. You can have several sets of Breakout Rooms, named for their functions. As an example, in an event where you want discussion on four different topics to go on at four different times during the event and you want to keep the content developed in those rooms separate, create four sets of Breakout rooms called Principles, Material, Function and Team.

   

To create a set of breakout rooms, click on <add breakout room> to bring up the add breakout room window. Type a name and change the number of breakout rooms to create, if necessary.

 

The Breakout name is limited to 64 characters long, including spaces, and cannot contain the character '&'. Click on OK to create the rooms.

   

Your breakout rooms are created with the name you specified and a number. The number in brackets shows the number of people in the breakout room, while the names of each person are listed underneath as long as the room name is expanded.  

 

Note that although rooms are numbered sequentially, you can move rooms above or below others using drag and drop.

   
Once your breakout rooms are set up, you can add more breakout rooms by right clicking on the Breakouts heading and choosing Add breakout from the menu. Add the new name and number of rooms and click OK.

   

Your new breakout rooms are created and placed in order in the list.  

 

Note that although rooms are placed and numbered sequentially, you can move rooms above or below others using drag and drop.
