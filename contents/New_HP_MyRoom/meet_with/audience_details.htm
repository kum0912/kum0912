Audience Details

Audience Details can be viewed by anyone in a chat or event. You can use the Audience Detail to target your presentation to your audience or participants.

 

The Audience Detail window is displayed by either right clicking in the People panel and choosing Audience Details or, as a Presenter, clicking the Tools to manage people and room features button in the People panel.   Audience Details are available to anyone in a chat or event. In a chat, the Role column is not present.   On the right hand side of this window is the Save Audience Details button. Click this button to save a snapshot of the details in an HTML file on your local computer. This snapshot is not cumulative but only reflects the details as of the moment the button is clicked.   You can click on any column header to sort or resort by that column, in ascending or descending order.    

   

Icon

Name

Description
Name Displays the name used when entering the event.
Hand up Shows who has raised their hand.
Yes/No Vote Displays the voting preference of each person.
Speaking You can see who is speaking.
Video You can see who has a camera turned on.
Role Displays the presenter icon for a Presenter. Not present in a chat as there is no Presenter/Participant differentiation in a chat.
Blocking Private chat Shows who in the room has used the Block Private chat function.
Network health indicator Indicates the network health for each person in the room.
Platform Shows the operating system in use.
Version Displays the version number of HPE MyRoom in use by each person. If someone has an issue in an event and a presenter sees they are on an older version, a first step might be to upgrade to the latest version of HPE MyRoom.
Language Shows the language chosen by the individual on logging in to the event. Helpful for presenters to be able to gauge their own speaking speed and use of colloquialisms.
Timezone Displays the timezone of each person's local computer.
Save Audience Detail Click to Save Audience Details to an HTML file on your local computer.
