Annotation tools

Annotation tools are drawing and text tools used to draw attention to content. Annotation tools can be used on any content item where they appear, such as a whiteboard, picture or PowerPoint presentation, to add diagrams or draw attention to a portion of the center stage area. Participants do not see Annotation tools until the presenter gives them access through the Tools to manage people and room features .

 

Preserve annotations on slides from one room to another by dragging and dropping the slides from room to room, dropping them in the content area. If you drag a content item to the desktop or export it, the annotations become part of the slide.

 

As soon as you select a content item in the content panel, the menus appear above the Center area. Click the See more icon  to expand the annotation tools. As a presenter, you can use the tools in the Tools menu as well.

 

If you have granted access to Drawing tools in the Configure room for participants option in the Tools to manage people and room features list , your participants can also use the Tools menu to annotate content.

 

 
The following are the Annotation tools available in HPE MyRoom from either the Annotation toolbar area or the Tools menu:

 

 
Pointer tool: used to change the shape or look of the pointer for everyone in the room. Each presenter can choose their own pointer. Click on the tool to see the options.

Selection tool: used to select an individual annotation to  be moved, changed or deleted.

Pencil tool: used for freehand drawing on a content item.

Calligraphy tool: use for script calligraphic writing on a content item.

Highlighter tool: useful for highlighting portions of a content item.

Text tool: used to insert text onto a content item.

Oval tool: can be used to draw circles or ovals on a content item.

Filled oval tool: draws filled circles or ovals on a content item.

Square tool: can be used to draw squares or rectangles on a content item.

Filled square tool: used to draw filled squares or rectangles on a content item.

Line tool: draws straight lines on a content item.

Color tool: changes the color of any Annotation tool to your liking.

Eraser tool: Erases all annotations on the content item.
