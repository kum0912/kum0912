Using annotation tools

Annotation tools are easy to use to draw attention to your content. The Pencil tool will be used here as an example for the Highlighter, Pencil, Oval, Rectangle and Line tools. The Selection, Text and Eraser tools will be explained on the following page.

 

If you want to change the color of a tool, click on the Choose color icon. By default, the Highlighter is yellow and all other tools are light blue.

 

 
The Select color window appears. You can choose an existing color or create a new one for your Annotation tool. Click on OK when completed.

 

 
Click on the Pencil tool in the Annotation toolbar.

 

 
Use your pencil to mark on the content object by left-click and dragging the mouse.

   
A presenter can increase the size of an individual annotation by  right clicking the annotation on the whiteboard and choosing Make larger (Ctrl +).
   
The annotation becomes larger. Repeat until you get the size you need. Changes here reflect on the screens of others in the room.
   
You can repeat the steps choosing Make smaller (Ctrl -) to reduce the size of the annotation. 
 

 
Annotations remain on the Content item until a presenter erases them.
