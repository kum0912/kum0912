Tools to manage people and room features - event

You can use the Tools to manage people and room features icon in the People panel to customize how your room is available to the people in it. In an event, only presenters see the button. You can also right click on a name in the People panel for more extensive options particular to that one person.

 

You will see all the choices available in this menu if you have purchased MyRoom University. If you have the MyRoom Assist option along with MyRoom Premium or MyRoom University, you will see the Copy Clipboard, Get remote system information, Remote control, Tables and optionally, the Waiting room, options. Waiting room is only available in the MyRoom Assist option. Please go to the MyRoom Features page for a complete description of the features available in each option.

 

When you as a presenter select a name in the People list and right click on it, the menu at right appears. If you have not selected a name or if you choose the icon, a reduced version of the same menu appears.  

 

The following pages explain the additional features available on this menu from those on the same menu in  a chat. See the Tools to manage people and room features for the chat functionalities.
