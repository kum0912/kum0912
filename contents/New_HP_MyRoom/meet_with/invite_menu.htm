Invite menu

The Invite menu gives you ways to invite others into your event directly, by email or by using different keys. You can also schedule events using the Invite menu.

 

 

 

Icon or Area

Name

Comments

Invite contact

Click to invite a contact to the event directly. See the Invite contact page for more details.

Invite by email

Click to invite someone through email. See the Invite by email page for more details.

Quick key

Click to generate a Quick key. See the Quick key page for more details.

Presentation keys

Click to generate Presentation keys. See the Presentation keys page for more details.

Persistent keys

Click to generate Persistent keys. See the Persistent keys page for more details. Visible only when room is not in use by an event.

Events

Click to create an event. See the Events page for more details. Visible only when room is not in use by an event.
