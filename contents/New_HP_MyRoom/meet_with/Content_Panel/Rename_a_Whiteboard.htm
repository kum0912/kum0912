Rename a content item

Rename an item in the Content panel if needed. As an example, rename a whiteboard to reflect the notes written on it.

 

If the Content panel is not displayed, click on the Show content panel button.

 

 

Right click the content item to be renamed in the Content panel.

 

Select Rename from the menu.

 

 
Use the Backspace key to erase the current name and then type in the new name for the content. Click on OK to change the name or Cancel to abort the operation.

 

 
The content is renamed in the Content panel.
