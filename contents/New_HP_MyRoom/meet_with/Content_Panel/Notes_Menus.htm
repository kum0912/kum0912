Notes menus

The Notes Menus are used for saving and exporting files as well as for formatting. Only presenters see the Notes menus.

 

The Notes menus appear for everyone except participants as soon as a note is selected in the content panel.

 

 
The following are the Notes menus available in HPE MyRoom:

 

 
File menu

 

 

Edit menu: these are the regular edit functions: Undo, Redo (only available after an Undo), Cut, Copy and Paste (only available after Cut or Copy).

 

 
Format menu: these are regular text formatting options for text, line justification and color.

 

 

Bullets menu: Changes the current and subsequent lines to the list style selected.

Click on the picture to access help for using these tools if the picture is hyperlinked.
