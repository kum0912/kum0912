Notes tools

You use the Notes tools to format the note. The Notes tools are a standard group of tools available for formatting in many applications.

 

The Notes tools appear for everyone except participants as soon as a note is selected in the content panel.

 

 
The following are the Notes tools available in HPE MyRoom:

 

 
Undo: erases the last text entry.

Bold: changes the text to bold.

Italic: italicizes the text.

Underline: underlines text

Left align: aligns text to the left on the note.

Center: centers the text on the note.

Right align: aligns text to the right on the note.

Justify: justifies text across each line on the note.

Text Color: changes the text color.

Font and font size: changes font and font size.

These tools are used in the usual manner. Changing any of the above changes the text that is entered afterwards. You can also select a portion of text and then apply any of the effects to it by clicking on the appropriate button.  
 

 
Take Notes editing button: shows other presenters who is entering text into the notes. A presenter who wants to write in the note has to click on the button to take control of the note. The presenter can also give up control of the note by clicking again so no name appears. The tooltip shows who is editing the note as well when a presenter hovers over the button.
