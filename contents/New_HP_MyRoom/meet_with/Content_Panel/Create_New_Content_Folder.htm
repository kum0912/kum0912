Add new content folder

 

Content can be organized using folders. Adding a folder is the same for both chats and meeting rooms, however, you must put content in a folder to be able to share it with participants in a meeting. Add a new folder by right clicking in the content panel and choosing Add, then Folder from the menu. You can also click the Open content file menu in the content panel to get the same menu.

   
 Enter a name for the new folder and click OK.          

   
Drag and drop files into the new folder as desired or right click on the new folder and choose Import. Navigate to the file and click Open.
