Description of Event room

The HPE MyRoom Event room can be used for scheduled meetings or chats, allowing collaboration, written chat, file transfer and sharing, use of web cameras and microphones as well as desktop sharing. It is a dynamic and powerful tool for meetings and groups to use. An event room has a larger variety of options than the chat room. Chat room options can be seen here. The additional event room features are described below and in the following pages.
