Roles

Roles are created to simplify the process of assigning rights to a user. Roles control users' access across the product. The roles define the access rights such as the ability to create collection, modify collection, enrich collections, etc.

Creating Role

Each user is assigned one or more roles that are required by the individual. There is no restriction on the number of roles that can be created in the system or assigned to a user.

To create Roles:

Click Setup.

Home page

Click Roles.

Setup Page - Roles

System is redirected to Roles page. Click CREATE.

Roles page

Fill in Name, Description and select ‘Yes’ or ‘No’ from the drop-down of “Is this a Business Partner Role”. Click SAVE.

Note: Business Partner Role is applicable only for Farmer Connect and Supplier Connect apps. For clients who do not have business partner role, the value will be ‘No’.

Create Roles page

‘Role created successfully’ pop-up will be displayed.

Role creation success pop-up

Newly created role will be displayed in the Role Listing page.

Roles Listing Page

Modifying Role

User can modify/edit a role at any given point in time.

To modify/edit a Role:

Click the Roles icon. The Roles list page will be displayed.

Click icon against the Role you wish to modify.

Select role to be modified

Modify-Role pop-up will appear.

Modify Role

Enter the details and click Save. The role is modified and displayed in the Roles List page.

Copying a Role

The system allows users to Copy a Role. By copying a role, all existing permissions are copied.

To copy a role:

Click the Roles tile. The Roles listing page will be displayed.

Click icon against the Role you wish to Copy.

Copy - Roles Page

A Copy Role Window is displayed.

Copy Role Page

Enter Name and Description in the text boxes.

Click Save. The Role is copied with all existing permissions of the original role and displayed in the Roles listing page.
