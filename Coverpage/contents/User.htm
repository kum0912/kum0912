User

Admin user can create a user and associate a role to that user.

To create a user:

Click the Menu icon and click SETUP.

Home page

System is redirected to Setup page. Click User.

Setup Page

System is redirected to Users page. Click CREATE.

Users page

The Create User page is displayed. All the fields in this screen must be filled.

Create User Page

Enter the details in User Name, Name, Email.

Mark the check box ‘Consent has been taken from user to use their emaild id for user if creation’.

Select ‘Include Business Partner Roles’ under Access & Role section in order to include business partner roles apart from the roles mentioned in the drop-down list.

Note: Business Partner Role is applicable only for Farmer Connect and Supplier Connect apps. For clients who do not have business partner role, the value will be ‘No’.

Select required roles from the drop-down list in the Access & Role Section. Multiple roles can be selected, if necessary.

Click Save. The user is created and added to the Users list page.

Modifying a User

Users can be modified from the User List page.

To modify a user:

Click the Menu icon and click SETUP.

Home page

System is redirected to Setup page. Click User.

Setup Page

Select the user you wish to modify and click on the icon to edit/modify.

Users page

Modify User screen will be displayed.

Modify User

Modify the details you wish to modify and click Save.

Note: Business Partner Role is applicable only for Farmer Connect and Supplier Connect apps. For clients who do not have business partner role, uncheck ‘Include Business Partner Roles’.
