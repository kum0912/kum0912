About Font Awesome in RoboHelp

Font Awesome (FA) allows you to use Scalable Vector Graphics (SVGs) in your Web pages that can instantly be customised using the power of CSS. This is really useful for adding responsive icons or spot illustrations that will look good at any resolution, across all devices.

The easiest way to implement FA icons is by copying the FA icon's embed code into your HTML and then controlling the style of the icon by targeting it with some CSS.
