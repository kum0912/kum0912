Implementing SVG icons with Font Awesome

Each Font Awesome (FA) icon has a unique embed code that you can copy direct from the FA website. This can be referenced in your pages by placing it inside a HTML element such as <i> or a <span>. The style of the icon can then be customised by applying your own styles. For example, you may want to change the colour or size of the icon.

To insert a FA icon:

  1. Go to https://fontawesome.com/start.
  2. Enter your email to create a Font Awesome kit and hit Send Kit Code. You'll receive an email, from which you'll need to verify your email address and then create a Font Awesome account.
  3. Copy the Kit Code into the <head> of your topic (or master page).
  4. Next, go to https://fontawesome.com/icons?d=gallery&s=solid&m=free and find the desired icon.
  5. Copy the desired icon's code and add it into your topic using a HTML element, like <i> or <span>. For example:
    • <i class="fas fa-question-circle"></i> OR
    • <span class="fas fa-question-circle"></span>
  6. In your style sheet, create a class to style the SVG icon. For example, like .fa-icon below.

.fa-icon {
 font-size: 128px;
 color: firebrick
}
