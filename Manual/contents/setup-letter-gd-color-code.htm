Setup > Marks & comments > Letter grade groups and colour coding 

Overview

 

Clicking GO will bring the user to the following interface. 

 

 

There will be one card for each letter grade group that has been defined. In the header of each card will be the following:

 

  • Letter grade group title
  • Up and down arrows - only enabled if a letter grade has been selected (see section on moving letters around)
  • Copy symbol (see section on copying letters from another group) - icon should not appear if the letter grade group has the box checked for "This letter grade group is used for the provincial report card"
  • Add symbol - icon should not appear if the letter grade group has the box checked for "This letter grade group is used for the provincial report card"
  • Edit symbol
  • Collapse/expand arrow

 

 

Within each letter grade group card the following fields will be identified: 

 

  • Check box - to select a specific letter grade
  • Letter
  • Letter title - will show the English title if the screen language is English or French if the screen language is French
  • Numeric equivalent
  • Lowest numeric mark
  • Colour - if defined, the background and foreground colours will show in the column; if not defined, the background will be transparent and the foreground will be black
  • Range - calculated field; shows the actual range that the letter will represent

 

 

An 'Add' FAB will be accessible at the bottom of the list of letter grade groups which can be used to add a new letter grade group.

Moving Letters Around

To define the order of the letters in a group, select the letters that need to be moved. Each letter will need to be selected and moved one at a time. Selecting a letter will cause the up and down arrows to become enabled in that letter grade group's header. 

 

When clicking the up or the down arrow, the selected letter will get moved up or down accordingly. 

 

Please make sure to save and refresh the list right away when letters are being moved around. 

 

Editing/Deleting Letter Grade Groups

On an existing letter grade group, click on the pencil 'edit' icon to access the following interface. 

 

Selecting the Edit option will open a standard edit form. 

 

On the Edit form, show the DELETE icon in the app bar. 

 

The following fields will be visible, just like the Add form:

 

1. English/French: Default screen language. This controls whether the user is entering the English or French version of a field that has unique values for each language. Fields with underlines have unique values for each language.  

 

2. Identifier: alphanumeric field. Identifier in English or French must >0 characters in order to save, max character count = 5

 

3. This letter grade group is used for provincial report card: check box


4. Letter grade group title*: alphanumeric field. Time frame title in English or French must be >0 characters in order to save, max character count = 50
 
5. Letter grade group description: show English description if screen language = English or French title max length 255, not required 

 

6. Allow numeric (percentage) marks: check box

 

7. Allow in markbooks: check box 

 

8. Denominator: max character count = 3 

 

CLOSE or X will close the details form and return to the table (at the same page where user was).

 

 

One can tab over to NUMERIC RANGES, only if the box for 'Allow numeric' has been checked.

 

 

Selecting the back arrow on the main app bar should prompt the user to save edits. Standard prompt should appear:

 

 

The garbage can icon will be available in the top right, but will only be enabled if this letter grade group is not in use in the mark-entry definition for any term/topic.

 

When selected, a warning will appear to confirm you want to delete the record from the table. The DISCARD button will remove the record from the table. CANCEL will retain the record in the table.

 

 

Defining Numeric Ranges

On the NUMERIC RANGES tab, the following interface is available. 

 

 

Row-click on the numeric range to bring up the edit screen where the garbage can icon can be accessed to delete the range, or additional changes can be made. 

 

The lowest numeric mark is required. SAVE will be disabled if the lowest numeric mark is removed.

 

 

Adding Letter Grade Groups

To add a new letter grade group, click the Add FAB.

 

 

It should show the following interface. 

 

 

1. English/French: Default screen language. This controls whether the user is entering the English or French version of a field that has unique values for each language. Fields with underlines have unique values for each language.  

 

2. Identifier: alphanumeric field. Identifier in English or French must >0 characters in order to save, max character count = 5

 

3. This letter grade group is used for provincial report card: check box


4. Letter grade group title*: alphanumeric field. Time frame title in English or French must be >0 characters in order to save, max character count = 50
 
5. Letter grade group description: show English description if screen language = English or French title max length 255, not required 

 

6. Allow numeric (percentage) marks: check box

 

7. Allow in markbooks: check box 

 

8. Denominator: max character count = 3 

 

NUMERIC RANGES tab will be disabled until all required fields are filled in. 

 

The SAVE button will only be enabled if all required fields are filled in. 

 

Copying Letters From Another Group

In the main grid of letter grade groups, the copy icon will not appear if the letter grade group has the box checked for 'This letter grade group is used for the provincial report card'.

 

 

Clicking the copy icon in the header of the letter grade group, a pop-up menu will appear as shown below. The drop down menu will list all of the other letter grade groups that have been defined and can be copied. 

 

When one of these groups is chosen, letters will be copied with their full definitions from the selected letter grade group into the current one. 

 

DO NOT copy in duplicate letters. If a letter already exists in this letter grade group, leave it as is and do not copy the other one in from the other letter grade group. 

 

Letters will appear in the letter grade group lists right after they have been copied over. 

 

 

Editing/Deleting Letters

In the main grid of letter grade groups and letters, row-click on each letter in the grid to produce the following options: 

 

 

The edit screen should look as shown below: 

 

 

 

1. Language radio buttons: English and French. Default to screen language.  

 

2. Text box for ‘Letter’ – required. Allow 5 characters maximum please.  

 

3. Text box for Letter title – allow 100 characters please. Shows the English title or French title depending on the radio button selection.

 

4. Colour-picker field for Foreground colour.

 

5. Colour-picker field for Background colour. 

 

6. Sample of what the letter is going to look like in mark-entry.

 

7. Numeric field for Numeric equivalent – only positive numbers

 

8. Numeric field for Lowest numeric mark – only positive numbers 

 

PREVIOUS and NEXT buttons will allow one to step between letters in this group. PREVIOUS is disabled for the first letter in the list and NEXT is disabled for the last letter in the list.

 

SAVE button – saves changes to the letter being edited, closes this window and goes back to the list of letter grade groups and letters.  

 

CANCEL button – closes this screen without saving anything.

 

Deleting or Changing Letters Already Used

Marks cannot be changed or removed if they are being used in students' marks.

Sharing to Schools

Letter grade groups should automatically be shared to the schools that need them, based on the shared term and topic mark entry definitions.
