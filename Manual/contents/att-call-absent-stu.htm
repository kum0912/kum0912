Attendance > Call home > Absent students

 

Overview

As soon as attendance is entered, the office should examine the absent students, and then follow the appropriate school protocol for determining the reason for the student’s absence. For younger students with unexplained absences, most schools are required to contact the student’s family.  

 

The Absent students option may be used to

 

  • identify the students who have unexplained attendance today (or any date),
  • list all students who have any type of attendance entered today (or any date).

 

If a student’s attendance is recorded as Absent (e.g. no specific attendance code was entered), it is considered an unexplained absence.

 

Some schools actively follow up with all students, regardless of their age, to determine the reason for the absence. From the list of students, select any student to see additional information to assist with contacting the family and updating attendance information.

 

From the main menu, choose Attendance > Call home. This will take you to the Call home section of the Attendance page where you will find the Absent students card:

 

 

Click GO to access the Absent students page.

Absent students

When the Absent students card is selected, it will automatically default to show today’s unexplained absences. An unexplained absence is an attendance entry of Absent with no reason provided. For example, students with the following attendance code examples would be excluded because you know the reason for their absence: ill, ill – cold, ill – flu, team event – football. Although we don’t recommend defining reasons for the Absent attendance code, if a school has defined Absent – ill, students with this code will be excluded because a reason is selected. The entry of student attendance always provides the ability to select an attendance code only or an attendance code with a reason (e.g. Absent or Absent – ill).

As you examine the list of students, select a student to see additional information about that student. The following information will be available:

 

  1. Today’s attendance - quickly enter any new attendance information. If you find out the reason for the student’s absence, change the attendance code from Absent (without a reason) to any other attendance code or reason. This will mean that the student will no longer appear in the list of students with unexplained absences. See Common features – Edit individual student’s attendance – morning/afternoon method or by period method if additional information is needed.
  1. Contacts - find someone to call and explain why the student is not at school today. See Common features – Contacts if additional information is needed.
  2. Attendance notes – browse notes that have been entered in the past. See Common features – Attendance notes if additional information is needed.
  3. Attendance for the school year – view the student’s attendance history for the school year. See Common features – Attendance (for the school year) if additional information is needed.

 

Hint: Use the NEXT button at the bottom of the page to go to the next student in the list.
