Student records > Personal > Indigenous


Overview

The Personal menu option in a student’s record contains student-specific information, i.e. information that is the same regardless of what school the student attends. Only the school that is the record owner can edit this information.

 

Choose Personal > Indigenous from the submenu.

 

 

 

Indigenous

edsembli SIS has an application to create an XML file containing student Nominal Roll specific information, which is uploaded to Indigenous and Northern Affairs Canada (INAC) via their School Information System Export (SISE) application. These are all the fields under the subheader 'Indigenous and Northern Affairs Canada info' as explained later.

 

Different provinces may require additional fields for part of their Ministry reporting as follows:


British Columbia only: Students will have a Yes/No field called Aboriginal ancestry which should be set to yes, if applicable. This field is used in the 1701 data collections for the Ministry.

 

Alberta only: Students who have self-identified as First Nations, Metis, or Inuit should have their declaration recorded in the FNMI declaration field (first field on the card) so that it may be reported to PASI for funding purposes.

 

Saskatchewan only: Students will have an Aboriginal status field that is used in the SDS reporting. Select a value if the student is an aboriginal student: Non-status Indian, Metis, and Inuit/Inuk.

 

Manitoba only:  Students may have up to 4 Aboriginal cultural-linguistic identities entered by selecting from the following values defined by the division using Setup > Indigenous > Aboriginal identities. These 4 identities are reported to the Ministry in the EIS data collection.

 

Ontario only: OnSIS collect's the student's FNMI declaration and the date it was received.

Indigenous and Northern Affairs Canada info

 

Please refer to the government's nominal roll documentation if additional clarification is needed for any of the fields in this area. Many of the Indigenous fields use drop down lists to select the most appropriate value for the student. Some of the fields use user-defined tables that are set up by the board/division/district using Setup > Indigenous. The following tables are user-defined: Bands, Reserves, Status codes, Districts of financial responsibility, Accommodations, and Residence statuses. Provinces may also have additional tables in the same set up area. Values provided below should be set up with the codes noted in brackets (e.g. On reserve (1) represents a value in the table called 'On reserve' with the Ministry code of 1).

 

Indigenous status code: This field was previously collected by INAC and was referred to as Aboriginal status code or Native status. Since it is no longer collected, this field may be optionally used for the board/district/division's own purposes.  This table previously contained statuses such as::

  • Unknown
  • Child of government employees
  • Indians not yet registered but whose parents are registered
  • Inuit
  • Other
  • Registered Indians On-reserve or Crown Land Indians
  • Reinstated Indians.

 

Extent of Indigenous language instruction: Select the most appropriate value that represent's the student's extend of aboriginal language instruction:

  • No First Native Language Instruction (0100)
  • Taught as a Subject only (0800.01)
  • Subject and part-time medium in some other subjects (0800)

 

Indigenous register number (3 digit band #), Indigenous register number (5 digit family #), Indigenous register number (2 digit family #): These 3 fields make up the student's Indian Register Number. Each student is assigned a number if the student is registered under the Indian Act.

 

Residence status: Select a value that describes where a student resides from these values:

  • On reserve (1)
  • On crown land (2)
  • Off reserve ( 3)

 

Band of residence: Select from the user-defined table of values.A band is defined as a body of Indians for whose collective use and benefit lands have been set apart or money is held by the Crown, or who have been declared to be a band for the purpose of the Indian Act. This field indicates the band where the individual resides.

 

Reserve of residence: Select from the user-defined table of values. The 2010 version of Canada's Indian Act, at §2(1) defines a reserve as: ... "The term reserve means any tract or tracts of land set apart by treaty or otherwise for the use or benefit of or granted to a particular band of Indians of which legal title is in the Crown, but which is unsurrendered.". This field indicates the reserve where the individual lives.

 

Accommodation: Select the most appropriate value that represents the student's current accommodation:

  • Group home (3425)
  • Student residence (1673.02)
  • Boarding school (1673.01)
  • Mandatory room and board, Mandatory private home placement (1678.01)
  • No accommodation provided (1680)
  • Room and board (5 days) (1678.02).

 

District of financial responsibility: Select from the user-define table of values. This field refers to the district that is funding the student's education. It is a 3 digit numeric code. The first digit in the District number, refers to the region. The next two numbers divide the region up into districts. While some regions have only one district, several regions have several districts. For example, Ontario has four districts: Sudbury, Southern, Western and Sioux Lookout. Historically, this code has been used for funding purposes. This coding has been designated by Lands and Trust Services, as each band and tribal council belongs to a specific district.

 

Band of financial responsibility:  Select from the user-defined table of values. This field refers to the band that is funding the student's education.

 

Home language: Select the individual's language spoken at home from the following values:

  • English (EN)
  • French (FR)
  • First Nation (NAI).

 

Elementary/secondary program: Select the program from the following values:

  • Regular (0100)
  • Immersion (9999)
  • Bilingual (1000)
  • Adult/Mature (1501)
  • Alternative/Outreach (2600)
  • Technical Training (0308)
  • Vocational Training (0399)

 

Note: each school is configured to determine which values apply for the school. If a value is not available, use Setup > School > School configuration to se the school programs offered (on the INAC card).

 

Program delivery method: Select the program delivery method from the following values:

  • Classroom (0605)
  • Distance Education (correspondence courses, Independent Learning Centre, etc.) (0608.01)
  • Virtual (Internet) (0608.02)
  • Home Schooled (0610)
