Suspension report

To create a Suspension report choose Student reports > Incidents and select  RUN on the Suspension report card.

 

 

The Student reports - Suspension report screen will appear with 4 tabs:.

 

INCIDENT tab

In the Print report for area select a From and To date to print suspensions.

 

Students with suspensions - Select the students you want included in the report.

 

Summary - On the Summary card specify these options:

Reported by

Date of incident

Time of incident

Police involved

Officer's name

Number of offender(s) involved

Violent incident report required

Student's comment note

Incident note

Student incident note

 

 

SUSPENSION tab

School information - Select the options that you want included in the report.

School name

School number

School address

Principal

 

Infractions - Select the infraction options that you want included in the report.

Mandatory expulsion

Discretionary expulsion

Separate mandatory and discretionary

 

Suspension information - Select the suspension options that you want included in the report.

Number of days for suspension

First day of suspension

Last day of suspension

Number of other suspensions this year

Type of suspension

Suspended by, this options also has the option to include the name.

Parent(s) contacted

 

Support services - Select the support services that you want included in the report.

Special ed

Student has IEP

Exceptionalities

 

 

DEMOGRAPHICS tab

Student name - Select the options that you want included in the report. If you select Use name fields then decide on the fields you want:

Surname

Given name

Middle name

Usual name

Legal surname

 

If you do not choose to use name fields then select from the predefined Name formats:

 

 

Personal - Select from these options:

Date of birth

Home phone

Grade

Age

Student number

Parent/guardian

OEN number

Homeroom

 

Address -  Select Use complete address to include all fields. If you do not choose to use complete address then select the fields that you want.

 

 

FORMAT tab

Margins - Select Top, Left, Bottom and Right margins.

 

Other - Select a Date format from the drop down list. The Header can include school and or board information.

 

Copies for - Select who will receive copies:

Student OSR

Parent

Board (director)

Attendance counsellor

Superintendent affiliate

Teachers

 

Additional notes - Select from the Report notes available.
