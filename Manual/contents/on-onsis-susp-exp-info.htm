OnSIS > Elementary school suspension/expulsion > Suspension/expulsion info

OnSIS > Secondary school suspension/expulsion > Suspension/expulsion info

 

 

The Suspension/expulsion info option will appear for the following submission periods:

  

  • Elementary school suspension/expulsion (publicly funded schools only)
  • Secondary school suspension/expulsion (publicly funded schools only)

 

This option is used to create batch files, using the CREATE button. The batch file is uploaded to OnSIS through the OnSIS web site.  Once OnSIS processes the batch file, a results file is created. The results file is downloaded from OnSIS and is then processed using the PROCESS RESULTS button.  

 

This section describes the suspension/expulsion info batch file. See the OnSIS Important concepts section for info on Updating data in OnSIS using the ‘Create/Upload/Process’ cycle.

 

The file provides information about the incidents in the school that resulted in a student being suspended or expelled from school. Suspension and expulsion data for the school year is collected at the end of the year (August 31) and it uses the info in the student’s record under the Incidents option. If the school is working on this submission period after transferring over to the new school year, it is possible to create this batch file for the previous school year’s incidents.

 

Follow the link below for the file specifications.

 

Incidents batch file  

 

 

Select the CREATE button on the Suspension/expulsion info card.

 

  

Select the CREATE button to see this form:

 

 

 

The Incidents batch file option is detailed below. Prior to selecting the CREATE BATCH FILE option, determine whether other options on the form should be selected.   

 

Purge:  Y/N field. See Important concepts – Purge check boxes for more info.

 

Use this school year (yyyy – yyyy) or Use previous school year (yyyy – yyyy): Select one of the options to indicate whether the batch file is to include incidents from the current school year or the previous school year.  For example, if the school has rolled over to a new school year in late August, when creating the file in September, use the previous year option. If the school has not yet transferred to current, use the ‘this school year’ option.

 

Incidents batch file (Elementary and secondary schools)

Default file name: E_######_Incident_yyyymmdd_hhmm.xml (elementary) or S_######_Incident_yyyymmdd_hhmm.xml (secondary)

 

The incidents batch file contains information about the students involved in incidents that lead to the student’s suspension or expulsion from the school.  Very detailed info is reported based on the info recorded in the student’s record using the Incidents option. This includes general info about the time/location of the incident, infraction and consequence info, appeal info, and participation in programs for suspended/expelled students.

 

The fields collected by OnSIS are detailed below.
