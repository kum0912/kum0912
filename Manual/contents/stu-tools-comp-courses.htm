Calculate compulsory credits

Each course code entered into the ‘Completed courses’ (transcript) area of a student’s record has an automatically calculated Y/N field that indicates whether or not that course is compulsory (required to fulfill graduation requirements).  This calculation is done automatically every time a student’s completed course is accessed, and it is done automatically as final marks are posted to the completed courses area.

 

There may be the occasional time where you may be asked to run the calculate compulsory credits tool which will go through all current or past students to recalculate the Y/N compulsory field.  Typically, this is only run upon the advice of Maplewood, however, there is no harm in running it at any time.

 

To access this tool, select the Menu icon in the left corner, then select Student tools, then Completed courses.

 

 

This will take you to the Completed courses section where you will find the Calculate compulsory credits  card.

 

 

Select either the CURRENT or PAST button to begin the recalculation of compulsory credits.  The CURRENT button will process all the students who have attended the school at some point this year or will be attending the school next year.  The PAST button will process all the students who have attended the school in previous years.
