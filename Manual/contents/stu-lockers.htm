Student records > Lockers

 

The Locker option in a student’s record provides easy access to the student’s locker information.  Locker information is available for current and future students only.  Students have one locker so the re-assignment of lockers for the next school year should not be done until the current school year is finished. If a student is concurrently enrolled in multiple schools, locker information may be recorded at each school.  Some schools opt to record only the student's locker number if the effort in maintaining accurate lock info does not provide significant benefit.  

 

Before a school may begin recording locker information for the students, there are some setup tasks that must be done.  For example, establishing the lockers that are available to the students, and defining lock combinations for school owned locks that are supplied to the students.  New schools will often have this information imported to minimize the amount of data entry required.

 

The student’s locker information is broken down into two main areas:  info about the locker and info about the lock.  

 

Locker:  Before lockers are assigned to students, the lockers in the school must first be defined.  Each locker is defined by a locker number, locker bank, locker type, capacity (number of students using the locker), and whether or not the lock is built into the locker.  As part of this setup, lockers will default as ‘available’ but this should be changed if the locker is not available (e.g. during renovations).

 

See Setup > Lockers for information on defining the school locker types, locker banks, and lockers. Locker types typically describe the locker using names that are meaningful to the school (e.g. Single, Half, Triple tier, Shared, Red, Blue, Double-width).  Locker banks typically describe the location of the locker (e.g. Music Hall, Tech Wing, 2nd Floor North Hall).  

 

The locker setup is generally a one-time process, and edsembli SIS provides the tools to quickly add ranges of lockers.  

 

Lock:  A lock is used to secure the student’s personal belongings in the locker.  Depending upon the school, any of the following locks may apply:

 

  • The lock is the student’s personal lock.  The lock owner is the student.  The lock is not built-in to the locker (e.g. the padlock can be taken off the locker and the student can take it home when they withdraw or graduate from the school).  Lock information is automatically removed when a student’s school year status changes to withdrawn or past.

 

  • The school supplies the lock to the student.  The student borrows the lock for one or many years. The lock owner is the school.  The lock is not built-in to the locker (e.g. the padlock can be taken off the locker and stored elsewhere in the school when not on loan to a student).

 

  • The lock is built-in to the locker. A built-in combination lock cannot be taken off the locker and used on a different locker. The lock is defined as part of the locker, and when a student is assign one of these lockers, lock information automatically appears.  A school or student owned lock cannot be used on these lockers.

 

Each lock will be defined by a type, serial number, and a combination.  If the lock is not built into the locker, it will have an owner (either the school or the student).  Before lock information is recorded for a student, the school creates a list of lock types (e.g. manufacturer or description of the type of lock), as described in Setup > Lockers > Locks.  

 

If the students are assigned new lockers each year, rather than editing locker information one student at a time through student records, there are some tools that may be helpful:

 

  • Student tools > Lockers > Clear lockers will clear locker information for groups of students. This will make the lockers available to be assigned to other students.
  • Student tools > Lockers > Assign lockers will assign lockers to a group of students.
  • Student tools > Lockers > Clear locks will clear the school-owned lock info for a group of students. Use this if the students receive new school-owned locks each school year. Each student's new lock info would need to be entered as each new lock is distributed. 

 

Access the student's locker info by choosing the Lockers option from the student's record.

 

 

If the student has no locker/lock info, use the Add icon in the locker card.

 

 

If the student has locker/lock info, it will appear in the locker card. The student's current locker number is also included in the card title (e.g. this student is currently assigned to locker 0136).

 

 

Use the Edit icon to edit the student's locker/lock info.

 

 

The following fields appear on the edit locker form:

 

Locker (locker # - locker bank - locker type):  Select the student’s locker from the list of lockers that are available.  The list will include lockers that are not yet assigned to a student, and lockers that have capacity for another student where lockers are setup for multiple students. The lockers are sorted by locker number so you may type the locker number if easier. In addition to the locker number, the locker bank and locker type are included to provide the full locker details.

 

To remove a locker from a student, select the blank locker value at the top of the drop down list of lockers.

 

Lock (serial #  - combination - lock type): In some situations, the lock information will automatically appear when the locker is selected:

  • If the selected locker has a built-in lock, the lock information will automatically appear and no additional information is required.  For these lockers, lock information may not be edited from the student’s record.  

 

  • If the student is assigned a locker that has a capacity of more than 1 student, lock information is shared with all the students assigned to the locker. Therefore, when the student’s locker is assigned, and the shared locker already has another student, that other student’s lock information will automatically appear.  Changing the lock information for one student will change the information for the other student(s) automatically.

 

If the student is provided a lock from the school, use the option Existing lock, and then start typing the serial number of the lock to see matching locks from the previously set up list of available school-owned locks. In addition to the serial number, the combination and lock type are included to provide the full lock details.

 

To remove a lock from a student, enter N/A for the lock serial number.

 

If the lock is not found in the list, or if the student is using their own personal lock, use the option New lock, and then enter the lock information as shown here:

 

 

 

Serial number:  Enter the serial number of the lock, if provided by the manufacturer.

Combination:  Enter the combination of the lock in the format ##.##.##.

Lock type:  Select the type of lock (e.g. type or manufacturer or style) of the lock from the list defined by the school.

Lock owner:  Select the lock owner from the list.  Two options exist:

 

  • School supplied lock:  The lock is owned by school and is loaned to the student. When a school owned lock is no longer used by the student, the physical lock is returned to the office.  The lock information remains in the school’s table of locks so that the lock may be used again by another student.

 

  • Student’s personal lock: The lock is owned by the student.  When a student-owned lock is no longer used by the student, it is removed from the school’s table of locks.

 

Once the student’s locker and lock information is correctly entered, select the SAVE button in the bottom right corner.
