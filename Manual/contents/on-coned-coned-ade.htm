Continuing education - ADE

 

This option is available for board users only.

 

The Continuing education ADE option provides a report that mirrors the OnSIS ‘Average Daily Enrolment by Course Offering’ enrolment detail and summary report. It shows Average Daily Enrolment (ADE) info for students on the following continuing education registers:

 

  • Register of Enrolment in Continuing Education Credit Courses
  • Summer School Register
  • Register of Enrolment in Correspondence/Self-Study/e-Learning Courses
  • Register of Enrolment in the International & Indigenous Languages Elementary (IILE) Program
  • Register of Enrolment in the Literacy and Numeracy Program
  • Register of Enrolment in Adult Indigenous Language Instruction

 

The report also allows the user to report ADE for credit or non-credit courses. Non-credit courses are classes with the course offering types for IILE, Literacy and Numeracy, and Adult Indigenous Language Instruction.  

 

To ensure that continuing education ADE data is being captured correctly for OnSIS reporting, Ontario publicly funded boards, continuing education schools, and summer schools may create this OnSIS validation report - without having to access the OnSIS application, and without the need for OnSIS extract files.  

 

The ADE calculations on this report will be based on the information recorded on the registers (see links above). Refer to the instructions on each register for details on ADE for these students.

 

To access this report, select Ontario Reporting > Continuing education > Continuing education ADE.

 

 

Select Go on the Continuing education ADE card to see this form:

 

 

 

Submission period (OnSIS): Select Continuing education/night school or Summer school from the drop down list.

 

If you select this option from the continuing education or summer school, the appropriate submission period will be defaulted. The report count date will automatically match the current year’s count dates (e.g. August 31).

   

Select the Print button form to submit the report to the task queue. You can leave this application or generate other reports. You will receive notification when your report is ready. Reports are saved in file manager under Provincial > Continuing Education.

 

 

Hint: See previously created reports by selecting the folder icon to open file manager at this location.

Students and their courses will be categorized by the course offering type (e.g. correspondence in the above example)  and student board residence status (e.g. pupil of the board in the above example). The semester indicator is automatically set to Summer Course, First Semester Course, Second Semester Course or Full-year Course based on the schedule assigned to the class.

 

At the very end of the report, a Grand Total ADE will show the sum of all the student ADE calculations for all courses on the report.

 

A report of OnSIS warnings may also be automatically generated if warnings exist.  Carefully examine these warnings prior to submitting data to OnSIS. These are the same warnings that are generated when creating OnSIS batch files.
