Period attendance report

This report lists the dates a student has attendance recorded (e.g. dates where a student is not present all day). For each date, it lists all the attendance information entered for the day. This includes homeroom attendance (if applicable), the class the student was taking during each period, and the student’s attendance for each period. 

 

Choose Attendance > Reports > Period attendance or Student reports > Attendance > Period attendance and select RUN on the Period attendance card.

 

   

Prior to generating the report, the setup of the report parameters should be examined to ensure that you generate the desired report.

 

You can print this report in many different options:  Alphabet, group, grade, homeroom, advisor group, defined character, excursion list:

 

 

Select a print order from the header drop down list. Alphabet is the default but you can also choose Class, Group, Grade, Excursion, Homeroom, Defined character, and Advisor group. See Student reports common features for more Print order information.

 

Include students registered - Select the registered type, both this year and next year. This year only (not returning), Next year only or Withdrawn (no longer enrolled this year).

 

Report parameters - Check the options that you want from the following:

One student per page - If generating the report for multiple students, and each student is to start on a new page, check this box.

Show perfect attendance message - If a student has only present attendance entries, the student will be excluded from the report unless you check this box. Students with perfect attendance will see “The student has perfect attendance” as shown here:

 

  

Student mailing information - If this option is selected, the school’s address, and contact information is included, along with the student’s address, student numbers, grade and homeroom.

Include attendance notes - The attendance notes for the current school year will be printed if this option is selected.  Each note includes a date and the text of the entered note for that date.
