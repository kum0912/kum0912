Overview Care and/or Treatment, Custody and Correctional (CTCC) facilities

The Care and/or Treatment, Custody and Correctional (CTCC) facilities submission period applies to publicly funded secondary schools offering education in any of the CTCC facilities.

 

Schools need to have their school configuration set up with a school operating type of ‘CTCC’ to enable access to this OnSIS submission period.

 

This submission period captures elemental data based on an August 31 count date.  

 

If you are new to OnSIS, or as a refresher with each new OnSIS submission period, take a moment to review these important concepts.  

 

Things to check before commencing the 3 step ‘Create/Upload/Process’ cycle:

 

  • All students have OENs  
  • Run pre-validation reports

 

 

Upon completion of these steps:

 

  • Run verification reports
  • Sign off in OnSIS
