Overview - Prepare for next year

In the winter, schools typically start preparing for the forthcoming school year. The first step is to ‘Prepare for next year’ which involves some steps that are done once at the start of the scheduling progress.

 

  • Initialize next year
  • Lunches (not applicable to elementary schools)
  • Sem/term/day period

 

These 3 steps correspond to the 3 cards found when you select Scheduling next year > Prepare for next year.

 

 

The student and school class information for the next school year is distinct from the corresponding information for this school year. Each year, when you are ready to start scheduling, you must first run the initialize next year process. This process performs the following primary functions based on the options selected as part of this process:

 

  • Sets the school year status to This Year Only for students attending the school this year in the school’s highest grade. Each student’s exit date, exit description, and destination school (if applicable) is set to the defaults set by the school. Students in the highest grade in the school are assumed to be leaving the school at the end of this year, and those who aren’t leaving can be adjusted manually after the process is completed.
  • Advances each student's grade next year based on the progression of grades set by the school. This year's grade is not changed.
  • Each student's homeroom next year can be removed (set to not assigned) or it may be set to the same homeroom as this year (if the list of homerooms is copied from this year to next year). Similarly, each student’s advisor group can be removed or set the same as this year.

Initialize next year has other functions as explained in Scheduling next year > Prepare for next year > Initialize next year.

 

When determining the best location to schedule classes, and when assigning students into classes, if the school requires a period to be reserved for a lunch break, the lunch periods have to be defined using Scheduling next year > Prepare for next year > Lunches.  Lunch periods are optional, and often are not defined if all students share the same lunch period.

 

It is imperative that you make sure your semester term and day/period sets and patterns are defined correctly (Scheduling next year > Prepare for next year > Sem/term/day/period) as explained in Important concepts.
