Scheduling next year > Analyze master timetable - Overview

Every time a class schedule is changed, the school must run load students in order to update the students' timetables. It also updates the conflict information used to analyze the master timetable, an important step in the scheduling process.

 

The Analyze conflicts option is the main tool used to analyze the master timetable. It helps to quickly identify poorly scheduled courses. These poorly scheduled courses can be analyzed further using Master builder’s manual option which can identify if a better location is available if the teacher and/or room were changed.  

 

Although teacher and room conflicts may not be created through the automatic master builder tool, it is wise to confirm that the school has not accidentally created conflicts by editing the classes directly. A teacher conflict occurs when the teacher is scheduled to teach multiple classes at the same time, and a room conflict occurs when multiple classes are scheduled in the same room at the same time. Classes that are intentionally grouped together in the master timetable are not considered conflicts (e.g. when a teacher teachers 2 different courses at the same time).

 

The tools to analyze the school's master timetable for next year are as follows:

  1. Analyze conflicts: Use Scheduling next year > Analyze master timetable > Analyze conflicts to identify courses that have large numbers of students unable to get assigned into classes. Select the class to access the Schedule option (Master builder - Manual) to identify if there is a better placement considering different teachers or rooms.  
  2. Teacher conflicts: Use Scheduling next year > Analyze master timetable > Teacher conflicts to see if any teachers have more than one class scheduled at the same time and the classes are not grouped.
  3. Room conflicts: Use Scheduling next year > Analyze master timetable > Room conflicts to see if any rooms have more than one class scheduled in the room at the same time and the classes are not grouped.
     

In addition to the options above, another tool to analyze the master timetable is available using Scheduling next year > Schedule classes > Timetable of specific group. Here are some examples of how this may help:

  • By looking at the total number of students in each period, you can sense whether the timetable is balanced. For example, typically all students in grade 9 have full timetables (no spares), and these students typically only take grade 9 subjects. After scheduling the grade 9 classes you can see the number of students in each period. Looking at the nominal class size, you will see the theoretical number of students in the class assuming classes are perfectly loaded without conflicts (e.g. 2 classes with 42 students requesting the course would show 21 students per class).The numbers should be fairy even across the periods. If one period has too few students, and another period has too many students, consider analyzing the impact of moving classes around by using Master builder - Manual on a class in an over-loaded period to see if it can be moved to an under-loaded period).  
  • By looking at the actual class size, you can identify classes that are overloaded (if load students was not limited to the maximum class size).  In this case, the overloaded classes are scheduled in a great spot, so consider analyzing the impact of moving the under-loaded class to a better location (using Master builder - Manual).
