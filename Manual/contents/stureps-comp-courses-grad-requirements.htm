Graduation requirements

The ‘Graduation requirements’ area of each student’s record will clearly indicate whether or not a student has earned all the required credits in order to graduate. Rather than examining students one at a time, there is a report that can be run that analyzes the students’ completed courses to generate a list of students who have or have not met their graduation requirements. The report has an option to consider classes that are not yet finished so that you can determine whether the student will fulfill their requirements by the end of the school year, assuming they successfully pass those classes.

 

To create a Grad requirements report choose Student reports > Completed courses and select the MET or NOT MET button.

 

 

Both buttons will take you to the same interface, but the ‘Graduation requirements met?’ option will be set automatically to yes if you selected MET, or set to no if you selected NOT MET. There will be two tabs on the main screen, GRADUATION REQUIREMENTS and SHSM REQUIREMENTS.

 

MET

GRADUATION REQUIREMENTS

 

Select a print order from the header drop down list. Alphabet is the default but you can also choose Class, Group, Grade, Excursion, Homeroom, Defined character, and Advisor group. See Student reports common features for more Print order information.

 

Include students registered - Select the registered type, both this year and next year. This year only (not returning), Next year only or Withdrawn (no longer enrolled this year).

 

Options - Report on students with graduation requirements will be set to Met. You can choose to Exclude students with diploma already granted. You can also choose to include completion status for the Literacy test and Community involvement.

 

Projected requirements - On this card Include current classes in timetables not yet completed will require that you select the semester.

 

 

 

SHSH REQUIREMENTS

 

Select a print order from the header drop down list. Alphabet is the default but you can also choose Class, Group, Grade, Excursion, Homeroom, Defined character, and Advisor group. See Student reports common features for more Print order information.

 

Include students registered - Select the registered type, both this year and next year. This year only (not returning), Next year only or Withdrawn (no longer enrolled this year).

 

Options - Report on students with graduation requirements will be set to Met. You can choose to Exclude students with diploma already granted.

In the Number of credits to validate, select numbers from The Major credit and Co-op grades to include.

You can also choose to Create a new excursion list containing students in the report.

 

Projected requirements - On this card Include current classes in timetables not yet completed will require that you select the semester.
