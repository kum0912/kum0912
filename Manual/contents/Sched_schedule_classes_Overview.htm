Scheduling next year > Schedule classes - Overview

The Scheduling next year > Schedule classes option is where you’ll find all the tools to help schedule the classes into the best placements. A placement is the assignment of a specific pattern for the semester/term and day period. For example, if a class has the semester/term set called 'Single semester' and the day/period set called 'Single period', the possible placements are semester 1 period 1; semester 1 period 2; semester 2 period 1; semester 2 period 2, etc.  Since set and pattern names are used defined, they should use terminology that is familiar to the school (e.g. the pattern called period 1 in one school might be called Block A in another school).

 

The best placement for a class is:

  • the placement with the lowest number of student of conflicts AND
  • the teacher and room are available in that placement (e.g. the teacher is not teaching another class and the room is not occupied with another class).

 

If more than 1 placement has the fewest conflicts, the placement with the lowest balance is considered the ‘best’.   

 

Before deciding the best placement for a class, the application will analyze the impact of scheduling the class in every placement. This way it can compare the number of conflicts and balance for each placement to determine the best placement. The ‘# Conflicts’ is the number of student conflicts that will be generated if the class is scheduled in that placement.  The ‘# Balance’ is the theoretical number of students who will be scheduled into classes in that placement (actual class sizes are not known at this step, so if 100 students have requested a course and there are 4 classes, theoretically each class will have a balance of 25 students). The balance sums the theoretical number of students in classes that are at the same grade level as this class.

 

Typically, a school will schedule a small group of classes, and then will run the Load students process then Analyze conflicts. Once satisfied with the results, these steps are repeated for another group of classes. Refer to the secondary scheduling overview for suggested steps to schedule the school. There are different approaches to the determining which groups of classes to schedule first.  Since every school is different, there is no right or wrong order.  We recommend that classes with the most restrictions be scheduled first. Restrictions may be due to rooms (only 1 gym and it is used every period).

 

The tools to schedule classes for next year are as follows:

 

  1. Master builder - Automatic:  Use Scheduling next year > Schedule classes > Master builder - Automatic to automatically schedule classes.  
  2. Master builder - Manual:  Use Scheduling next year > Schedule classes > Master builder - Manual to see all the information to help you decide where to schedule the class, including the number of conflicts that would be generated in each potential placement. Manual placement may highlight a better placement not available in the automatic mode because the teacher and/or room were unavailable.
  3. Unscheduled classes:  Run Scheduling next year > Schedule Classes > Unscheduled classes to see if there are any classes that are not yet scheduled. Students cannot be assigned into these classes. If classes are found, you may select the class to access the schedule option which will open Master builder - Manual, for the class.
  4. Timetable of a specific group:  Use Scheduling next year > Schedule classes > Timetable of specific group to see a master timetable view for a group of classes, including the actual or theoretical (aka nominal) number of students who will be scheduled into classes in each period. This will help determine if the periods are balanced and will help identify if any classes may be overloaded (after running load students without limiting classes to their maximum occupancy).
  5. Available teachers:  Scheduling next year > Classes > Available teachers to identify what teachers are available for a specific semester/term/day/period.

  6. Available rooms:  Use Scheduling next year > Classes > Available rooms to identify what rooms are available for a specific semester/term/day/period.
