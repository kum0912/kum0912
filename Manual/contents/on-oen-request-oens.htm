Request OENs from Ministry

 

If the publicly funded school or board has determined that it has students without OENs, and those students should have OENs, then the school (or board) may create a file to request OENs for the students. This file is referred to as the ‘batch process’ method to obtain OENs for a large number of students, when using the Ministry’s online individual student method would not be practical.

 

The file will automatically exclude students who are not attending the school until next year, and it will exclude students with an entry description that indicates that the student has come to the school from another publicly funded school (because they should have an OEN already assigned that should be located in the student’s OSR). It will also omit continuing education/summer school students who are parents attending a literacy/numeracy program, in order to support their child.

 

Choose Ontario reporting, Ontario Education Numbers (OEN) from the main menu.



Select the GO button on the Request OENs from Ministry card to access the Create batch file page. The Request OENs option simply requires you to select the CREATE BATCH FILE button. If the request is made from the board, keep the default to select all schools, or use the standard select schools function to select individual schools.

 

 


A task will be submitted to create the file. You may leave the application because an automatic notification will appear when the file is created. The notification will link you to the created file. You can download and inspect this file. If satisfactory, submit it to the Ministry’s OEN website as directed by your board.  

 

Hint: select the folder icon to jump to the file manager to the Provincial > OEN > Request location, which is where the newly created file will be saved.
