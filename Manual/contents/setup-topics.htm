Overview

 

Clicking GO will bring the user to the following interface. 

Selecting any record in the table (e.g. row click) will present 6 options: Edit, Details, Add main topic, Add subtopic, Move up and Move down.   

CLOSE button on the bottom bar will return the user to the setup page (landing page with all the cards).

 

Adding Table Records - Board/Division ONLY

If logged in from the school, no add icon is shown.

Click Add icon. Standard Add interface will appear with fields in the following order (field names are bold – do not include #s or colon), and the card title is called Add topic. 

1. English/French: Default screen language. This controls whether the user is entering the English or French version of a field that has unique values for each language. Fields with underlines have unique values for each language.  

2. Identifier: alphanumeric field. Identifier in English or French must >0 characters in order to save, max character count = 5

3. Is this reportable?: check box

4. This topic is used for provincial report card: check box 

5. Allow markbook setup: check box

6. Topic title*: alphanumeric field. Term title in English or French must be >0 characters in order to save, max character count = 255

7. Topic description: show English description if screen language = English or French title max length 255, not required 

8. Selected schools: select which schools will be assigned access to run classes in this term

 

Under the MARKBOOK DEFAULTS tab the following fields are available: 

1. Radio buttons: choice between 'Use standard markbook defaults' or 'Use these defaults'

 

When 'Use these defaults' is selected, the following fields are available:

1. Categories/subcategories:

  • Denominator: numeric field; maximum charcter count = 3

  • Weight: numeric field; maximum character count = 6

  • Letter grade group: drop-down field; lists all letter grade groups that have been defined and have 'Allow in markbooks' selected

  • Publish: checkbox

  • Calculate weight based on the sum of item denominators: checkbox

2. Items:

  • Denominator: numeric field; maximum charcter count = 3

  • Weight: numeric field; maximum character count = 6

  • Letter grade group: drop-down field; lists all letter grade groups that have been defined and have 'Allow in markbooks' selected

  • Assessment type: drop-down list including Summative, Formative, Diagnostic

  • Publish: checkbox

SAVE button is enabled when all required fields are entered.  Selecting SAVE will save the new entry and return the user to the table. 

CANCEL button is always enabled.  Selecting CANCEL will not save the new entry. 

Selecting the back arrow on the main app bar should prompt the user to save new entry IF ALL required fields are entered.

From a school, row click will present the Edit and Details options only. 

 

Selecting the Details option will open a standard Details form. 

DONE or X will close the details form and return to the table (at the same page where user was).

 

Selecting the Edit option will open a standard edit form.

Deleting Table Records

Editing a table record (row click > edit option), will show the delete icon. This icon will be disabled if the role has been assigned to any staff. 

DISCARD button will remove the record from the table. CANCEL will retain the record in the table.

 

Table Tool Bar - Search

Search icon in the tool bar should perform the standard search as per design standards.

 

Table Tool Bar - More Icon - Export

Selecting the MORE icon from the tool bar should give the option to Export. 

 

Moving Topics Around

The topics are sorted by the print sequence field in the database, and that field gets updated/changed by moving the topics around using the interface. 

There are 2 ways to move the topics up and down in the list: 

a. By checking one topic in the left-most column, and clicking the up or down arrow in the Define topics card title bar. 

 

b. By row-clicking a topic and choosing either 'Move up' or 'Move down'.

 

Main topics can only be moved among other main topics, and subtopics can only move within their current parent (main) topic group. 

 

When main topics are moved, all of its subtopics (if it has them) will also move with it. 

 

Viewing Topics From a School

In a school, if Edit is chosen from the row click menu, everything should look the same as the Central Office interfaces, except: 

 

a. There is no checkbox column or up/down buttons in the main list of topics. 

 

b. In the edit screen, all fields are disabled, but tabs can still be viewed. There is still the option to choose English or French and to use the Next and Previous options to move through the main topics and subtopics.

 

c. There is no 'Select Schools' card and no ability to delete a topic.
