Continuing education student type

 

This option is available to Ontario publicly funded school boards only. Schools do not have access to this report.

 

The Continuing education student type option provides a report for boards to ensure that each student that appears on a continuing education enrolment register has the correct ‘Continuing education student type’ field (as recorded in the student’s record under Enrolment > Enrolment register info). Prior to submitting OnSIS batch files, this report should be run for each OnSIS count date to ensure the students have the correct type.

 

The student’s continuing education student type field is used to determine whether a student is eligible for funding through some of the continuing education enrolment registers, specifically the registers for credit courses, correspondence/self-study/e-learning, and summer school.   

 

For example, the Register of Enrolment in Continuing Education Credit Courses and the Register of Enrolment in Correspondence/Self-Study/e-Learning Courses have a count of sessions (lessons/work units) recognized for funding. Sessions recognized for funding will be summed if the student’s Con. Ed. Student Type is at least one of the following:

 

  • Only Attends Con. Ed. School
  • Also Attends Day School (Full-time) *
  • Also Attends Day School (Part-time) *

 

This register will exclude sessions for students when the student’s Continued education student type (aka Con. Ed Type) is one of the following::

 

  • Full-Time Day School (exclude from register totals)
  • Part-time taking same course(s) in day school (exclude from register totals).

 

* The summer school register will only include students with either of these 2 types.

 

To access this report, from the board, select Ontario Reporting > Continuing education > Continuing education student type.

 

 

Select Go on the Continuing education student type card to see this form:

 

 

Select the report date (October, March or June) corresponding to the Onsis submission report dates.  The Report date will automatically match the current year’s count dates (e.g. October 31, March 31, or June 30). A different report date, if applicable, may be selected by checking the Specify date Y/N field.

 

Select the Print button located at the bottom of the form. Once the report has been submitted to the task queue, you may leave this application or generate other reports. You will receive notification when your report is ready. Reports are saved in file manager under Provincial > Continuing Education.

For each student, the report will generate information about the student as of the report date. This will consider the student’s internal transfers to determine the student’s continuing education type and attendance type on the count date.  

 

The continuing education/summer school ‘short’ name is listed along with the student’s start and end date at the school. The student’s continuing education student type at the CE/summer school is listed.   

 

Compare that continuing education student type to the student’s day school attendance type provided on the report. If the student is enrolled in more than one day school on the report date, it will use the school that was the Main School/ Holder of the OSR on that date.  These are the possible attendance types:

 

FT (Full-time)

PT (Part-time)

IS (Independent study – part-time)

SAL (Supervised Alternative Learning – part-time)

1-4 (Full days on alternate day schedule 1, 2, 3, or 4)

 

For example, a student whose attendance type at a day school shows FT (full-time), should have a Con. Ed. Student type of “Also attends day school (Full-time)”.

 

Any corrections to a student’s continuing education student type will need to be made directly in the student’s record.
