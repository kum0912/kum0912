Student tools > Homerooms > By class

Scheduling next year > Homerooms > By class

This tool provides a quick way to assign the same homeroom to all the students in a specific class. This is much faster than editing each student’s homeroom, one at a time, however it is only assigning the homerooms for one class. The By period tool may be better suited if homerooms are to be assigned to everyone based on the student’s class in a specific period (e.g. period 1 class determines the student’s homeroom).

 

A student has a homeroom for the current school year and the forthcoming school year, where applicable. Therefore, this tool is available for assigning this year’s or next year’s homerooms.

 

When a student’s current year’s homeroom is changed, a dated record of the change is stored in the student’s internal transfers. Internal transfers are accessible from the Internal transfers option within the Enrolment area of the student’s record. Along with the date of the change, it records the student’s original homeroom and their new homeroom.

 

From the main menu choose Student tools > Homerooms.

 

Select THIS YEAR or NEXT YEAR on the By class card, depending on whether homerooms need to be assigned to this year’s or next year’s homerooms.

 

 

Enter class and homeroom fields here:

First, select the class which determines the group of students that will be assigned to the new homeroom.

 

Next, select the new homeroom.

 

Once the class and homeroom are selected, select the ASSIGN button at the bottom of the page. Once the process is complete, the students in that class will be assigned the new homeroom.

Depending upon your school’s report card setup, sometimes report card information is tied to the student’s homeroom. This information may include marks (such as overall student learning skills), comments (such as an overall comment on the student), and programs (such as IEP, ESL). If the report card information cannot be transferred because the original and the new homerooms are not setup the same way, a message will appear.

We would suggest you view the list of students, unless you know that teachers have not yet begun report card entries. If you examine the report card information for the students, you will be able to see the information for both the old (deleted) homeroom and the current homeroom. If required, transfer tools are available.
