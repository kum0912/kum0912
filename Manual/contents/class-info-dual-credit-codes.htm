Setup > Class info > Dual Credit Codes

 

Ontario secondary schools use dual credits.  The table of dual credit codes is centrally maintained by the board.

 

The Dual credit codes table lists all the possible dual credit codes, including information as it pertains to the post-secondary institution.  Dual credits allow students to participate in apprenticeship training and college courses. These credits can count towards their high school diploma as well as a postsecondary certificate, diploma, degree or apprenticeship certification.  

 

For example, students can take a course about entrepreneurship taught by a college professor and a high school teacher on a college campus. By passing the course, a student would earn both a high school credit and a college credit.  

 

Dual credit programs are being run across the province and most boards and colleges are currently participating. These programs are part of the School College Work Initiative funded by the Ministry of Education and Ministry of Training, Colleges and Universities.

 

There is an import feature, which when performed at the board level will also import the information from the Ministry into the dual credit codes table.

 

 

The board sets up dual credit codes, the schools have view only access.

 

 

School access

The board defines the dual credit codes, and the school is limited to only those codes.

 

Choose Setup > Class info and select GO on the Dual credit codes card.  

  

 

The Setup - Dual credit codes page will appear.

 

 

The Dual credit codes table lists all the codes offered in the board/division (as defined by the board), and it shows the school the codes that are offered at the school.  Row click to see code details.

 

Board access

The board defines the grades, and the school is limited to use only those grades. For this example we will use the Ontario codes.

 

Choose Setup > Class info and select Go on the Codes card.  

  

 

The Setup - Dual credit codes page will appear.

 

 

Use the search icon to find a specific code, or use the page navigation..

 

 

Add a code

Use the Add icon in the Dual credit codes tool bar to add a new code.

 

 

The Dual credit codes table lists all the Dual credit codes offered in the board/division (as defined by the board), and it shows the school the Dual credit codes that are offered at the school.  Dual credit codes are maintained by the board/district (Central office). Schools have view only access.

 

 

 

 

 

 

Edit a code

Row click the code and select the Edit menu.

 

 

The Edit code form will appear:

 

 

 

 

The edit fields are the same as described in Add a code.

 

You can navigate through the codes by using the PREVIOUS and NEXT buttons. Selecting any of the buttons at the bottom of the page will save edits.

 

Board/division edits to the 3 key fields (code, number, or grade) will automatically change those values EVERYWHERE the code is used. This includes all students’ completed courses (transcripts), all schools’ courses and classes tables for this and next year. This is rarely required but provides an opportunity to correct mistakes in these fields.

 

Board/division edits to the other code fields (e.g. fields that are not the 3 key fields) will not impact any existing use of the code.  It will impact future use only. For example, changing a credit value will only impact the default credit value when a new class is created (credit values of existing classes will not change). Changing the category field (used to determine if the course fulfils graduation requirements) will not change the category for those students that have already recorded the course in their completed courses (transcripts), however, the next students who have this course added to their completed courses will use the new category.

View code details

Selecting a code from the list provides access to the Details option. This allows access to the code setup information without going into the Edit option. Those without edit rights will go directly to the Details option when a code is selected.

Delete a code

Access the Edit form and select the Delete icon. You will be asked to verify the deletion.

 

You cannot delete a code if it is used in any student’s completed courses (transcript), or if the code is used in any school’s courses or classes tables (this or next year).

 

 

 

 

Export codes

On the Codes card, select the more menu and use the Export option. An Excel file will download to your computer.

 

 

Show/hide inactive codes

 

On the Codes card, select the more menu and use the Show inactive codes option, or Hide inactive codes option.
