Student records > Notes


Overview

The Notes menu option in a student’s record is an easy way for you to record and view student-related observations/activities. Notes are available for current, past and future students and access to each is controlled through security. Notes are retained from year to year, however, tools are available to discard bulk notes.

 

Some notes are student-specific, which means the information is the same regardless of the school that the student attends. Only the school that is considered the edsembli record owner can edit these notes:  Medical notes, Special education notes and Alternate programs notes.

 

All other note areas are school specific, meaning these notes are unique for each school and may be edited regardless of whether or not the school is the edsembli record owner. These notes include Notepads 1/2/3, Attendance notes and the 'Dated notepad'.

 

Notepad 1, 2, and 3 are the default notepad names but the board or school can change/customize to better reflect the purpose of the notepad (e.g. Principal, Secretary, and Guidance). The dated notepad title is user definable as well (e.g. Discipline). See Setup > User-defined fields > Notepad titles for details on how these are set up. The board/division/district may set these up for selected schools, and if they are not setup centrally, the school may set them up.

 

The notepads with user defined titles (Notepads 1/2/3 and the Dated notepad) will not be shared with other schools, however, they may be accessed only by those with security rights through the board/division/district.

    

The Notes submenu options shown below will be unique to each school and student. Only special education students have access to the special education notes. Alternate program notes are only available to those students who are flagged as having an alternate program in the Enrolment > Student enrolment area.

 

 

 

Using the Attendance notes example below, you can use the tab strip to navigate between notes.

 

If the student’s record exists in other schools, use the drop down arrow to select a different school. The notes with user defined titles are not visible when looking at another school. 

 

 

 

The 3 user-defined notepads (Notepad 1, 2 & 3) are memo style note areas that have no character limitations. If a note exits, select the note (row click) to edit it. If no note exists, use the Add icon in the tool bar to add the note for the very first time.

 

 

Attendance, Medical, Special education, Alternate programs and Discipline notes are dated notes and you can only record one note for each date. If another note is added on the same date, the new note will be appended to the end of the original note. Use the Add button to add a dated note, or select an existing note (row click) to access the Edit option. The delete icon is available on the tool bar in the edit form.

 

 

 

One user-defined notepad (called the 'dated notepad' during the setup) has 2 additional fields when adding the note.  The user can optionally select from the 2 user-defined list fields that have user-defined titles and values in the drop down list. If the notepad were called Discipline notes, the user could call the 2 fields 'Infraction' and 'Consequence'. This would allow the school to record a discipline note and it will allow the user to select a single infraction and a single consequence for each discipline note.
