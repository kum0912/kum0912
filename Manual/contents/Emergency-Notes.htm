Overview

The Emergency menu option in a student’s record contains information that tends to be student-specific, in other words, information that is the same regardless of what school the student attends. Only the school that is considered the edsembli record owner can edit this information.

 

Every student may have an emergency note and medical alerts that may be prominently displayed when the student's records is first accessed on the Summary page.  This information also triggers the Alert icon that appears in the student's Profile.  The Alerts & notes option is where this information is entered.  

 

The Alerts & notes option is located in the Emergency submenu.

 

 

The alerts & notes information is grouped and presented on separate cards titled Medical alerts, Emergency and Medical notes, as described below. Some schools track a health insurance number, but for privacy reasons, this number is only visible on the edit of the Emergency card.

 

Medical alerts

Before the medical alerts may be entered for students, the board/division/district must set up the alert types (Setup > User-defined fields > Medical alerts). Each alert type may be optionally set up to appear on the Alert icon in the student's profile.  All active medical alerts will appear on the student's summary page, regardless of whether they are set up to show on the profile icon.

 

When the Alerts & notes option is selected, the Medical alerts card will appear.

 

 

Use the Add icon to add a new medical alert.  Select the alert (e.g. row click) to edit an existing alert. Alerts should be given an expiry date when they no longer apply to the student, or they may be deleted using the Delete icon in the tool bar of the edit interface.  Only active alerts appear in the student's summary page and alert icon. To determine whether the alert is still active, today's date is compared to the effective and expiry dates entered here.

 

The following fields are available for each medical alert:

 

Alert type:  Select from the user-defined list of alert types. An alert type is required.

 

Effective date:  Enter an effective date, if appropriate.

 

Expiry date: Enter an expiry date, if appropriate.  

Emergency

The Emergency card will show the student’s emergency note. The Emergency note is a concise note that is visible when a student’s record is first accessed as it appears on the student’s summary page.

  

 

Use the Edit icon to record the emergency note, and if required, the student’s health card number. The health card number is only visible on the edit interface.

 

 

Medical notes

There are unlimited medical notes for the student; each note appears on the Medical Notes card.

 

 

Use the Add icon to add a new medical note.  Only one medical note can be added per day.  Select the note (e.g. row click) to edit an existing medical note. The Delete icon is available on the edit interface.

 

The following fields are required for each medical note:

 

Date:  The date of the medical note. If adding a new note and a note already exists on this date, upon save, the new note will be appended to the previous note.

 

Note:  The medical note has no size limitation.
