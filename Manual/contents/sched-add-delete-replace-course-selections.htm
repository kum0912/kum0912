Scheduling next year > Course selections > Add / delete / replace course selections

This tool provides a quick way to

  • add one course or more to a group of students
  • delete one course or more from a group of students
  • replace one course with another for a group of students.

To access add/delete/replace course selections, choose Scheduling next year, Course selections. Select Go on the Add/delete/replace course selections card.

The Course selections tabbed page will appear. See explanations below.

 

Add course selections

To add a course, or multiple courses to the course selections of a group of students, follow these steps:

1.  Select the ADD COURSE SELECTIONS tab as shown below:

 

2.  Select the group of students. Use the drop down arrow beside the ‘Alphabet’ option to select a different group.  Here’s an example of the Grade group, for students in grade 9 next year:

Typically, the ‘Include students registered’ will have both options checked, so that you include the students who are currently attending your school this year and next year (both this year and next year), as well as the students new to your school next year (next year only).

 

3.  From the Course selection settings card, choose either Preferred or Alternate depending on whether the course is to be added as a preferred course selection or an alternate course selection.  If it is a preferred course, check the box ‘Can the course be substituted with an alternate course if necessary?’ if applicable.  Also check the ‘Locked’ box if the course is to be locked to prevent students from removing the course (if given access to edit course selections online).

 

In the box below the settings, start typing the course.  A list of matching courses will appear. When the desired course is highlighted (e.g. MAT2791), press the Enter key or select it from the list.  

 

The course will appear in the Courses to add card.  Use the delete icon to remove a course.  If the settings were not defined correctly, delete the course and then add it again.

4.  When all the courses are correctly listed in the Courses to add card, select the ADD COURSE SELECTIONS button in the bottom right corner.

 

 

 

Delete course selections

1. To remove a course, or multiple courses from the course selections of a group of students, select the DELETE COURSE SELECTIONS tab as shown below:

 

2. Select the group of students. Use the drop down arrow beside the ‘Alphabet’ option to select a different group. Here’s an example of the Grade group, for students in grade 9 to 11 next year:

Typically, the ‘Include students registered’ will have both options checked, so that you include the students who are currently attending your school this year and next year (both this year and next year), as well as the students new to your school next year (next year only).

 

3. From the Course selection settings card, check/uncheck the ‘Delete locked course selections for ALL selected courses’ box. If it is checked, the course will be removed even if it is locked (locking prevents students from removing it when they are given access to edit their course selections).

 

Then start typing the course to be deleted in the box.  A list of matching courses will appear. When the desired course is highlighted, press the Enter key or select it from the list.  

 

The course will appear in the Courses to delete card. Use the delete icon if a course is to be removed from the card (e.g. it is no longer a course selection that is to be removed).

 

4. When all the courses are correctly listed in the Courses to delete card, select the DELETE COURSE SELECTIONS button in the bottom right corner.

 

Replace course selections

To replace one course with a different course, follow these steps:

1. Select the REPLACE COURSE SELECTIONS tab as shown below:

  

2.  Select the group of students. Use the drop down arrow beside the ‘Alphabet’ option to select a different group. To replace the course for everyone in the school, use the default options (Alphabet order and include student registered options where both options are checked). Here’s an example of the Grade group, for students in grade 8:

 

3. From the Course selection settings card, set the old course (the course to be replaced) and the new course (the course that will be added). To set these courses, start typing the course and a list of matching courses will appear. When the desired course is highlighted, press the Enter key or select it from the list.  

Check/uncheck the ‘Replace locked course selections’ box. If it is checked, the course will be replaced even if it is locked (locking prevents students from removing it when they are given access to edit their course selections).

 

4. When the old and new course are set, select the REPLACE COURSE SELECTIONS button in the bottom right corner.
