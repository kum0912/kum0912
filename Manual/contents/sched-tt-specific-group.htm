Scheduling next year > Classes > Classes > More icon > Master timetable

Scheduling next year > Schedule classes > Timetable of a specific group

This option provides a list of classes ordered by when they are taught. By default, it will include all classes in the school when the search icon is selected. This will provide a view of the school’s master timetable for this year or next year. It can be changed to report a subset of classes using Semester, Term, Day, and Course grade filters (remember to select the Search icon to refresh the info).  

 

If all periods don't show on the page, a horizontal scroll bar is provided. The period X column will appear if the school has floater classes (e.g. a class has a semester/term pattern, but the day/period pattern has been defined without any periods).

The info presented provides the following functions:

  • Shows the actual number of students in each class. While working with next year’s classes, this setting is useful if the load students tool has been run to populate students into classes.
  • While working with next year’s classes, it can show the Nominal class size, which takes the theoretical number of students that will be loaded into the class in a perfect loading. For example, a math course with 50 students requesting the course, and 2 class sections, will show a nominal class size of 25 students for each class. This setting is useful if the load students tool has not been run. The school can analyze the timetable to identify problems before taking the time to run load students. For example, showing the nominal load for grade 9 courses should show similar student totals for each period if the timetable is evenly balanced.
  • While building next year’s timetable, the school may analyze the actual class sizes to identify those classes that are over or under-loaded. If a class is over-loaded, that class is in a good location in the timetable. If it is under-loaded, it is not in a good location and the class may be analyzed further through the Master builder – manual tool to find a better period.
  • While scheduling, the total class size for each period can identify periods that are over or under-loaded.  For example, if grade 9 students typically don’t have spares, filter the list to show only grade 9 courses to see if the total students is fairly equal between each period. Consider moving classes from the over-loaded periods to the under-loaded periods.  Analyze the class through the ‘master builder – manual’ tool to make an informed decision on whether the move will be beneficial.   
  • Classes that are not yet fully scheduled (e.g. the semester/term pattern or day/period pattern is not assigned) will appear in a card titled Unscheduled classes.  No students can be assigned to these classes until both patterns are defined.  

Note: this option is also available using the Master timetable option available from the More option in the setup of the classes (this or next year).
