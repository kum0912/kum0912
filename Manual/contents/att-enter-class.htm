Attendance > Enter attendance > Class

 

Overview

Class attendance may apply to your school, depending upon your school’s setup. For example:

 

  • School uses the ‘by period’ attendance method: Students have classes in their timetables and attendance is recorded based on whether the student attends their classes.
  • School uses the morning/afternoon attendance method: The attendance taking process is repeated later in the day, and rather than taking attendance based on the student’s in a homeroom, the students rotate to different classes so the afternoon attendance is taken based on a class in the student’s timetable in a later period in the day.

 

Ideally, the teacher takes attendance immediately and enters it directly into edsembli CONNECT.

 

From the school office perspective, the Class option is used in the following situations:

 

  • Teachers do not have access to edsembli CONNECT in the classroom so attendance is recorded on paper and delivered to the office for entry.  
  • Regardless of who enters the attendance, the office will need to monitor that attendance has been recorded for all classes. This is usually monitored for each period, especially if the school has to follow up if students have unexplained absences.

 

From the main menu, choose Attendance then Enter attendance.  This will take you to the Enter attendance section of the Attendance page where you will find the Class card:

 

 

Click GO to access the class attendance page.

Class attendance

The school office can use the attendance page to

 

  • Find classes that have not yet taken attendance,
  • Enter class attendance if teacher cannot.

 

The class attendance page starts with a date field to set the date for which attendance is to be entered. Today’s date will be defaulted, but it can be changed if necessary. Then set the period field to show just the classes in that period (or set it to All to show all classes). Below the date and period fields will be a card containing the classes in the selected period as shown here:

 

 

Find classes that have not yet taken attendance

If the teachers enter class attendance directly from the classroom, the school office will access the class attendance page to see what teachers have not yet entered attendance. If the Entered column shows No, that class not yet entered attendance.

 

The office has the ability to show or hide the classes that have already had attendance taken. One of the following 2 icons will appear in the tool bar to indicate whether or not the list shows or hides those classes that have already had attendance entered. Select the icon to change to the other icon.

 

 List does not include homerooms that have attendance taken. If all homerooms have had attendance taken, the list of homerooms will be empty.

 

List includes homerooms that have attendance taken.

 

If the attendance is not yet taken, the office can remind the teachers to complete attendance so that the office can start tracking down the students with unexplained attendance.

 

Please see Important concepts, Attendance taken/not taken for additional information related to showing or hiding the classes that have attendance already entered.

Enter class attendance if teacher cannot

If a teacher is unable to enter class attendance directly from the classroom, then the school office will access the class attendance page to enter the attendance information.

 

Select the class; the selected class will appear in the title of the second card on the page. This card will include the students currently assigned to that class.

 

Select an attendance code for each student in the list. See Important concepts, Attendance codes and reasons for details. At this time you can also enter Attendance notes.

 

There is a Bulk apply feature available for office users to assist in the entry of the same attendance codes for multiple students. See Bulk apply for information on this feature.
