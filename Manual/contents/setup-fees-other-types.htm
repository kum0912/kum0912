Other school fund types

In addition to student fees and deposits, some schools may want to track other school funds that are not student-based nor associated to students.  This might include money submitted to the office for a school fundraiser, or a corporate donation to the school, or money collected from the vending machine, or ticket sales to a school production.  These funds are simply recorded for record-keeping purposes only, with no record-keeping for what happens to the funds after they are collected.   

 

If the schools wants to record these other school funds, they need to be categorized into a fund type to provide a general reason for the collection of the funds.  Once these fund types are set up, the school may track the collection of these funds, which includes specifying the fund type.  

 

Once this is set up initially, it only needs to be accessed if new other school fund types are introduced. 

 

The following fields are available to define the other school fund type: 

 

Description (English/French): For the appropriate language(s), enter a description for the other school fund type.   

 

Asset account number:  The asset account number will default the same text entered for the description, however it can be changed as needed.   

 

Revenue account number:  The revenue account number will default REV-description (e.g. REV-the text entered for the description), however it can be changed as needed. 

 

Applicable taxes:  Select the tax(es) to be defaulted for this other school fund type.  This field will be hidden if no taxes are setup (see Setup - … taxes).
