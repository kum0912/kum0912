Scheduling next year > Print timetables > Teacher timetable

Some schools print next year's timetables for teachers even though the teachers will see their timetable when they use edsembli CONNECT.

 

Select Scheduling next year > Print timetables.

 

 

Select the GO button on the Teacher timetables card. This will open the teacher timetables screen where you can select the report options.

 

 

Once the options are set, select the Print button in the bottom right corner.
