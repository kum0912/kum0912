Student records  - Select a student

 

 

In Maplewood connectEd, you will be asked to select a student (or multiple students). For example, selecting a student is the first step when trying to access a student’s record. By default, the program will assume you want to find a student using the ‘Surname, Usual name’ method as shown here:

 

 

With this default search method, enter as much or as little of the student’s name as you want. Upper or lower case letters don’t matter. Then press the Enter key or select the search button on the right. For example, enter ‘S’ for a list of students whose surnames start with S, or enter ‘Smith’ for a list of students whose surnames start with Smith. If you know the student’s usual name (e.g. if Joseph’s usual name is Joe), enter ‘Smith, Joe’ to go directly to the student record. If multiple students match the entered name, select the desired student. Hint: enter % for a list of all students.

 

Any students who are currently attending your school will be automatically considered in the search for students. The Search criteria area can be expanded to display more search methods and allows additional filters such as including withdrawn, future and past students. See Search criteria below for more information.

Soundex

Soundex is a phonetic algorithm for indexing names by sound, as pronounced in English. A Soundex search algorithm takes a written word, such as a person's name, as input, and produces a character string that identifies a set of words that are (roughly) phonetically alike. It is very handy for searching large databases when the user has incomplete data. For correct matches, the first letter needs to be correct. Visit this site to read the details about how Soundex works.

Search criteria

Sometimes finding a student by surname is not the best search method. Select the Show more icon to expand the search options. Once it is expanded, the icon will change to Hide all (as shown below).

 

 

 

 

 

The Search by drop down gives you these additional search options:

 

Surname, Usual name (default)

Surname, Given name

Given name

Middle name

Legal surname

Provincial student number (e.g. ASN in Alberta)

Grade

 

When searching by a name, the search can also look for a match in the student’s previous names (the log of name changes) and/or Alias (also know as) names.

 

The default searches will consider students who are currently enrolled in the school. Every student is assigned a status reflective of the student’s enrolment at the school being accessed. Students with the following statuses may be checked on or off:

 

This year and next year: student is actively attending school this year and is scheduled to attend school next year.

 

This year only: student is actively attending school this year but is not scheduled to attend this school next year.

 

Next year only: student is attending this school next year (e.g. new to the school next year).

 

Withdrawn: student was attending this school year but has withdrawn from this school (formerly referred to as Deleted).

 

Past: student attended the school in previous years.

 

Specify the search criteria and click the search button to see the results. The search criteria will be collapsed to maximize the available space for the results. After you selected a student from the results, the next/previous student stepping feature will step based on the students matching the search criteria.
