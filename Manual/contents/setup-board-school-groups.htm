Setup > Board > School groups


You can create add, edit and delete a group of schools. To add a group there has to be more than 1 school in the database (including closed schools). From there you can use groups to assign security access to an individual to access the group of schools, or for functions where you can select from a group of schools (e.g. reporting). Schools will not see this card.

 

For example, a board may create groups to define the location of schools such as NW, NE,SW, and SE, and groups for school types such as summer schools, continuing education. Schools may exist in multiple groups.    

 

By default, these three groups are automatically defined and are not editable.   

 

  • All schools = all schools with school status of “still in operation”
  • Elementary schools = all schools with MW license type = Elementary and in operation
  • Secondary schools = all schools with MW license type = Secondary and in operation

 

Choose Setup > Board and select GO on the School groups card.  

 

 

The School groups page will appear.

 

 

Each school group is represented in one card.  The card title = name of the group.  All cards are collapsed by default.

The default school groups appear in this order, and are non-editable (no add/edit icons):

 

  • All schools
  • All elementary
  • All secondary

 

Default groups only include schools that are ‘still in operation’ (setting in school configuration). The default school groups appear only if there are 1 or more schools in the group.  For example, if a board has no secondary schools, then there is no ‘All secondary’ card.

Add a group

Select the Add FAB to add a school group. You will see this School group form. Give the group an English or French name and click SAVE when you have finished.

 

 

Add schools to existing group

 

To add a school to an existing group select the add icon. A standard select schools form will appear. Select schools by using the check boxes.

 

Edit an group

To edit a group select the edit icon. Use the radio buttons to change the language. Edit the School group name. Select SAVE or CANCEL if you change your mind. If you want to remove the School group select the delete icon. You will be asked to confirm the deletion.
