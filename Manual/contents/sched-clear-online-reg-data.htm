Scheduling next year > Year-end roll over > Clear online registration data

Board/division/district only.

 

Before the year-end roll over, or at any other time, boards/divisions/districts should clear online registration data if they have set up an online site for parents/guardians to register students into schools. By default, it will clear all registrations received before the selected date, where the registrations have been processed by the schools (e.g. the students have been added to the school using Student tools > Add new student > Manage online registrations).  

 

Set the date and optionally, set whether it should also clear the registrations that have not been imported into the schools. Once the options are set, select the CLEAR button in the bottom bar.
