Withdrawing students with fees/deposits

Withdrawing a student does not automatically change or adjust any of the student’s account information with respect to fees and deposits, however, messages will appear that will remind you of additional actions with respect to the student’s accounts (fees and deposits).   

   

If an enrolled student’s status changes to Withdrawn and there is an outstanding unpaid balance, you will receive the following message:

 

This student still owes money for fees.  Please print an account bill for the outstanding balance.

OK

 

Similarly, if a student has payment credits and their status changes to Withdrawn, you will receive the following message :

 

This student has payment credits that need to be refunded.

OK

 

If the student has payment credits and outstanding fees, and their status changes to Withdrawn, you will receive the following message:

 

This student has payment credits not applied to outstanding fees.  Please apply payment credits to fees.

OK
