Attendance > Enter attendance > Individual student

 

Overview

The school office will typically receive phone calls and notes regarding the students’ attendance for the current day (Adam is home ill today), previous days (Alicia was away yesterday because she was ill), or future days (Amy is going to be away on a family vacation next week). Students also show up to the main office when they arrive to school late to school.

 

As this information becomes known, it needs to be entered into edsembli SIS. From the main menu, choose Attendance then Enter attendance. This will take you to the Enter attendance section of the Attendance page where you will find the Individual student card:

 

 

Select the GO button and then enter the student’s name using the standard select student form found in student records.

 

 

When the desired student is selected, 4 cards will be available, providing access to the following information for the student:

 

Edit attendance – today’s date: Today’s attendance is editable directly on this card as explained in:

 

Common features – Edit individual student’s attendance – morning/afternoon method   

Common features – Edit individual student’s attendance – by period method

 

We suggest that you avoid using the attendance code A – Absent for absences where the reason for the absence is known. Absent is typically reserved for unexplained absences.

 

Attendance notes: This is the student’s attendance notes as explained in Common features – Attendance notes.

 

Attendance monitoring: (Ontario schools only). See the Attendance > Notes option in the student's record for more info on this functionality.

 

Contact notes: (Ontario schools only).  See the Attendance > Notes option in the student's record for more info on this functionality.

 

Attendance: This card provides access to the student’s attendance for this school year, where attendance may be added or edited for any school day (past, current, and future) as explained in Common features – Attendance (for the school year).

 

Contacts: This is the student’s contacts as explained in Common features – Contacts.

 

As soon as this attendance information is entered by the office, it will appear on the teachers’ attendance pages so that they will know the reason for a student’s absence.
