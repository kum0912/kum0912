Setup > Demographics & enrolment > First languages

 

Overview

The First languages table lists all the possible First languages that may be assigned to a student. First languages are centrally defined by the board/division to ensure accurate Ministry reporting. 

 

 

To define the languages typically used in the students' homes, select GO. 

Selecting any record in the table (e.g. row click) will present 2 options: Edit and Details.

 

*NOTE: if logged into a school, the option to Edit is NOT available. See the section "Row click - Editing table records & Details" for further instructions. 

 

 

CLOSE button on the bottom bar will return the user to the setup page (landing page with all the cards). 

 

 

Adding Table Records - Boards/Divisions ONLY

If logged in from the school, no add icon is shown. 

 

If logged in from the board/division, the Add icon appears in card tool bar if user has edit rights. Users with view only rights do not get the Add icon. 

 

 

Clicking on the Add icon will present a standard Add interface with fields in the following order (field names are bold - do not include #s or colon), and the card title is called Add first languages. 

 

 

1. English/French: Default screen languages. This controls whether the user is entering the English or French version of a field that has unique values for each language. Fields with underlines have unique values for each language.

 

2. First languages title: alphanumeric field.

 

3. Ministry code: numeric value, max character count = 3

 

4. Board code: alphanumeric field, max character count = 3

 

5. Effective date: date field - cannot be after expiry date

 

6. Expiry date: date field - cannot be before effective date

 

SAVE and ADD buttons are disabled until the required fields exist.

 

SAVE button is enabled when all required fields are entered. Selecting SAVE will save the new entry and return the user to the table. 

 

ADD button is enabled when all required fields are entered. Selecting ADD will save the new entry and then open a new (empty) Add interface (so user can add another record to the table).

 

CANCEL is always enabled. Selecting CANCEL will not save the new entry. 

 

Selecting the back arrow on the main app bar should prompt the user to save the new entry IF ALL required fields are entered. 

 

 

Row Click - Editing Table Records & Details Option

From the table, clicking any row with table records (row click) will give 2 options: Edit and Details. 

 

 

From a school, row click will present the Details option only. Edit option is not available for schools. 

 

Selecting the Details option will open a standard Details form. CLOSE or X will close the details form and return to the table (at the same page where the user was. 

 

 

Selecting the Edit option will open a standard Edit form. 

 

On the Edit form, show the DELETE icon in the app bar. 

 

The following fields will be visible, just like the Add form:

 

1. English/French: Default screen languages. This controls whether the user is entering the English or French version of a field that has unique values for each language. Fields with underlines have unique values for each language.

 

2. First languages title: alphanumeric field.

 

3. Ministry code: numeric value, max character count = 3

 

4. Board code: alphanumeric field, max character count = 3

 

5. Effective date: date field - cannot be after expiry date

 

6. Expiry date: date field - cannot be before effective date

 

 

CANCEL and SAVE buttons on the bottom bar are always enabled.  Selecting either of these buttons will exit the edit form and return the user to the location in the table from which they were prior to editing (e.g. should see the row with the record that they edited).  SAVE will save the changes, CANCEL will not save the changes. 

 

PREVIOUS (if there is a previous row in the table) and NEXT (if there is a next row in the table) buttons will appear.   

 

Selecting PREVIOUS button will save the edits and then reopen the edit form with the previous record (row) in the table. Selecting NEXT button will save the edits and then reopen the edit form with the next record (row) in the table.

 

Selecting the back arrow on the main app bar should prompt the user to save the new entry IF ALL required fields are entered. 

 

 

Deleting Table Records

Editing a table record (row click > edit option), will show the delete icon. This icon will be disabled if the first language has been assigned to any students. 

 

DISCARD button will remove the record from the table. CANCEL will retain the record in the table. 

 

 

Table Tool Bar - Search

Search icon in the tool bar should perform the standard search as per design standards.

 

Table Tool Bar - Change View Style (List/Grid)

Standard list/grid view icon appears in the grid. Default grid view but allow user to change to List view.

 

 

Table Tool Bar - More Icon - Export

Selecting the MORE icon from the tool bar should give the option to Export.
