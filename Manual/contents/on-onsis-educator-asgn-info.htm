Educator assignment info

 

The Educator assignment info option will appear for the following submission periods:

  

  • October elementary school  

  • October secondary school

  • March elementary school

  • March secondary school

  • June elementary school

  • June secondary school

 

Private elementary schools, that are not combined with an inspected secondary school, will also have access to this option since they must report this info in October, March and June (along with the aggregate info required in October only).  If they are combined with an inspected  private secondary school, the secondary school will report the educator assignment info.

 

This option is used to create batch files, using the CREATE button. The batch file is uploaded to OnSIS through the OnSIS web site. Once OnSIS processes the batch file, a results file is created. The results file is downloaded from OnSIS and is then processed using the PROCESS RESULTS button.  

 

This section describes the Educator assignment info batch files. See the OnSIS Important concepts section for info on Updating data in OnSIS using the ‘Create/Upload/Process’ cycle.

 

The file provides information about the educators in the school. There are slight variations for the educator info option for elementary vs secondary submissions. Follow the links below for the file specifications.

Educator class assignment batch file  

This batch file applies to elementary and secondary submissions (private and public schools).  

 

Default file name: E12_######_Ed_ClassAssignment_yyyymmdd_hhmm.xml (elementary)  or S12_######_Ed_ClassAssignment_yyyymmdd_hhmm.xml (secondary)

 

The educator class assignment batch file contains information about the educator(s) assigned to each class reported in the class list batch file. For elementary submissions, the classes in OnSIS are the homerooms, so the homeroom teachers and early childhood educators are reported in this batch file. For secondary submissions, the teachers of the classes taught during this submission period are reported in this batch file.

 

For schools with both elementary and secondary submissions, the educator class assignment batch file for the elementary submissions will exclude any homerooms that are assigned to secondary students, based on the grade level of the first student in the homeroom.

 

To correctly identify the teacher, the educator’s position type is included in the file along with the MEN# of the educator. The MEN# must be accurately recorded in the staff member’s record, and the position type will be populated based on the info in the extract recommended for this file.  The steps ‘Update educators in OnSIS’, then ‘Get extracts from OnSIS’ should have been done prior to building this batch file.

Educator subject assignment batch file  

This batch file applies to elementary submissions only (private and public elementary schools). If the private elementary school is combined with an inspected secondary school, this option does not  apply, however, the batch file option will be available for the secondary private school submission.

 

Default file name: E13_######_Ed_SubjectAssignment_yyyymmdd_hhmm.xml (elementary)  or E16_######_Ed_SubjectAssignment_yyyymmdd_hhmm.xml (private elementary school that is  not combined with an inspected secondary school).

 

The educator subject assignment batch file contains information about non-core subjects for educators in the school. This file contains classes if the educator is teaching a subject outside of his/her core class to another group of students. Non-core subjects are ANY subjects taught by an educator and not taught by the educator assigned to the class code (homeroom).

 

Make sure that all courses representing non-core subjects have a value in the course number field (Setup > Class info > Courses).  The value entered into the course number will represent the elementary subject type most appropriate for the class (e.g. enter AM in the course number for a music course).  The subject types are in the Ministry’s reference tables and here is a sample of the types:

 

ELEMENTARY_SUBJECT_TYPE AA Théâtre

ELEMENTARY_SUBJECT_TYPE AD Dramatic Arts

ELEMENTARY_SUBJECT_TYPE AE Exploring the Arts

ELEMENTARY_SUBJECT_TYPE AL Comprehensive Arts

ELEMENTARY_SUBJECT_TYPE AM Music

 

Note that the reference table includes ELEMENTARY_SUBJECT_TYPE ZMS All Subjects (Core) This code is not valid for public elementary schools. It is only valid for elementary aggregate private schools as noted below.

 

When the elementary subject file is created, only those classes with 2 or 3 character course numbers will be included in the file. The class will be excluded if either of the following is true:

 

  •  the class has the reportable box unchecked (in Setup > Class info > Classes)
  •  the class is taught by the homeroom teacher of the students (it will use the homeroom teacher of the first student in the class to determine this). Only non-core subjects are to be reported. These classes will generate the following warning message: "Class excluded from subject assignment file. Only classes taught by a teacher other than the homeroom teacher are included."

 

Private Schools Note: If a secondary inspected private school includes elementary students, and if the elementary teacher is a core teacher (core teacher flag selected in the educator assignment), then the elementary subject assignment file must also contain this subject: ELEMENTARY_SUBJECT_TYPE = ZMS (All Subjects (Core)) along with any non-core subjects. When creating the elementary subject assignment file, the ZMS subject will be automatically added to the file, if the teacher is a homeroom teacher (students in homeroom must be grade 8 or lower).

 

In the batch file, a subject assignment is created for each unique subject/class/grade combination. For example, a teacher is teaching music to 2 classes of straight grade 5 students and 1 class of combined grade 4/5 students. The file will send the info to OnSIS so that it represents a subject assignment of AM - grade 5 - 2 classes and another subject assignment of AM - grades 4/5 -1 class.
