Overview

The student reports area provides access to reports based on student information. For access, click the Menu icon in the left corner, then select Student reports.

 

 

Each student report is presented in a card with a brief description of the report.  Click the cards below to see more about the reports. If there is a MORE button that means not all of the cards are displayed.
