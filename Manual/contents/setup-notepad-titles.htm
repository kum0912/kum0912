Setup > User-defined fields > Notepad titles

Overview

In each student’s record, there are 3 ‘regular’ user defined notepad areas that can be used to enter unlimited notes about the student. Each notepad has a user defined title that would describe the purpose of the notes.   
 
Notepad titles are the same for all students in the school, however the notepad titles may be different between schools. 

 

The board/division can optionally control notepad titles for some or all of the schools. If the notepad title is not defined by the board for the school, the school can then change the notepad title to match its needs.   
 
There is also 1 user defined ‘dated’ notepad that also includes user defined 2 list fields.  This may be used for tracking minor discipline incidents in the school (where field 1 could track infractions and field 2 could track the consequence), or it may be used by guidance to categorize appointments (field 1 is Scheduled apt or drop-in and field 2 is the reason (career, course planning, timetable, crisis, etc). Since the list fields are centrally defined, this dated notepad is only defined by the board/division. 

 

Tabs are used to navigate between the setup of the regular notepad titles, versus the dated notepad with user definable list fields. 

 

Notepad Titles (Board/District)

 

 

Clicking GO will bring the user to the following interface for Regular Notepads. 

 

If the notepad number has no board defined titles, the card will appear collapsed by default. 

 

If a notepad number has titles defined, the titles will appear in a standard grid with the following columns:

 

1. Title: show the title defined by the board (Will be in English or French, dependent on the screen language)


2. Description: Show the description defined by board. (Will be in English or French, dependent on the screen language)


3. Schools: Show the proper school names, comma separated. 

 

 

Adding Table Records

The Add icon appears in card tool bar if user has edit rights. Users with view only rights do not get the Add icon. 

 

Click Add icon. Standard Add interface will appear with fields in the following order (field names are bold – do not include #s or colon), and the card title is called Add notepad title.

 

1. English/French: Default screen language. This controls whether the user is entering the English or French version of a field that has unique values for each language. Fields with underlines have unique values for each language. 
 
2. Notepad # title*: (# = 1 or 2 or 3 corresponding to the notepad # being added) alphanumeric field. Title in English or French must be >0 characters in order to save. 
 
3. Description: alphanumeric field
 
4. Standard ‘selected schools’ card. This is used to select the schools that will get this title.

 

 

SAVE and ADD buttons are disabled until the required fields exist.  

 

SAVE button is enabled when all required fields are entered. Selecting SAVE will save the new entry and return the user to the table. 

 

ADD button is enabled when all required fields are entered. Selecting ADD will save the new entry and then open a new (empty) Add interface (so user can add another record to the table). 

 

CANCEL button is always enabled. Selecting CANCEL will not save the new entry. 

 

Selecting the back arrow on the main app bar should prompt the user to save new entry IF ALL required fields are entered.

 

 

Row Click - Editing Table Records & Details Option

From the table, clicking any row with table records (row click) will give 2 options: Edit (make it first option) and Details.

 

 

From a school, row click will present the Details option only. Edit option is not available for schools.  

 

Selecting the Details option will open a standard Details form.

 

 

CLOSE or X will close the details form and return to the table (at the same page where user was).

 

Selecting the Edit option will open a standard edit form. 

 

On the Edit form, show the DELETE icon in the app bar. 

 

The following fields will be visible, just like the Add form:

 

1. English/French: Default screen language. This controls whether the user is entering the English or French version of a field that has unique values for each language. Fields with underlines have unique values for each language. 
 
2. Notepad # title*: (# = 1 or 2 or 3 corresponding to the notepad # being added) alphanumeric field. Title in English or French must be >0 characters in order to save. 
 
3. Description: alphanumeric field
 
4. Standard ‘selected schools’ card. This is used to select the schools that will get this title.

 

PREVIOUS (if there is a previous row in the table) and NEXT (if there is a next row in the table) buttons will appear.   

 

Selecting PREVIOUS button will save the edits and then reopen the edit form with the previous record (row) in the table. Selecting NEXT button will save the edits and then reopen the edit form with the next record (row) in the table. 

 

 

Selecting the back arrow on the main app bar should prompt the user to save edits.  Standard prompt should appear:

 

 

Deleting Table Records

Editing a table record (row click > edit option), will show the delete icon. This icon will be disabled if the role has been assigned to any staff. 

 

DISCARD button will remove the record from the table. CANCEL will retain the record in the table.

Dated Notepad (Board/District)

From the board/division setup of notepads, selecting the second tab called DATED NOTEPAD to see the following interface. 

 

 

1. Radio buttons: English and French (defaulted to user's Language). Language will determine if English or French database values are shown. 

 

2. Notepad title: text field

 

3. Field 1 title: text field

 

4. Field 2 title: text field

 

There are now 2 cards that both have standard tables containing the same field names, however the values in the tables will correspond to the values for field 1 (e.g., discipline note infraction) and field 2 (e.g., discipline note consequences), respectively. 

 

These cards contain the values that are used in list fields that are presented when entering a student's note.

Notepad Titles (School)

 

 

Clicking GO will bring the user to the following interface for Regular Notepads. 

 

There are 2 tabs: REGULAR NOTEPADS and DATED NOTEPAD

 

The titled fields are:

 

1. Notepad title 1: title currently assigned to notepad 1 at the school

 

2. Notepad title 2: title currently assigned to notepad 2 at the school

 

3. Notepad title 3: title currently assigned to notepad 3 at the school

 

 

If the title is defined by the board/division, the Notepad title field will be disabled. 

 

Users with edit rights can change fields that are defined at the school level. 

 

The SAVE button will be enabled if a field is changed and the CANCEL button will return to the User-defined fields landing page. 

 

If a user has view only access to this setup area, all 3 fields will be disabled and there will only be an option to CLOSE out of the setup interface. 

 

 

Dated Notepad (School)

Selecting the DATED NOTEPAD tab will bring up the following interface. 

 

 

2 cards will appear that contain a table with the board-defined values created by the board/division. 

 

Values cannot be added to the tables or edited from the school level. Only details can be viewed by row-clicking on a value in the table.
