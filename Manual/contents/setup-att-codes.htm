FUTURE RELEASE

Overview

In preparation for recording student attendance, attendance codes are setup. The setup of the attendance codes is generally done during the initial setup of a new board/district or school, and rarely needs to be revisited.

We strongly suggest that all codes are centrally defined by the board/division to ensure the same meaning for an attendance codes is used in any reporting (e.g. you don’t want different schools using the same attendance code for very different purposes). Schools may be granted view only access to the attendance codes to prevent the addition of their own codes.

Prior to setting up attendance codes, please read Attendance – Important concepts – Attendance codes and reasons.  

Attendance is recorded using a list of attendance codes and reasons that may be defined centrally by the board/division or by the school. The attendance code A is reserved for Absent, and P is reserved for Present, but all other letters of the alphabet are user defined. An unlimited number of reasons may be associated with each code to further classify the attendance code. For example, a school might have a L – Late attendance code with 3 reasons: L – Late – Missed bus, L – Late – Slept in, L – Late - Other. Reasons are optional as some schools prefer to enter the reason in the student’s attendance notes.

 

If a student is absent and you know the reason, then record attendance using the appropriate code and not just Absent. For example, if the student is absent due to an appointment, rather than recording A - Absent, the attendance should be recorded using an alternate attendance code such as M - Medical appointment, or M - Medical appointment – dentist (if reasons are defined).

 

We recommend that reasons are not defined for A – Absent so that this code may be used to quickly identify the students with unexplained absences.

Sometimes, the attendance information needs to be summarized in a manner other than based on the A to Z attendance codes entered for the students. For example, you may just want to know how many times the student was absent from school considering all the different codes. Each attendance code has 2 settings that help summarize attendance information.

 

1. Every code has an official setting that indicates whether it is considered Absent, Present, Late, Suspended, or Grant day (e.g. unexpected school closure).

2. Every code also has a classroom setting that indicates whether the student is expected to show up in the classroom. If the student is not expected to show up to the classroom, there are 2 options: absent (student is absent from school) or excused (student is excused from class because they are elsewhere attending a school related activity).

Examples:

A – Absent Official = Absent Classroom = Absent

F – Field trip Official = Present Classroom = Excused

I – Ill Official = Absent Classroom = Absent

L – Late Official = Late Classroom = Present

M – Medical Official = Absent Classroom = Absent

P – Present Official = Present Classroom = Present

T – Team event Official = Present Classroom = Excused

W – With reason Official = Absent Classroom = Absent

 

From the examples above, you can see that there are 3 different ways to view the student’s attendance depending on whether you want to see the school codes, the official codes or the classroom codes.

The setup area for attendance codes is used in the following situations:

• New attendance codes need to be added.

• New reasons need to be added to existing attendance codes.

• An existing attendance code needs to be changed.

Access to the setup of attendance codes is available from 2 locations:

Select the Menu icon in the left corner, then select either Attendance, then Setup or select Setup, then Attendance.

 

This will take you to the attendance setup section where you will find the Attendance codes card.

  

Select the GO button on the card to access the setup of the attendance codes.

Each attendance code has a number of fields that define the purpose and interpretation of the code.  See Attendance code fields for information on each field. Adding a new attendance code

Use the Add icon in the attendance codes tool bar to define a new attendance code, using a letter that is not  already presented in the list of codes.

Then proceed to enter the fields required to define the new attendance code. Please see Attendance code fields for information on each field.

 

Editing an attendance code

Select the attendance code from the list to access the Edit option.  Schools may not edit codes that have Yes in the centrally defined column.

Click edit and make any changes that you want. While editing an attendance code, you can navigate to the next or previous code in the list using the NEXT CODE and PREVIOUS CODE buttons at the bottom of page. When you click the buttons at the bottom of the page any edits will be saved automatically.

Removing an attendance code

You cannot remove an attendance code that has been assigned to any student during the current school year .  If you need to remove a code, select the code from the list to access the Edit option and then click the Discard icon.

 

Attendance code fields

When a code is added or edited, the following fields are available to setup the attendance code:

 

Letter: A letter (A to Z) is defined for the code. This letter will be used in areas of the application where the attendance is condensed such that the title cannot be displayed. The letter A is reserved for Absent and the letter P is reserved for present. If adding a new code, the new code must use a letter that is not yet defined. If editing an existing code, the letter may be changed only if the existing letter has not yet been used this school year.

Official: Select the most appropriate official definition for this code from the following options: Absent, Present, Late, Suspended, or Grant day (unexpected school closure). This definition is used when the attendance information is summarized using the official definition rather than the A to Z attendance codes. For example, you may just want to know how many times the student was absent from school considering all the different codes.

Classroom: Select the most appropriate classroom definition for this code from the following options: Absent, Excused, or Present. Absent is used when the code represents that the student is absent from school for non-school related reasons (e.g. student is sick or on vacation). Excused is used when the code represents that the student is considered present for school but is not expected to show up to the classroom because of a school-related activity (e.g. field trip, sport team game, music performance). This definition is used when the attendance information is summarized using the classroom definition rather than the A to Z attendance codes. For example, you may want to know how much time the student was not in the classroom (missing class lessons). The sum of the codes defined as absent or excused represent the total number of times the student was not in the classroom.

English title: The title, in English, to clarify the purpose of the attendance code. E.g. the code M might have the title Medical.

French title: The title, in French, to clarify the purpose of the attendance code. This is optional.

Reasons: An unlimited number of reasons may be associated with each code to further classify the attendance code. For example, a school might have a L – Late attendance code with 3 reasons: L – Late – Missed bus, L – Late – Slept in, L – Late - Other. Reasons are optional as some schools prefer to enter the reason in the student’s attendance notes.  

From the Reasons card, use the Add icon in the tool bar to add a new reason, or select a previously defined reason if edits are required.

 

Whether adding or editing the reason, there are 3 fields: an English and French title to describe the reason, and an optional identifier. The identifier can be up to 5 characters in length. Identifiers may be useful for external data analysis.

A reason may be removed using the Discard icon on the edit reason tool bar. If the reason has been assigned to anyone this year, it may not be removed .

Centrally defined codes can be assigned to specific schools

If adding/editing attendance codes centrally from the board/division, the Selected schools card will appear to provide control over which schools will have this attendance code. The list will show the schools that currently have access to this code.  

  

Use the Add school icon to add schools to this list. If the new school already has a code defined for that letter, a dialogue will be presented to allow you to decide whether or not the school’s existing code will be replaced with the centrally defined code.

If a school is to be removed from the list of selected schools, check the box to the left of the school to enable the Discard icon. When the Discard icon is selected, all the checked schools will be removed from the list.

Printing/exporting attendance codes.

From the Attendance codes card, select the More menu to access the options to print or export the list of codes.
