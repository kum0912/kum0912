Attendance > Reports > Consecutive absences

 Student reports > Attendance > Consecutive absences

 

 

This report generates a list of students who have been absent for a specified number of consecutive days. For schools with students using the ‘by period’ attendance method, you can also generate a report of students with consecutive absences in a class. You can decide the number of consecutive absences to consider and the date range to search.

 

For example, you can search for students with 5 consecutive full days absent. If a student is absent all day, for 5 or more school days, the student is included in the report as shown below. A grant day, school holiday (e.g. pa day), board holiday, or a non-instructional day that occurs within an otherwise unbroken series of 5 school days of absence does not break or add to the series of absent days.

 

 

The Letter column can be ignored as it represents an internal ID automatically assigned to students. Ontario schools will also see another column after Grade called 'Contact Dates' which will provide information about the dates an attendance counsellor made contact with the student. See Student records > Attendance > Notes for more info on this feature.

 

Choose Attendance > Reports > Consecutive absences or Student reports > Attendance > Consecutive absences and select AM/PM or BY PERIOD on the Consecutive absences card. The corresponding Student reports - Consecutive absences screens will appear.

 

    

Note: if a school uses both attendance methods (morning/afternoon and by period), the report will need to be run twice, by selecting each of the buttons separately. Selecting the AM/PM button will only search the students with the morning/afternoon attendance method. Select the BY PERIOD button will only search the students with the ‘by period’ attendance method.

Setup

Prior to generating the report, the setup of the report parameters should be examined to ensure that you generate the desired report.

 

BY PERIOD

 

Report parameters (BY PERIOD) - The report parameters allow you to set the number of consecutive absences, the date ranges to search, and whether to include students who have left the school (Maplewood status is set to withdrawn).

 

If running the report for the ‘by period’ students, first select whether the consecutive absences are search for full day absences or absences in a class. If search for class absences, select the semester and term (if applicable) corresponding to the classes to be searched. If a school has 2 semesters and 2 terms, selecting semester 1 term 1 will consider any class that is taught during semester 1 term 1. This includes year-long classes, semester 1 classes, and classes taught in semester 1 term 1 only. This does not include classes that are taught in semester 1 term 2 only. The semester and term drop down lists include “All” options (e.g. All terms would include any class taught in term 1 and any class taught in term 2). Classes that are not scheduled into a period are automatically excluded from the report since attendance is not recorded for these classes.\

 

Select Full day absences or Class absences. Then enter a Start and End date and Number of consecutive absences you want to search. Optionally, you can choose to Include withdrawn students.

 

 

 

AM/PM

 

Report parameters (AM/PM) - This is the selection card you will see if you are running the report for the students with the morning/afternoon (AM/PM) attendance method.

 

 

Enter a Start and End date and Number of consecutive absences you want to search. Optionally, you can choose to Include withdrawn students.
