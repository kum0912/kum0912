OnSIS > Continuing education for day school (IILE/Literacy & numeracy) > IILE/Literacy & numeracy info

 

 

The IILE/Literacy & numeracy info option appears for the Continuing education for day school (IILE/Literacy & numeracy) submission only.  Regular, e-learning, and CCTC schools may be configured such that they require access to continuing education registers. Only schools configured this way have access to this submission period and batch file option. This configuration is done using Setup > School > Configuration. Check the box called ‘Requires access to continuing education registers’ if the school has students that need to appear in either of these registers:

 

  • Register of Enrolment in the Literacy and Numeracy Program
  • Register of Enrolment in the International & Indigenous Languages Elementary (IILE) Program

 

Non-credit ADE info needs to be reported for students on these registers using this IILE/Literacy & numeracy info option. Use the register links above for detailed info on how to enter data for students on these registers.

 

This option is used to create batch files, using the CREATE button. The batch file is uploaded to OnSIS through the OnSIS web site.  Once OnSIS processes the batch file, a results file is created. The results file is downloaded from OnSIS and is then processed using the PROCESS RESULTS button.  

 

This section describes the creation of the non-credit ADE batch files. See the OnSIS Important concepts section for info on Updating data in OnSIS using the ‘Create/Upload/Process’ cycle.

 

Follow the links below for the file specifications.

Select the CREATE button on the IILE/Literacy & numeracy info card to see this form:

 

  

The OnSIS batch file options are detailed below. Prior to selecting the CREATE BATCH FILE option, determine whether other options on the form should be selected.   

 

Purge:  Y/N field. See Important concepts – Purge check boxes for more info.

 

Save results to CSV file (aka Also create CSV file): Y/N field. If set to Yes, the CREATE BATCH FILE option will create the requested batch file and the contents of the batch file will also appear in a comma delimited file (same file name as the XML batch file but with a CSV extension). Some boards/schools generate these csv files to perform data analysis on the information sent to OnSIS.

 

Non-Credit ADE – Other programs batch file  

 

For regular day schools, this file includes those students taking classes that appear on the official Register of Enrolment in the Literacy and Numeracy Program. If a class is to appear in this batch file, the setup of the class (Setup > Class info > Classes) requires one of the following values in the course offering type field:  

 

  • Literacy and Numeracy for Parents/Guardians
  • 7/8 Literacy & Numeracy – Adult
  • 7/8 Literacy & Numeracy – Con Ed
  • 7/8 Literacy & Numeracy – Fully High Credit
  • 7/8 Remedial Literacy & Numeracy
  • 9/10 Literacy & Numeracy – Adult
  • 9/10 Literacy & Numeracy – Con Ed.
  • 9/10 Literacy & Numeracy – Fully High Credit
  • 9/10 Remedial Non-Credit Literacy & Numeracy

 

Please see the enrolment register instructions for the Literacy & numeracy program for detailed instructions on how to set up these classes and students. It also includes info on how ADE is calculated and instructions for setting up parents/guardians enrolled in literacy & numeracy programs.

 

This batch file will create the Non-Credit ADE - Other programs batch file just like a night/continuing education school.  Follow the link below for detailed file specifications from the ConEd/Night school info section.
