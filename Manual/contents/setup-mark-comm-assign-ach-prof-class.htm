Setup > Marks & comments > Assign achievement profiles to classes (Schools)

Overview

Allows schools to assign achievement profiles to classes and homerooms based on the default achievement profiles. 

Clicking GO will bring the user to the following interface. 

There are 4 tabs across the top: ASSIGN DEFAULT PROFILES, ASSIGN BY CLASS, EXAMINE DISCREPANCIES*, and UNASSIGNED**. 

 

* The number of discrepancies detected. Further explained below. 

 

** The number of classes or homerooms without an achievement profile. Further explained below. 

 

The following fields can be in the grid for 'Assign default profiles':

 

Note: the 'Only apply to classes without achievement profile' check box option is unchecked by default. 

 

A DONE button is found at the bottom and will bring up a confirmation message when selected. 

DISCARD CHANGES will return to the main Marks & comments landing page when selected; without saving any changes. 

 

An ASSIGN button is found at the bottom and will assign the selected achievement profiles to the default classes defined by the Central office. 

 

If ASSIGN is selected and the 'Only apply to classes without an achievement profile' box is checked, the selected achievement profiles will only be applied to classes that do not have one. 

 

The ASSIGN button is disabled until at least one profile is selected from the list. 

 

Assigning by Class

Selecting the ASSIGN BY CLASS tab will bring up the following interface. 

 

1. Only apply to classes without achievement profile: check box; default checked

 

2. Select profile to assign: drop down containing a list of the achievement profiles shared to this school

 

  • Includes the option to 'Remove profile' which will remove the terms and topics from the selected classes/homerooms if selected.

 

Use the radio button to filter between Classes and Homerooms. 

 

The Classes or Homerooms card will display only the classes/homerooms that apply to the filter.

 

Additional filter fields include: 

1. Course code: text box; maximum character count = 7

2. Grade start with: drop down; options include all grades in the school as well as blank 

3. Grade end with: drop down; options include all grades in the school as well as blank

  • If blank is chosen, the profile will only be assigned to one grade (the one in 'Grade start with')

4. Teacher: options include all teachers in the school

 

A DONE button is found at the bottom and will bring up a confirmation message when selected. 

 

DISCARD CHANGES will return to the main Marks & comments landing page when selected; without saving any changes. 

An ASSIGN button is found at the bottom and will assign the selected achievement profiles to the default classes defined by the Central office. 

 

If ASSIGN is selected and the 'Only apply to classes without an achievement profile' box is checked, the selected achievement profiles will only be applied to classes that do not have one. 

 

The ASSIGN button is disabled until at least one profile is selected from the list. 

 

Removing Terms and Topics

If a class/homeroom has its profile changed, any terms or topics that no longer apply to it will be removed, ONLY if there are no marks, comments, topic programs, or markbooks in that term/topic.

This includes when a profile is removed using the ASSIGN BY CLASS tab.

If a class/homeroom ends up with an invalid term or topic (for example, if marks were entered and the profile was changed but could not remove the topic because marks were entered) then that class should appear in the EXAMINE DISCREPANCIES tab. 

 

Examining the Discrepancies

The EXAMINE DISCREPANCIES tab is used to find the classes or homerooms that have different term/topic configurations than what their profile has. The interface should show the following. 

The number in brackets (#) should reflect the number of classes or homerooms detected with discrepancies. 

 

It will include classes that do not have an achievement profile selected, but do have terms and topics assigned. 

 

Fields under this tab include: 

1. Class - course code or homeroom designation

2. Title - course title or blank if it is a homeroom

3. Teachers

4. Achievement profile 

5. Discrepancy type 

 

Discrepancy type text options include (any number of discrepancy types can be listed for one class): 

1. Invalid term 

2. Missing term 

3. Invalid topic 

4. Missing topic 

5. Invalid weights 

6. Invalid markbook style 

 

At the bottom, a DONE button which will bring return to the Setup > Marks & comments landing page. 

 

Unassigned Classes/Homerooms

The UNASSIGNED interface should show the following. 

The number in brackets (#) should reflect the number of classes and homerooms in the school that do not have an achievement profile assigned. 

 

Fields under this tab include: 

1. Class - course code or homeroom designation

2. Title - course title or blank if it is a homeroom

3. Teachers

4. Semester/term pattern 

5. Select achievement profile - drop down field; contains the list of achievement profiles assigned to this school 

 

A DONE button is found at the bottom and will bring up a confirmation message when selected. 

DISCARD CHANGES will return to the main Marks & comments landing page when selected; without saving any changes. 

 

An ASSIGN button is found at the bottom and will assign the selected achievement profiles to the default classes defined by the Central office. 

 

If ASSIGN is selected and the 'Only apply to classes without an achievement profile' box is checked, the selected achievement profiles will only be applied to classes that do not have one. 

 

The ASSIGN button is disabled until at least one profile is selected from the list. 

 

  Setting Achievement Profile Through the Classes Table

The achievement profile can also be set one-by-one through the Classes table and the Homerooms table.
