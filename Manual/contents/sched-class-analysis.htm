Class analysis

This tool provides a quick way to add or remove classes from the school’s list of classes.  This tool is typically used before any students are assigned into classes.   

 

For each course, the following information is presented:

 

  • Department
  • Number of classes that already exist in the school’s list of next year classes.
  • Total number of seats available for students (the sum of the seats in each of the existing classes).
  • Number of preferred requests (students who have selected this as one of their preferred course selections).
  • Difference between total seats and preferred requests.  
  • Number of alternate requests, provided for informational purposes only.
  • Icons to add or discard classes.

 

The list of courses may include all courses, or only those courses with student requests (students have selected the course as one of their preferred requests).  The view of courses may be toggled back and forth by selecting one of the following 2 icons:

 

List includes only those courses with preferred requests.  Select icon to change view.

 

List includes all courses.  Select icon to change view.

 

Selecting any row will provide access to the options Class details and Counselling list.

 

It also provides a quick way to delete all next year’s classes if a school wants to build the list of classes from the beginning.  

 

Since this tool can make significant changes to the classes, it will prompt a message suggesting the creation of a snapshot.

 

To access the class analysis tool, choose Scheduling next year, Classes. Select Go on the Class analysis card.

 

 

As a precaution, a message will appear suggesting that a snapshot be created if you plan to make extensive changes through this tool.   

 

 

For more information on snapshots, including information about restoring a snapshot, see Scheduling > Snapshot.

For each row, use the information provided to determine whether additional classes are needed, or if classes need to be removed.  This information includes the number of classes (in next year’s classes), the total seats (sum of the seats for all the classes), the number of student who have selected this course as one of their preferred requests, and the difference (Total seats – Preferred requests).  The number of alternate requests is also provided however this information typically has no influence on the number of classes the school should provide.

 

The report may be sorted by any column.  Sorting based on the difference column will help the school identify courses that have too many or few classes.  

 

 

If a course has 2 classes, both classes have 27 seats, and 78 students requested the course, the total seats = 54, and the difference = - 24 (54-78).  The school may want to consider adding another class to accommodate the 24 students over capacity.

 

 

If a course has 2 classes, one with 31 seats and one with 33 seats, and 23 students have requested this course as one of their preferred courses, the total seats = 64, and the difference = 41 (64 -23).  The school may want to consider removing a class since there is a surplus of seats.

 

 

 

You can add. edit and delete classes simply by using the icons located in the row.

      

 

 

Add a class

Selecting the ‘Add new class’ icon will create a new class using the next available section letter just as if you were adding the class from the classes table.  All class information, if known, may be entered.  Be sure to set the correct schedule set for semester/term and day/period.  The corresponding pattern for the schedule may be left empty if you plan to let the automatic scheduling tools (master builder) pick the best location for the class.

 

 

 

If the schedule is a ‘special schedule’, sets and patterns don’t apply.  These schedules are unique and must be entered (automatic scheduling tools will not schedule these classes).

 

 

Delete a class

If you want to delete a class, selecting the delete icon to see this Select class form.

 

 

Before you select the delete icon be sure that you know which class section you want  If you have doubts, row click the course and select the Class details menu to see class sections. Once you know the section you can safely delete the class.

Search

The search icon may be used to find a specific course or department.   

 

 

For example, if you search for Mathematics, the list of courses will include any course with Mathematics as the department.  The title in the card will include a count of the number of classes for that department.

 

 

 

Delete all classes

If ALL next year classes are to be removed, use the ‘Delete all classes’ option found in the More actions icon. Creating a snapshot is strongly recommended prior to using this option.

 

 

 

Export

You can export the information in the class analysis grid using csv format by selecting Export from the More menu.
