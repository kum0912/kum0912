Setup > Special education > Custom fields

Overview

Sometimes a school/board/division would like to record information about a student’s special education, but there is no field currently available for the entry of that information. Custom fields, aka user-definable fields specifically for special education related information about the student, can be created to record this information. 

Special education custom fields may be defined by the school or centrally by the board/division. All custom fields are set up in the same manner regardless of whether they are student custom fields OR staff custom fields OR special education custom fields.   

 

Clicking GO will bring the user to the following interface. 

 

The Custom fields option will show the user-defined fields in cards. In the example below, the 'General' cards are collapsed (by default all cards are expanded). The title of each card is user defined, but the General cards are the default group titles (they won't appear if there are no fields are setup in the General group).

CLOSE button on the bottom bar will return the user to the setup page (landing page with all the cards).

Adding Custom Fields

The setup of the centrally defined custom fields will determine whether the fields are student-specific, in other words, the information entered into the field is the same regardless of what school the student attends. Only the school that is considered the edsembli record owner can edit these fields.

 

These fields would appear for all students at all schools (but editable only by the edsembli record owner). If, on the other hand, these fields were defined by a school, they would only be available for the students in that school only.

Sometimes a centrally defined custom field is setup so that each school can record its own unique values for the fields in that category. Centrally defined groups can contain both school and student-specific fields.

 Editing Custom Fields

If one field needs editing, the fastest way to do this is to select the field on the card. This will open up the edit for just that one field. Entry instructions, if defined as part of the setup, will appear beside the field.

If the field has been set up with instructions to the user (to help them understand the purpose of the field or how it is to be entered), select the info icon located beside the field.

 

Note: Yes/No fields will actually have 3 possible values (Yes, No, Not assigned) and will be set using a drop down list. Not assigned will appear until the student is intentionally set to yes or no.

 

Deleting Custom Fields

Editing a custom field will show the delete icon in the top right. Central custom fields can only be deleted in Central Office and School custom fields can only be deleted in the school. 

 

DISCARD button will remove the custom field from the table. CANCEL will retain the custom field in the table.
