Account bill

To create an Account bill report choose Student reports > Fees and select  RUN on the Account bill card.

 

The Fee reports - Account bill screen will appear.

 

OPTIONS tab

Select a print order from the header drop down list. Alphabet is the default but you can also choose Class, Group, Grade, Excursion, Homeroom, Defined character, and Advisor group. See Student reports common features for more Print order information.

Include students registered - Select the registered type, both this year and next year. This year only (not returning), Next year only or Withdrawn (no longer enrolled this year).

Report options - These options are available.

Exclude zero balances

Exclude concurrent student

Exclude future dates

Filter report - If you choose to Filter by fee type, then check the box and select a Fee type from the drop down list.

 

NOTES tab

Note - Select the Include note checkbox. Then select the type of note to include from the drop down list. The note text as it will print will appear below.

 

STUDENT INFORMATION tab

If you want to include student information use the STUDENT INFORMATION tab:

Personal - Select Birthdate, Homeroom, Grade and or Teacher checkboxes.

Student number - Select the student number information relevant to your jurisdiction.

Contact information - Select Student home phone, Student email and or Lives with. If you select Lives with there are a number of additional specifications you can make.
