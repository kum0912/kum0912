 Student reports > Attendance > Individual attendance

 

 

This report shows the daily attendance for a student for the entire school year. Students with by period attendance will have their attendance reported as am/pm. Here is an example of the individual attendance report:

 

   

See Attendance reports common features for detailed information on the presentation of the student’s attendance in this report.

 

This report is run for the students with the morning/afternoon attendance method by selecting the AM/PM button on the card, and is run for the students with the ‘by period’ attendance method by selecting the BY PERIOD button on the card.

 

 

Prior to generating the report, be sure to check all of the settings to ensure you generate the right report.

 

Select a print order from the header drop down list. Alphabet is the default but you can also choose Class, Group, Grade, Excursion, Homeroom, Defined character, and Advisor group. See Student reports common features for more Print order information.

Setup

The Setup options for the monthly attendance report are explained below.

Attendance codes

Note: Students with the ‘By period’ attendance method cannot use school attendance codes in this report. Therefore, this setup option only appears when the report is run for the students with the morning/ afternoon attendance method. See the common features section titled ‘By period’ attendance method – conversion to AM/PM for additional information.

 

Set your preference for the attendance codes that are to appear on the report by selecting from these options:

 

  

School codes: This shows the exact attendance code (A –Z) assigned to the student.

 

Official codes: This shows official codes only (e.g. S – Sick recorded for a student will show as A – Absent). Each school attendance code has an equivalent official code to indicate whether the code is considered Present, Absent, Late, Suspended, or Grant day (for unexpected school closures due to weather, infrastructure issues).

 

Note: By period students may not have this option available if the report is converting the attendance to AM/PM as explained in ‘By period’ attendance method – conversion to AM/PM.

Additional information

The individual attendance report will include 3 columns for the student’s absence and late information for each month, along with either the student’s full day absences or presence as a % of possible days.

For a total of 2.5 days absent out of 18 possible school days for April (the tot column). So far this year, the student has been absent 8.5 days out of 136 possible days (the 1st acc  column). During April, the student was absent 2 full days (the F.D. column) and has had 8 full day absences since the start of the school year (the 2nd acc column). Note: ‘acc’ is the short form of accumulated.

 

It will also show the number of lates the student had for the month broken down by morning afternoon (am/pm column), total for the month (tot. column), and the accumulated number of lates since the start of the school year (acc.). Here is a description of the options available from the Additional information card:

 

Full day absence: If this option is selected, it will show the number of full days the student was absent (e.g. if the student has A/L for a day, that day is not considered a full day absence). In the example below, the student has A for 5 dates in November, so the student is absent 5 full days (F.D. column) for a total of 12 full days so far this year (the 1st acc. Column).  Note how the /A and A/ dates do not count as a full day.

 

 

Presence as a % of possible days: If this option is selected, it will convert the student’s attendance into percentage of time the student was present at school for the month and the accumulated total for the school year.

 

In the example below, the student was absent 1 day out of 18 possible days in September (the 1st acc column).  Therefore, the student was present 17/18 days. This is represented as 94% (17/18 x 100) in the % P column. At the end of November, the student accumulated 2 absences out of 54 possible days (the 1st acc column). This is represented as 96% (52 days present/54 possible days x 100).

 

Report parameters

This is where you select certain student information to be included or excluded from the report by checking/unchecking the boxes beside the options in the report parameters card shown below.

 

  

Include student details: When student details are included, this information is printed above the attendance information:

 

Address: The student’s address.

 

Parent(s)/Guardian(s): The student’s contacts that are flagged as the student’s legal guardian (first 2 if more than 2).

 

Emergency: The name and phone number of the student’s contact that is flagged as the first emergency contact. The emergency note will also appear in this area unless you check the box to exclude it.

 

Enrolment: The student’s attendance type, entry date to this school and exit date. The word “latest” will appear if the student has previous origin/retirement information at this school (e.g. previous enrolment).  These previous dates are not included on the report.

 

Include code summary: The code summary will look different depending on whether the student has the ‘by period’ or the ‘morning/afternoon’ attendance method.

 

By period: Since only the official codes appear for these students, the code summary includes only those codes (Absent, Late, Suspended, Grant). It will show the number of times the student has each code in the morning and in the afternoon as shown here:

 

 

Morning/afternoon: The code summary for a student with the ‘morning/afternoon attendance method will categorize the school codes based on the official definition of the code (e.g. whether it is an Absent, Late, Suspended, Grant day, or Present) as shown here:

 

  

Include attendance notes: The attendance notes for the current school year will be printed if this option is selected. Each note includes a date and the text of the entered note for that date.

 

Include legend: If selected, a legend explaining the calendar and attendance codes will appear below the student’s attendance. See Common features – Legends for more information.

 

Exclude emergency note: Some schools enter non-emergency information into the student’s emergency note field, and therefore, the emergency note may not be appropriate to include on this report.

Printing the report

To print, click the Printer icon that you will see in the bottom right corner.

 

Running this report centrally

When this report is run centrally (from the board/division/district), the Selected schools card will be available. Each of the selected schools will be included in the report.
