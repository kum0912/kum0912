Accessing help in edsembli SIS

Specific help will be available within edsembli SIS. When you click on the help icon on the page where you are working, the corresponding help topic from the manual will appear.

Create your own help

Sometimes, it may be beneficial to provide easy access to help documentation that has been created specifically for your users in the board/division/district. Security access may be granted to specific users to add  specific documentation, referred to as ‘Your help’.  Your help can be created for as many or as few pages as needed, and it will be as granular as the help topics provided in edsembli SIS.

  

If the user has rights to add centralized (board/district/division) documentation, the ‘Add your help’ icon will appear in the application bar if your help has not yet been defined. The Add your help icon has the plus symbol which is a web standard for ‘add’, and it is located to the right of the edsembli SIS help icon.

 

    

 

Selecting the ‘Add your help’ icon will provide a URL field where the board can define the URL containing the board specific documentation. Once the URL is defined, all users in all schools will see the ‘Your help’ icon in the application bar. The Your help icon has the star symbol which is a web standard for ‘favourites’.

 

 

If a user has access to add this board specific documentation, a ‘Change your help’ option will be available from the ‘More option’ icon in the tool bar so that the URL can be changed or removed.   

 

Selecting the Your help icon will open a new browser window at the specified URL (web page).

 

Because each unique page in edsembli SIS can be directed to a unique URL, the ‘Your help’ icon on the main edsembli SIS dashboard page can be directed to the board’s own help website (URL) providing easy access to all the board’s help topics, documentation, videos, etc.
