Teacher master timetable

The Teacher master timetable option provides access to a listing of teachers along with the classes they teach in each period during a specific semester/term/day.  By default, it will search for the teachers’ classes in all semesters, terms, and days, but it may be filtered. Select the Search button to show the teachers’ timetables.  It lists teachers in rows, and periods in columns, with the class assigned to the teacher appearing under the appropriate period column. Each class includes the course code and section letter, course title, and room.

 

To access the Teacher master timetable choose Scheduling next year > Classes. Select the GO button  on the Teacher master timetable card.

 

 

You'll see the Teacher master timetable screen:

 

 

Select the Semester, Term, Day and Teacher(s) you want to see and select the Search icon.

 

 

If you want to print a copy select the Printer icon.
