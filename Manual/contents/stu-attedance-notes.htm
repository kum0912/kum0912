Student records > Attendance > Notes

Overview

The Attendance option in a student’s record contains information related to the student’s attendance. You can enter and modify attendance from the student’s record, but typically this is not where you enter daily attendance.  If you are not already familiar with the attendance functionality, see the Attendance section of the manual.

 

The Notes option allows provides access to all the attendance notes entered for the student, including notes from previous years.  Ontario schools will also find access to Contact notes and Attendance monitoring fields.

Notes

The student's attendance notes, including notes from previous years, will appear in the Attendance notes card.  Although notes are typically entered during the daily attendance process, they may be added/edited here if needed.  Select the attendance note to access the Edit option. Use the Add icon in the tool bar to add a new note for a new date.

 

Attendance monitoring

This card appears for Ontario schools only.  

 

 

Use the Edit icon in the tool bar if the following yes/no fields need to be changed:

 

Active attendance counsellor last school year or start of this school year:  Set this field to yes if the student has an active attendance counsellor last school year or at the start of this school year. When it is set to Yes, the student will be admitted to the enrolment register on the first day of school even if the student's attendance was entered as absent.

 

Habitually absent:  This field is for the school's use only to flag students who are habitually absent.  In the past, this was a field collected by OnSIS.

Contact notes

This card appears for Ontario schools only.

 

Contact notes may be recorded for those students with an active attendance counsellor at any time during the school year. This feature is available even if the attendance counsellor field is set to No on the Attendance monitoring card.

 

The attendance counsellor must attempt to make successful two-way communication with the pupil and/or the pupil's parent/guardian every 15 day-period in order to keep the pupil on the register. See the Ministry's Enrolment Register Instructions for Elementary and Secondary Schools for more information on these requirements.  When contact is made, a 'C' will be indicated on the student's Attendance Register. Contact notes are considered when finding students with consecutive absences (e.g. a search for 15 or more consecutive absences will exclude an absent student if there is successful contact with the pupil within the 15 days).

 

If a student is absent at the start of the school year, the Ministry enrolment register requires * (not enrolled) until the first time the student shows up to school; unless the pupil was referred to an attendance counsellor in the previous school year and whose file is still active. In edsembli SIS, if the student’s contact note on the first day of school has Yes indicated in the ‘Contact with pupil’ field, the student is considered enrolled, and therefore the student's attendance register will reflect the student’s actual attendance (e.g. A rather than *).

 

If contact notes have been entered, they will appear on the card.

 

 

Use the Add icon in the tool bar to add a new note. Select an existing note to access the Edit option. The date and a note are required, however the Contact with pupil field is only set to yes if the counsellor made successful contact.
