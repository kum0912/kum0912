Overview

 

Clicking GO will bring the user to the following interface. 

 

 

The following fields are found in the Board comment bank interface: 

 

1. Number: numeric field; comment number

 

2. English comment: English comment text

 

3. French comment: French comment text

 

4. Courses: all courses that have been chosen as a filter for the comment (blank if none)

 

5. Topics: all topics that have been chosen as a filter for the comment (blank if none)

 

6. Terms: all terms that have been chosen as a filter for the comment (blank if none)

 

7. Grades: all grades that have been chosen as a filter for the comment (blank if none)

 

8. Levels: all levels that have been chosen as a filter for the comment (blank if none)

 

9. Departments: all departments that have been chosen as a filter for the comment (blank if none)

 

10. Types: all types that have been chosen as a filter for the comment (blank if none)

 

11. Categories: all categories that have been chosen as a filter for the comment (blank if none)

 

12. Schools: all schools that have this comment selected; options include:

  • number of schools, and the list of schools (e.g., 2 schools: School 1, School 2)
  • all Elementary schools
  • all Secondary schools
  • all schools
  • no schools

 

Sharing Multiple Board Comments to the Schools

Utilize the check box column on the left to select schools that will be assigned comments from the Board comment bank. 

 

 

At the bottom are buttons for ASSIGN MULTIPLE COMMENTS TO SCHOOLS and REMOVE MULTIPLE COMMENTS FROM SCHOOLS. 

 

Both buttons remain disabled until at least one comment is selected using the check boxes, as shown above. 

 

Selecting ASSIGN MULTIPLE COMMENTS TO SCHOOLS will produce a 'select schools' interface, with no schools selected. 

 

 

Choosing schools from the list will assign the selected comments to those schools (or do nothing if a certain school already has that particular comment). 

 

Selecting REMOVE MULTIPLE COMMENTS FROM SCHOOLS will produce a 'select schools' interface, with no schools selected, as shown above. 

 

Choosing schools from the list will remove the selected comments from those schools (or do nothing if a certain school already does not have that particular comment).

Edit & Delete

Row-click to produce options for EDIT and DETAILS. 

 

Selecting EDIT will produce the following interface.

 

 

 

The following field information can be modified in the EDIT window.  

 

Under the 'General' tab: 

 

1. Language radio buttons: English/French; defaults to current screen language

 

2. Comment number: required text box field

 

3. Comment text box: required field

 

4. Insert context sensitive word: drop down list with options including,

  • Name
  • Surname
  • Given name
  • Student number
  • Grade
  • He/She
  • Him/Her
  • His/Her
  • he/she
  • him/her
  • his/her
  • himself/herself
  • Q1
  • Q2
  • Q3
  • Q4

 

French options include:

  • Nom
  • Nom de famille

  • Prénom

  • numéro d'élève

  • année

  • Il/Elle
  • il/elle
  • lui/la
  • lui-même/elle-même
  • son/sa

 

Under the 'Filters' tab: 

 

1. Course filters autocomplete text box: multiple courses can be selected; access to all courses in the board

 

2. Topic filters autocomplete text box: multiple topics can be selected; access to all topics defined in Central office 

 

3. Department filters autocomplete text box: multiple departments can be selected; access to all departments defined in Central office

 

4. Term filters drop down list: multiple terms can be selected; access to all terms defined in Central office

 

5. Grade filters drop down list: multiple grades can be selected; access to all grades defined in Central office

 

6. Level filters drop down list: multiple levels can be selected; options include 1, 2, 3, 4, and R

 

7. Types filters drop down list: multiple types can be selected; options include Next Steps, Strengths and Areas of Improvement

 

8. Category filters drop down list: multiple categories can be selected; options include Application, Communication, Knowledge and Understanding, and Thinking and Inquiring

 

The bottom bar has options for PREVIOUS, NEXT, CANCEL and SAVE. 

 

PREVIOUS will save and step to the previous comment (disabled on the first comment in the list). 

 

NEXT will save and step to the next comment (disabled on the last comment in the list). 

 

CANCEL will close without saving. 

 

SAVE will save and return to the Board comment bank setup window (disabled if any of the required fields are left blank). 

 

A trash icon is accessible in the top title bar of the screen. When selected, a standard message box will confirm the discarding of the selected comment.

Importing Comments

When IMPORT is chosen from the More (ellipse icon) menu, the following interface is produced. 

 

 

1. Browse to file: an XML file can be chosen from anywhere

 

2. 'Delete existing comment bank before importing': check box field; unchecked by default

 

3. 'If comment numbers already exist' radio buttons: options include 'Add to end with new numbers' (default choice) and 'Overwrite existing'.

 

IMPORT is enabled when a file has been chosen. 

 

Selecting CANCEL will close the window and return to the Board comment bank window. 

 

Selecting IMPORT will import the selected XML file using the options selected. 

 

Exporting Comments

When EXPORT is chosen from the More (ellipse icon) menu, the following interface is produced. 

 

 

 

1. Radio button options: 

  • All comments (filtered): will export the comments based on the filter
    • No filter - all comments are exported
    • Filter - only those comments that apply to the filter are exported
  • Selected: will export just the comments that have been checked off in the list
  • Range: will export comments within the given range; if range is selected, the following drop downs will be enabled
    • From - defaulted to first comment number in the bank
    • To - defaulted to last comment number in the bank

 

CANCEL exits this screen and returns to the Board comment bank window. 

 

EXPORT will export the comment bank or portion of it
