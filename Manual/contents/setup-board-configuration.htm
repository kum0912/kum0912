Setup > Board > Board configuration


 

As part of the setup area for every school and board/division (central office), there is a configuration area that has a number of varied options that help define how the SIS will operate in each school or from the board/division. Schools will not see this card.

 

Choose Setup > Board and select GO on the Board configuration card.  

 

 

The Board configuration page will appear:

 

 

 

Options

This card displays Specialist High Skills Major information.

 

 

 

Edit options

Select the edit icon to access the Edit options form. Users with security rights can edit.

 

 

SHSM requirements for ‘other credits’ (English, Math, Science, etc):  Use code’s category for all SHSMs or Select the SHSMs that can use the code’s category  

 

Click SAVE when you have finished making changes, or CANCEL if you change your mind.

  

Configuration

This card displays the board configuration and defaults for new continuing education students/classes settings.

 

 

Edit configuration

Select the edit icon to access the Edit configuration form. Users with security rights can edit. This sample shows setup for a First Nations school that is Native funded and Band operated.

 

 

Here are the configuration fields; the options you see depend on your selection of School operating type and Funding type.

 

School status: Still in operation, Closed after (then enter a date), Opened after (then enter a date).

 

License type: This field will say Board, Division or District and is not editable.

 

Language: Select from the drop down list.

 

Funding type: Select a Funding type from the drop down list. Native, Private, Public or Catholic

    If Funding type = Native, show ‘Native Operation’.   

    If Funding type = Native, ‘Native Operation’ must have a value selected from table.

    If funding type = Public or Catholic, hide ‘Type of School’ and ‘Affiliation/membership’.

    If funding type = Native or private, show  ‘Type of School’ and ‘Affiliation/membership’.

 

Native Operation: Select from the drop down list. This field is visible only if you select Native funding type.


Type of board: Select from the drop down list. This field is visible only if you select Native or Private funding type.

 

Affiliation /membership: Select from the drop down list. This field is visible only if you select Native or Private funding type.  Save button is disabled if funding type is Native or Private and no value is entered for Affiliation/membership.
