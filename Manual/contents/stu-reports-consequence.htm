Consequence letter

To create a Consequence letter report choose Student reports > Fees and select  RUN on the Consequence letter card.

 

 

The Student reports - Consequence letter report screen will appear with 2 tabs:.

 

INCIDENT tab

In the Print report for area select a From and To date.

 

Students with consequences - Select the students you want included in the report.

 

 

 

FORMAT tab

Margins - Select Top, Left, Bottom and Right margins.

 

Other - Select a Date format from the drop down list. The Header can include school and or board information.

 

Copies for - Select who will receive copies:

Student OSR

Parent

Board (director)

Attendance counsellor

Superintendent affiliate

Teachers
