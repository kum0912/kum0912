Demographics

 

Overview

The Personal menu option in a student’s record contains student-specific information, i.e. information that is the same regardless of what school the student attends. Only the school that is the record owner can edit this information.

 

Choose Personal > Demographics from the submenu.

 

 

 

Demographics

From the main Demographics card you can select the edit icon to edit fields. The fields presented will be similar but may have variations between provinces. Here is an example:

 

 

 

Edit demographics

To make changes select the edit icon to see this Edit demographics form:

 

 

Birthdate: Enter a birthdate or select from the drop down calendar.

 

Source document to verify birth: Select from the drop down list, Birth certificate, Baptismal certificate, Canadian Certificate of Registration, Record of landing, Certificate of Canadian Citizenship, Certificate of Indian status, Permit to come into or remain, Canadian Refugee Certificate, Marriage Certificate or Marriage license, Divorce decree, Legal Change of Name documents, Adoption papers, Visa- Student, Visa- Other, Canadian Refugee Travel Document, Passport

 

Citizenship (Residence status): Select from the drop down list, Canadian citizen, Visa Student under the Immigration Act, Other Visas, None of the above and attending an overseas school, Refugee Status.

 

Born in Canada: Select Yes or no from the drop down list.

 

Province of birth: Select from the drop down list. NWT, Nunavut, BC, Alberta, Saskatchewan, Alberta, Ontario, Quebec, New Brunswick, Nova Scotia, Newfoundland, PEI.

 

Gender: Select from the drop down list. Male, Female, Undisclosed, Identify in Another Way.

 

First language: Select student’s first language from the drop down list.

 

 

When you have finished select SAVE or CANCEL if you change your mind.
