Care and/or Treatment, Custody and Correctional (CCTC) facilities

 

OnSIS collects elemental (student specific) data for secondary CTCC schools only. Aggregate totals are collected for elementary CTCC schools.

 

The CTCC info option appears for the Care and/or Treatment, Custody and Correctional (CTCC) facilities submission only.  Only secondary schools configured as CTCC schools will have access to this submission period and batch file option. This configuration is done using Setup > School > Configuration.  

 

This option is used to create batch files, using the CREATE button. The batch file is uploaded to OnSIS through the OnSIS web site.  Once OnSIS processes the batch file, a results file is created. The results file is downloaded from OnSIS and is then processed using the PROCESS RESULTS button.  

 

This section describes the creation of the batch files. See the OnSIS Important concepts section for info on Updating data in OnSIS using the ‘Create/Upload/Process’ cycle.  

 

Follow the links below for the file specifications.

 

Student class enrolment batch file*

Prior Learning Assessment and Recognition (PLAR) batch file

 

* this also includes student school enrolment, other credit and diploma info.

 

Choose OnSIS > Care and/or Treatment, Custody and Correctional (CCTC) facilities.

 

 

Select the CREATE button on the CTCC info card.  

 

 

This is the form that appears when the CREATE button is selected:

 

  

 

The OnSIS batch file options are detailed below. Prior to selecting the CREATE BATCH FILE option, determine whether other options on the form should be selected.   

 

Purge:  Y/N field. See Important concepts – Purge check boxes for more info.

 

Save results to CSV file (aka Also create CSV file): Y/N field. If set to Yes, the CREATE BATCH FILE option will create the requested batch file and the contents of the batch file will also appear in a comma delimited file (same file name as the XML batch file but with a CSV extension). Some boards/schools generate these csv files to perform data analysis on the information sent to OnSIS.
