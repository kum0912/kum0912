Attendance > Enter attendance > Lock teachers

(Available only for schools using the 'by period' attendance method).

 

The Lock teachers card has a button that toggles between locking and unlocking attendance for teachers. If a school would like to prevent teachers for entering any further attendance for today, select the LOCK ATTENDANCE button on the card.  

Selecting the LOCK ATTENDANCE  button will:

  • prevent a teacher who is using edsembli CONNECT from entering 'by period' attendance for today (the locking will be automatically reset prior to the next day's attendance needs to be entered)
  • change the button on the card to UNLOCK ATTENDANCE in case the lock option was selected in error.

 

Locking attendance is an optional feature.
