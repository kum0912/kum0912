Setup > School > Email settings

The Email Settings tab is required to define the SMTP emails settings used by edsembli SIS as well as Report Builder. The tab should only be available if the school has edsembli SIS installed or is running edsembli Enterprise. The settings are stored in the edsembli SIS settings table and can be defined at the school level or Central Office. When the settings are defined in central office those values should be propagated to each school.

 

Enter the SMTP Server (Simple Mail Transfer Protocol) that is used to send and receive mail messages. User-level client mail applications typically use SMTP only for sending messages to a mail server for relaying.

 

Information about how email is handled is set up here:

  • SMPT server
  • Client domain
  • User name and password
  • Reply address

 

The User Name is the remote desktop User Name they use to connect to our servers. Then the password which is also the password they use to connect to our servers through remote desktop. Check that the email works by clicking the Test button and entering the email.

 

From the main menu select Setup > School. This will take you to the School section.  

 

 

Select GO on the card to open the Email settings page.

 

 

Edit email settings

  • SMPT server: Text box. An SMTP server is the machine that takes care of the whole email delivery process.
  • Client domain: Text box. A domain is a form of a computer network in which all user accounts, computers, printers and other security principals, are registered with a central database located on one or more clusters of central computers known as domain controllers.
  • Use SSL: Checkbox. SSL certificates (Secure socket layers) are used to create an encrypted channel between the client and the server. Transmission of such data as credit card details, account login information, any other sensitive information has to be encrypted to prevent eavesdropping.
  • Port: Text box. A computer port is a connection point or interface between a computer and an external or internal device. Internal ports may connect such devices as hard drives. Ports allow computers to access external devices such as printers.
  • Apply to all schools check box: Check box if you want all schools in the board to have these settings.
  • Overwrite existing valued check box: Disabled until Apply to all schools check box is checked.
  • User name: Enter up to 50 characters in this text box.
  • Password: Enter up to 50 characters in this text box. use the CHANGE PASSWORD button.
  • Apply to all schools check box: Check box if you want all schools in the board to have these settings.
  • Overwrite existing valued check box: Disabled until Apply to all schools check box is checked.
  • Reply address: Enter up to 255 characters in this email address box.
  • Apply to all schools check box: Check box if you want all schools in the board to have these settings.
  • Overwrite existing valued check box: Disabled until Apply to all schools check box is checked.

 

The Apply to all schools check boxes have direct impact on what schools are allowed to view and edit. If Apply to all schools check boxes are checked schools will not have access to edit. If Apply to all schools check boxes are not checked schools will have access to edit.  

 

To access Email settings, select the Menu icon in the left corner, then select Setup, School.

 

This will take you to the Email settings card.  

 

Click the GO button on the card to access the Email settings page.
