Other school funds

When the Record funds card is selected, a list of all previously recorded other school funds will appear here.

 

Recording a new fund (add payment)

When a new collection of other school (non-student) funds is to be submitted to the office, the new funds are recorded by choosing Fees > Other school funds and selecting GO on the Record funds card:

 

 

Proceed to enter the information about the payment in the Edit payment form shown here:

 

 

You can record these fields:

 

Payer:  This is the name that describes the source of the payment.  This may be an actual person (e.g. a person who donates money to the school), or it may be a description of where the money came from (e.g. Vending machine).  Use the drop down arrow to select from previously defined payers, or use the add icon to the right of the payee field to add a new payee.  If an existing payee requires edits, select that payee from the drop down list and then use the edit icon to edit the information.

 

At a minimum, each payee must have something entered in the Full name field.  Other fields are entered if appropriate or needed for the school’s records.  Here’s the add/edit payee form:

  

Amount:  Enter the dollar amount of funds being submitted by this payer.

 

Date:  The date of the payment (it will default today’s date).

 

Payment type:  Select the type from the list of possible types.  Example types might be Cash, Cheque, E-payment, etc.  See Setup payment types for additional information.

 

Fund type:  Select the type from the list of possible other school fund types.  Example types might be Donation, Fundraiser, Vending Machine Profits, etc.  See Setup other school fund types for additional information.

 

Cheque or approval number:  If applicable, optionally record a number corresponding to the number on the cheque or other document provided for payment.

 

Comments:  A comment may be optionally entered.

 

Once the information is entered, select the SAVE button to record the new fund.  

Hint:  Use the ADD button to save the new fund and open a new payment form to record another fund.

 

Editing/voiding/refunding other school funds

From the list of other school funds, select a fund (row click) to access the Edit, Void, and Refund options as shown here.

 

The Edit option will allow the payment amount, payment date, and comment to be entered.  The Payer field may not be edited, however the information about that payee may be edited.

 

The Void option will remove the other school fund payments.  The payment will be removed from the list, however there is a record of this in the database for audit purposes if needed.   

 

The Refund option will allow the school to return some or part of the payment to the payer.  As shown below, enter the amount of the refund, date, payment type, cheque or approval number (if required), and optional comment, then select the SAVE button.  

 

Note:  if this other school fund has previous refunds, they will be listed in a separate card titled Previous refunds, along with the total amount previously refunded (prior to the entry of this new refund).
