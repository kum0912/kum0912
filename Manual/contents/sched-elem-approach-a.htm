Approach A:  Classes are created using the ASSIGNED CLASSES feature in homerooms

Elementary schools typically have all students in a homeroom taking all or the majority of  the classes at the same time with the same teacher. Each homeroom can have a list of assigned classes (e.g. the students in that homeroom can be automatically assigned to the classes when the student is placed in that homeroom). If a homeroom has multiple grade levels, each grade can have a distinct list of classes (e.g. a grade 2/3 homeroom can have a list of classes for the grade 2 students and a list of classes for the grade 3 students). If student changes homerooms during the school year, where report card information is the same between homerooms, the report card information can optionally be transferred with the student so that any new teachers will have access to the information recorded by the previous teachers.  Approach A uses the functionality found when editing a homeroom, and selecting the ASSIGNED CLASSES tab.

A1. Prepare for next year

The scheduling process beings with running the ‘initialize next year’ process.  This process duplicates the school information from this year, into the next year.  This makes re-entering last year's homerooms, advisor groups, courses, classes, and semester/term/day/period information unnecessary if they will be similar for the next year.  It also advances the students’ next year grades.  

 

 

Run the initialize next year option.  We suggest that you use the option "Set students' next year homerooms to not assigned" so that no timetables are automatically assigned to students until their homerooms are assigned.

 

 

Update the semester/term/day/period info if needed.

A2. Remove all elementary level classes

Classes from the previous school year are automatically copied over to the next year's classes.  Since these classes will be all recreated for the new school year, last year's elementary classes need to be removed.

A3. Set up homerooms and their assigned classes

i. The homeroom should be adjusted to match next year's needs using Scheduling next year > Homerooms > Homerooms. This may include:

 

  • Adding new homerooms
  • Removing homerooms no longer needed
  • Editing homerooom teacher(s)
  • Editing room and designation info

 

ii. Edit each homeroom and use the ASSIGNED CLASSES tab to create all the classes that the students in this homeroom will need. Here is an example of a homeroom with the assigned classes defined. If printed lists of the students in a homeroom are needed, use the Homeroom lists card to generate those reports. Once the students are assigned into homeroom, they will have the auto-assign classes added to their next year timetables.

A5. Define class schedule

Step 3 would have created classes that are accessible using Scheduling next year > Classes > Classes. The actual schedule (when the class is taught) was not defined in step 3 so it should be defined now. Often the actual schedule of when the subjects are taught is not formally recorded in the SIS, allowing the teachers to teach subjects based on their own schedules.  Sometimes, classes are formally scheduled to avoid teacher and room conflicts (e.g. physical education, French, music). When classes are formally scheduled, the schedule is determined manually by the school and then it is entered into the SIS.

 

  • Classes not formally scheduled into periods may be assigned a floater pattern (a user-defined day/period pattern that has no periods selected). A semester/term pattern must be assigned (typically something representing full year).

 

  • Classes scheduled into periods may be assigned to semester/term/day/period patterns (e.g. full year, period 1) but often the schedule is random and creating unique patterns would be time-consuming.  In the set up of the class, it may be flagged as requiring a special schedule which then allows the user to select the specific semesters/terms and days/periods unique to that class.

 

If new rooms are added to the school next year, or rooms are no longer available, adjust the table of rooms using Scheduling next year > Classes > Rooms.

 

Once the classes are set up, see an overall view of the teacher and room timetables using the  Teacher master timetable and Room master timetable cards.

 

 

A6. Assign other classes to students

With the 'assigned classes' feature set up for homerooms (as described above), students will already have classes in their timetables based on their homerooms. If the students take additional classes (e.g. if the homeroom students break out into different classes for rotary subjects),  or if the 'assigned classes' feature is not used, these class assignment tools may assist:

 

  • Add classes
  • Delete classes

 

 

Note: clear timetables is not recommended because it will clear out the auto-assigned classes and they will not be re-added unless the student's homerooms were to be changed.

A7. Advisor groups

If advisor groups are used, they should be adjusted to match next year's needs using Scheduling next year > Advisor groups > Advisor groups. Advisor groups are similar to homerooms where a room and teacher(s) is/are assigned to the advisor group. The name of the group can be anything the school wants. Some schools use advisor groups to indicate the guidance councilor assigned to the student. Some schools have students meet regularly in groups where the students meet with an advisory teacher to discuss general school related topics.

 

If advisor groups change from year to year, there are many different options to assist in assigning the student's next year's advisor groups. These options are more convenient than editing each student, one at a time, and there should be one method that works well for your school.  Descriptions of each option is provided on the card.

A8. Print timetables

Printing timetables for students, teachers, rooms is often not necessary in elementary schools, but the option is available if needed. Class lists are also available.

 

 

A9. Year-end roll over

The board/division/district is responsible for the actual Transfer to current process, however schools should make sure that they have run the Archive attendance & achievement process. This process may be done at any time during the scheduling process, as long as it is run after the current school year is done and no further attendance or report card data is being entered.  

 

 

Typically the roll over is done over the summer, before the students start school again in the fall. It cannot be run before the current school year is finished (e.g. it cannot be done in June), nor can it be run when the school still needs the current year's info for reporting and other processes.

 

The transfer to current process takes all the 'next year' info and copies it into 'this year'. This info includes:

  • courses
  • classes
  • rooms
  • timetables
  • students' grades, homerooms and advisor groups

 

Student who are no longer returning to the school become past students, and all other students are assigned the school year status of 'this year and next year'.

 

Once the school has transferred to current, there should be no further use of the Scheduling next year options.  All changes to data need to be done to the 'this year' info.
