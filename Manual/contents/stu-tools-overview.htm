Overview

The student tools area provides access to features that many schools find useful when changes are needed to student data.  There are tools to assist in making changes to groups of students, such as assigning homerooms, advisor groups, lockers, adding/removing classes, and applying fees.  There are also tools to import and export student data, as well as tools to analyze and adjust student data.

 

To access student tools, select the Menu icon in the left corner, then select Student tools.

 

Each student tool is presented in a card with a brief description of the tool.  Use the links below to learn more about each tool.  Select a button on the card to access the tool.
