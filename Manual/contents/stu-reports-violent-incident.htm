Violent incident report

To create a Violent incident report choose Student reports > Fees and select  RUN on the Violent incident report card.

 

The Student reports - Violent incident report screen will appear with 2 tabs:.

 

1. INCIDENT tab

In the Print report for area select a From and To date.

Students with violent incidents - Select the students you want included in the report.

 

2. FORMAT tab

Margins - Select Top, Left, Bottom and Right margins.

Other - Select a Date format from the drop down list. The Header can include school and or board information.

Copies for - Select who will receive copies:

  • Student OSR
  • Parent
  • Board (director)
  • Attendance counsellor
  • Superintendent affiliate
  • Teachers

Additional notes - Select from the Report notes available.
