Summary of achievement

This report is available for secondary schools.

 

The summary of achievement report gathers information from a number of different areas of the student’s record to provide a summary report that many guidance counsellors find useful when meeting with a student to discuss timetable changes or course selections.  All the information is available from the student’s record, but some counsellors prefer a paper copy.

 

For each student, the following information may be included:

 

  • Current classes in the student’s timetable, their credit values, and attendance (# times absent from this class / # times excused from this class)
  • Course selections for next year (choices)
  • Completed courses (transcript) information: date, course, mark, credits earned, and (c) appears if the credits are compulsory (counting towards required credits for graduation)
  • A breakdown of the transcript credits based on the student’s graduation requirements.  For each category (e.g. customizable names such as Math or Sr. Math), the following credit totals are listed: achieved, required, and remaining.

 

To create a Summary of student achievement report choose Student reports > Completed courses and select RUN on the Summary of student achievement card.

 

 

The Student reports - Achievement screen will appear.

 

 

Depending upon your province, the Print options area will provide settings that control what appears on the report (e.g. failed courses, next year’s course selections (choices), completed courses, credit breakdown, additional student information, etc). All print options/settings are saved so that they are defaulted the next time you print the report.

 

Select a print order from the header drop down list. Alphabet is the default but you can also choose Class, Group, Grade, Excursion, Homeroom, Defined character, and Advisor group. See Student reports common features for more Print order information.

 

Include students registered - Select the registered type, both this year and next year. This year only (not returning), Next year only or Withdrawn (no longer enrolled this year).

 

Semester and term - select the Semester and term from the drop down lists.

 

Options - Specify to Include course selections for students, Include with course code (then select to use Title or Number).

If you turn on the Include transcript information switch then you can choose to Include only failed courses, Include transcript credits, Requirements breakdown by course code.

If you want a Separate page for each student check this box.

 

Student information - Choose from these options: Name, Student number, Ministry number, Address, Phone number and Exceptionalities.

 

 

To print the report, click the printer icon. This will download a pdf copy of the report which you can then print.
