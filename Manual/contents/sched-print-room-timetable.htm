Scheduling next year > Print timetables > Room timetables

 

Some schools print a timetable for each room that indicates what classes and teachers will be using the room for each period.

 

Select Scheduling next year > Print timetables.

 

 

Select the GO button on the Room timetables card. This will open the room timetables screen where you can select the report options. This report is also available within the Rooms table. See the here for more info on this report.
