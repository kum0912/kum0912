Student reports > Popular > Excursion

 

Popular Reports

The student reports in the popular section include those reports that schools most often generate beyond the traditional class and timetable reports.

 

Excursion

The school can assign any group of students into an excursion list that is assigned a title that is meaningful to the school. When you select the excursion list title, the students in that excursion list will be printed in this report.

 

To create an Excursion report choose Student reports > Popular and select Go on the Excursion card.

 

 

The Student reports - Excursion screen will appear with a tabbed menu for report option selection.

 

 

Filter

 

Excursion - Select your excursion group from the drop down list set up here.

Include students registered - Select the registered type, both this year and next year. This year only (not returning), Next year only or Withdrawn (no longer enrolled this year).

Other - Specify if you want the report Double-spaced.

 

 

Name formats

 

Student name - Specify to print the student’s name using their usual name or full given and middle names. You can also include the student number on the report.

 

 

 

Student information

 

Student name only - Use the toggle switch to specify that you want Student name only. This will disable all other options below.

 

Personal - Here you can specify add additional student data such as Gender, Birthdate, Telephone number, Address, Grade Insurance number, Student picture. Use the toggle to include Homeroom. If selected you can include Number, Designation and Teacher. If you select Teacher, be sure to include the name format by selecting from the drop down list.

 

Enrolment - Specify the enrolment types, Full time/part time, Date of entry, Accumulated credits and Provincial student number.

 

Contact phone - Select contact phone numbers that you want to include: Mother, Father and or Guardian.

 

Defined character - If your school or board has defined characters you can choose to include by checking the boxes.

 

Locker - Specify locker information to include: Locker, Type of locker, Locker bank, Locker serial number, Locker combination and Locker brand.

 

 

Bus

 

Include the student’s bus Inbound and Outbound information on the report, if appropriate.

 

Inbound bus - Check First route, Second route, and Comments.

 

Outbound bus - Check First route, Second route, Late bus and comments as needed.

 

 

Font

 

Customize fonts used in the report for the header/footer, title, student name, and student information.

 

Font - For each area, you may select a unique font name, font style (e.g. regular, bold, italic, etc.), and font size. Fonts installed on the server are available for selection. This is also where the date in the footer may be turned off or set to the operating system or school date.

 

Other - Set additional options for including the date in the footer: Operating system date, Admin system date or none for no date.

You can also save space by checking the Condensed line spacing option. You can left justify student information and even indent to accommodate 3 hole punch.
