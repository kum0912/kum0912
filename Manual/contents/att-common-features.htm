The attendance applications share many common features. Attendance information is typically presented in “cards” and the same cards are found throughout the attendance options. Please take a moment to familiarize yourself with these.

Edit individual student’s attendance – morning/afternoon method

This description applies to students with the morning/afternoon attendance method.

 

Edit attendance is used when editing today’s attendance or when editing the student’s attendance for any other date in the past or the future. When this appears during the daily attendance entry, the date will be automatically set to today’s date, otherwise the date will match the date you selected to edit.

 

  

Use the drop down list to select the attendance to be assigned to the student for the morning or afternoon, and add an attendance note if needed.

 

Hint: If a student needs the same attendance code applied to both morning and afternoon, select the check box in the header to automatically check off the morning and afternoon periods. Then select the attendance code from the drop down list and it will automatically apply to both the morning and afternoon.

List view

For smaller devices, the list view of attendance may be better (the standard table vs list view icon will appear in the tool bar).

The student’s current attendance will be represented by a single letter icon. A tooltip is available to provide the full attendance code title and reason assigned to the student. Select this letter icon to see a drop down list of available attendance codes that may be assigned to the student as shown here:

 

 

The sign in/out icon will represent the current state of the student (e.g. student has no sign in/out entry for this period, student is currently signed out for this period, or the student has signed in for this period). Click the icon to change the sign in/out state.

Edit individual student’s attendance – by period method

This description applies to students with the period attendance method.

 

Edit attendance is used when editing today’s attendance or when editing the student’s attendance for any other date in the past or the future. When this appears during the daily attendance entry, the date will be automatically set to today’s date, otherwise the date will match the date you selected to edit.

 

  

Use the drop down list to select the attendance to be assigned to the student for the selected period, and add an attendance note if needed.

 

Hint: If a student needs the same attendance code applied to all periods, select the check box in the header to automatically check off all periods. Then select the attendance code from the drop down list for any period and it will automatically apply to all the periods. Similarly, you can check some of the boxes to assign the attendance reason to just those periods.

Office can lock teachers from changing attendance

The school office staff have the ability to lock the attendance so that teachers cannot change the attendance information entered by the office staff. A check box in the Lock teachers column for the period will disable the attendance entry field for teachers.

Office can sign students in and out

The school office staff also have the ability to sign a student in and out of school by checking the boxes in the In and Out columns for each period. The exact time the office signs a student in or out can be automatically added to today’s attendance note if the check box “Apply sign in/out information to the note” is checked. See Important concepts - Sign in/sign out for a detailed explanation of this feature.

 

 

Edit attendance for a class or homeroom

This applies to both morning/afternoon and by period attendance methods.

 

Whether you are entering attendance for a class or a homeroom, the applications function in the same manner except that the list of students is either the students in a homeroom or a class. The actual entry of the attendance information will reflect whether the students record attendance for the morning/afternoon or by period.

 

Entering homeroom or class attendance will present 2 cards: one card to select the homeroom or class you want to enter attendance for, and one card to show the list of students in the homeroom or class and their corresponding attendance information for that homeroom or class. The first card with the list of homerooms or classes will automatically collapse when you select the homeroom or class in step 1.  It will automatically expand, to pick another class, when you finish recording the student's attendance and select the BACK TO CLASSES button.

1. Select homeroom or class

From the list of homerooms or classes, select the homeroom or class. The selected homeroom or class will be indicated in the card in step 2.

 

 

 

 

See Important concepts – Attendance taken/not taken for additional information related to showing or hiding the classes/homerooms that have attendance already entered.

2. Enter attendance for selected homeroom or class

Once the class/homeroom is selected, the class/homeroom card is collapsed and the selected class list will appear for attendance entry. Select an attendance code for each student in the list. You can also enter Attendance notes. See Important concepts – Attendance codes and reasons for details.

 

 

 

There is a Bulk apply feature that allows you to select an attendance code and then pick the students who will get that code. See Bulk apply for information on this feature.

 

 

Hint: From the table view of students, select any student’s name open that date's attendance for the student (e.g. to see more than 1 period's attendance).

 

Bulk apply

The office entry of homeroom or class attendance will have a handy feature that is available by expanding the Bulk apply section (it is collapsed by default). If the office has a class list with 5 students who are absent for the morning and afternoon, that one class will require the attendance code to be entered 10 times using the regular attendance entry. With the bulk apply you can set default attendance and then select the check box beside each student’s name to have those defaults applied.

 

 

Follow these steps to use the bulk apply feature:

 

  1. Select the show more (expand) arrow beside the Bulk apply sub-header to see the bulk apply feature.
  2. Set the default attendance as follows:

 

Morning/afternoon attendance:

-if default attendance code is needed for both the morning and afternoon, check the box beside the whole day option and then select the attendance code.

 

 

- if the default attendance code is different for the morning and afternoon, or if only the morning or the afternoon attendance is the default, check the box beside the morning and/or afternoon and then select the attendance code(s).

 

-enter a default note if needed

 

 

By period attendance:

- select an attendance code

- enter a default note if needed

 

  1. Check that you have the “Change attendance options” set correctly. This provides some control over whether or not the new attendance is automatically applied to students. For example, if a student has a planned vacation, you may not want to overwrite the previously entered vacation attendance code with something else.

 

It will retain your preferred setting so this step may be skipped if you know that it is set correctly. Select the Change attendance options icon to access the possible options:

  

 

Select one option corresponding to how you would like the application to apply the bulk attendance entries when a student is selected in step 4. The options are explained below.

 

Change all students automatically: The student’s attendance will be changed to the attendance specified in the bulk apply area as soon as the student is selected.

 

Don’t change if previous attendance: If the student’s attendance is currently present (e.g. no previous attendance has been entered), the student’s attendance will be changed to the attendance specified in the bulk apply area as soon as the student is selected. If the student has previous attendance, the student will not be changed.

 

Stop if previous attendance & let me decide: This is the recommended setting. If the student has no previous attendance, the student’s attendance will be changed to the attendance specified in the bulk apply area as soon as the student is selected. Attendance (for the school year)

The Attendance card detailed here shows the student’s attendance information considering the current school year. The previously entered attendance will appear colour coded with a legend as explained in Important concepts – Attendance codes and reasons. Here is an example of a student with the ‘by period’ attendance method, using the table view:

 

 

 

If the student is enrolled in more than 1 school in the board/division this school year, the select schools drop down will appear above the Attendance card to provide access to attendance information recorded at the other school (if permitted).

 

Any date in the Attendance card may be selected (e.g. click the row) to access the menu options Edit and Details.

 

 

Edit will allow the editing of that day’s attendance information as explained in Edit individual student’s attendance – morning/afternoon method and Edit individual student’s attendance – by period method.

 

Details will provide access to the information in a view only format.  Although all information is visible without using the Details option, it does provide the ability to see information about how the code is defined by showing the school, official and class definitions of the code as shown here:

 

 

The Add icon in the tool bar can add a new day’s attendance.

 

 

It will default today’s date, but it may be changed to any other date in the past or future. If attendance already exists, it will appear for editing.

 

If adding the same attendance information for more than 1 consecutive day, use the date range feature to set different ‘from’ and ‘to’ dates.  Since the student may have previously entered attendance, and this attendance may be different on each date, please note the following precautions:

 

  • The student’s previously entered attendance will not appear for the student.
  • The student’s classes will not appear for reference since they may be taught in different periods on different dates (affects only those with the ‘by period’ attendance method).
  • The attendance will default “no changes” for each period or am/pm block to signify that the attendance previously recorded for that time period will not be changed unless you select an attendance code.  
  • If the attendance is changed to a different attendance code, the new attendance code will automatically overwrite any previously entered attendance codes recorded for any dates in the date range.

 

 

For each of the 2 attendance methods, the attendance information for the school year may be presented in 3 different views. Use the Change view icon to select a view type: Table view, List view or Calendar view. The Change view icon will appear as the current display type.

 

 

Table view

In the table view, the dates are in reverse chronological order (e.g. most recent first). Only dates where the student has attendance other than present are listed (e.g. dates where the student is present do not appear in the table). Use the Add icon in the tool bar to add attendance where you will have the opportunity to select the date (or range of dates) to be added. Selecting any row in the table will provide access to the Edit and Details menu options.

 

The table view for a student is shown earlier, and here is a table view for a student with the ‘morning/afternoon’ attendance method:

 

 

List view

List views are better for smaller devices. In the list view, the dates are in reverse chronological order (e.g. most recent first). Only dates where the student has attendance other than present are listed (e.g. dates where the student is present do not appear in the table). Use the Add icon in the tool bar to add attendance where you will have the opportunity to select the date (or range of dates) to be added. Selecting any row in the table will provide access to the Edit and Details menu options.

 

Here is a table view for a student with the ‘morning/afternoon’ attendance method:

 

 

 

Calendar view

The calendar view presents attendance information in a monthly or yearly calendar presentation. A monthly presentation focuses on one month, while a yearly presentation focuses on the whole school year. The view can change by changing the mode from month to year in the tool bar.

Attendance notes

Attendance notes are dated notes that are intended for the recording of attendance related information. While entering attendance, there is always the ability to add a note. There is no limit to the number or size of the notes, however, only one note exists for each date.

 

As notes are recorded during daily attendance entry, the notes become part of the Attendance notes card shown below. They are also accessible from the Attendance and Notes areas in the student’s record. Use the Add icon in the toolbar to add a new note, or select an existing note to access the Edit menu. See Attendance notes in student records for detailed instructions on notes.
