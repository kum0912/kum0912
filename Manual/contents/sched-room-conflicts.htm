Room conflicts

 

The Master builder – Atomatic tool will never create room conflicts, however, when making room or schedule changes directly in the setup of the classes, it is possible to introduce room conflicts into the master timetable.  A conflict exists when the room is scheduled to be occupied by more than 1 class at the same time.  Sometimes, a room is truly occupied by multiple classes simultaneously, such as a grade 9 and 10 music class being taught at the same time.  For these situations, the classes should be grouped in the setup of the classes, to ensure that changes to the room, teacher, or schedule will be applied to all classes in the group.  Grouped classes are not considered conflicts.

 

The Room conflicts tool will identify any rooms where more than 1 class is scheduled in the room at the same time and the classes are not grouped.  If these classes are not grouped, these conflicts should be resolved by assigning a different room, or a different schedule to the class.

 

Quick access to the Master builder – Manual tool will allow you to select another room or location in the timetable by providing all the information needed to make an informed change to the class.

 

To analyze room conflicts choose Scheduling next year > Analyze master timetable.

 

 

Select GO on the Room conflicts card and the following screen will appear:The Room conflicts card will list the following information for each room conflict:  Room and the classes that are in conflict.

 

 

If you want to schedule one of the conflicting classes into a different room or placement (e.g. change the semester/term or day/period patterns), select the row containing the conflict.

 

 

Select one of the Schedule options to open a new browser window with the ‘Master builder – Manual’ tool opened with the selected class defaulted. Hint: in master builder, select the current placement to access the tools to see available rooms for that location.

 

In the More icon of the card, there will be 2 options:  Export and Open classes table.  Export will generate the standard export of the table contents.  Open classes table will open a new browser window with next year’s classes table opened, providing a quick way to access the setup of the classes so that you may group classes that are intended to be scheduled together.
