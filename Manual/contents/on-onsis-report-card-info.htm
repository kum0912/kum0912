OnSIS > Submission Period > Report card info

The Report card info option will appear for the following submission periods:

  

  • March elementary school (publicly funded schools only)
  • June elementary school (publicly funded schools only)

 

This option is used to create batch files, using the CREATE button. The batch file is uploaded to OnSIS through the OnSIS web site.  Once OnSIS processes the batch file, a results file is created. The results file is downloaded from OnSIS and is then processed using the PROCESS RESULTS button.  

 

This section describes the Report card info batch files. See the OnSIS Important concepts section for info on Updating data in OnSIS using the ‘Create/Upload/Process’ cycle.

 

At the end of the school year (June), OnSIS collects report card info (marks and learning skills) for all students enrolled at some point during the June submission period. It also collects the grade 7 and 8 subject averages as reported on the final report cards, along with the student’s attendance and grade placement for next September. It does not collect report card comments.

 

For the March submission period only, the report card info is only collected for students who have an enrolment end date during the March submission. Report card averages are not collected during March.  Therefore, if the submission period is March, the batch file option "Grade averages and student report cards" is renamed to "Student report cards for deleted students".

The OnSIS batch file options are detailed below. Prior to selecting the CREATE BATCH FILE option, determine whether other options on the form should be selected.   

 

Purge:  Y/N field. See Important concepts - Purge check boxes for more info.

 

Report card attendance already entered manually in OnSIS: Y/N field. If set to yes, it will omit attendance data in the batch file so that it does not overwrite attendance info already recorded in OnSIS.  This would be used in very rare situations where attendance is not recorded in edsembli.

 

Students report cards for deleted students (March); Grade averages and report cards (June) batch files

(Publicly funded elementary schools only)

 

Default file name: E14_######_ReportCard_yyyymmdd_hhmm.xml

 

The report card batch files for March and June are similar, however the grade averages fields are only included in the file created for the June submission period, and the March submission includes students who have report card data but have left the school during the March submission period (e.g. left on or before March 31). This file uses the data that prints on the students Ontario provincial report cards, so if the data printed correctly on the report cards generated by the SIS, it will be reported correctly to OnSIS.
