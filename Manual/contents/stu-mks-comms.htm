Student records > Achievement > Marks & comments

 

Overview

The Marks & comments option in a student’s record is an area where you can see all the marks and comments recorded for the student.  This option is located in the Achievement submenu.

 

 

A quick link to this option in a student's record is also provided from the Achievement > Marks and Comments option available from the main menu.  If accessing this page from the Achievement menu, select the GO button in the Individual marks & comments card, then use the standard select student form to select the student.

 

  

 

Either method will take you to the Marks & comments area of the student's record to show mark and comment info from the current school year.  Previous year info is available from the student's record using Previous years > Marks & comments.

 

 

Marks and comments are recorded by the teachers, and therefore edit rights to this info should be granted to only those who can overwrite teachers marks.

 

The presentation of the student’s marks and comments is driven by the report card setup, and report card setups can vary drastically between different schools and grade levels.  The school office is responsible for this setup so the terms (columns) and topics (rows) should be familiar to the school.

 

A separate card will appear for each class (or homeroom) with terms and topics. If the teacher is done mark entry for the class and have indicated that the official marks are done, then the tool bar of the card will include the student's official mark (e.g. the mark that will get added to the student's completed courses (transcript)).

  

 

 

The student’s current classes will be visible (their cards will be expanded by default so you can see the report card marks), while the completed and dropped classes will be hidden by default (their cards will be collapsed).  Use the Show more/Hide all icons on the card to expand and collapse as needed.

 

Scroll down past the current classes, or collapse the Current classes card for quick access to the cards containing Completed classes and Dropped classes (if applicable to the student).  Expand the card to see the classes.

 

 

Each class has its own set of terms (columns) and each term is given a unique name. For example, the first report card period might be called Term 1, or November, or Report 1.

 

Each class has its own set of topics (rows)  and if needed, subtopics (rows).  For example, an elementary school may have a topic called English with three subtopics called Reading, Writing, Visual & Oral Communication, whereas many high schools classes may simply have one topic called Grade.

Example 3 -  Topic Program Codes (e.g., IEP, IPP, etc)

Topic program codes are part of the report card setup to identify students with unique or alternate programs such as IEP or IPP, ESL or EAL, or any other codes that the school/board/division/district requires.  If the program code applies to the student, it will appear on the row with the topic/subtopic where it applies.  In this example, the topic program code IEP has been setup as a possible code that can be applied to students in the language arts 5 topic.  This student has been flagged as having the IEP code for all 3 terms, and therefore the title ‘IEP’ appears for all terms in the first row.

 

 

More options

There are additional options that may be used to change the display of the information in the Marks & comments area of the student’s record.  Use the More icon in the Marks & comment card to access these additional features.

 

 

This will present the additional options as shown here:

 

 

Select ‘Mark display’ to see formatting options for the marks.  The student’s marks will re-format automatically as you change between the different options.  Click anywhere outside the more options area to close it.

If you select ‘Show markbook weights’, and there are markbook items, the denominator and weight of the assessment will appear to the right of the mark.

 

 

If you select ‘Display letter colour coding’, and the board/division/school has defined colours for each letter grade, and letter grades exist for the student, setting this option will cause the mark cells to turn the appropriate colour when a specific letter grade is recorded.

Edit marks & comments

If you have rights, you can edit marks by clicking the edit icon in the Marks & comments tool bar.  This will allows marks to be entered into the enabled cells, and when you select a comment icon, the comment may be edited, then saved.  When all edits are done, select the check mark in the tool bar to disable editing functionality to prevent accidental edits.

 

 

 

Select class tool bar to open class marks

There may be times when you are looking at one student's marks and you want to compare that student to others in the class. This is possible by selecting the tool bar of the class.

 

 

 

When you do this, it will open the class marks & comments form for the selected class.
