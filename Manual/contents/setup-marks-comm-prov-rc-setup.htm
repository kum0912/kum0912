Overview

 

 

Manitoba Provincial Report Card

The edsembli interface should detect if any of the Manitoba provincial report card terms and topics have been defined in Central office. 

 

If provincial terms and topics have not been detected, the following interface will be displayed. 

 

Tabs on the Provincial report card setup window include: 

 

1. MARKS & COMMENTS SETUP (default tab)

 

2. LOCAL OPTION LEARNING BEHAVIOURS

 

3. LOCAL OPTION COURSES

 

4. TECH COURSES

 

5. REPORT CARD TEMPLATES

 

 

Selecting DEFINE will define the achievement tables. 

 

If provincial terms and topics have been detected, the following interface will be displayed. 

 

 

Selecting VALIDATE will redefine the achievement tables. 

 

The other tabs will be accessible once the tables have been redefined. 

 

Local Option Learning Behaviours

 

Enter the titles that will be defaults for the local option learning behaviours. 

 

Text box for Local option 1 topic title: maximum character count = 255

 

Text box for Local option 2 topic title: maximum character count = 255; only enabled if characters are entered into the local option 1 topic title box.

 

Selecting DEFINE will add the titles to the six local option learning behaviour topics. 

 

Local Option Courses

 

1. Subject heading (course title will print on report card)

 

2. Academic achievement indicators: disabled until characters have been entered into the subject heading 

 

Text boxes will populate with the local option topic titles if they already exist. 

 

Selecting DEFINE will define the local option topics. 

 

Tech Courses

 

1. Subject heading (course title will print on report card)

 

2. Academic achievement indicators: disabled until characters have been entered into the subject heading 

 

Text boxes will populate with the technology topic titles if they already exist. 

 

Selecting DEFINE will define the local option topics. 

 

Report Card Templates

 

If the report card group 'Manitoba Provincial Report Cards' exists, the following will appear: 

 

  • Share each report card template to the appropriate schools
  • Notes: When sharing a combined report, you do not need to also share the individual report. To avoid confusion, only share the report cards that each school requires. 

 

If the report card group 'Manitoba Provincial Report Cards' does not exist, the following will appear: 

 

  • Report card templates not found. 
  • To obtain report card templates, please contact sis.support@edsembli.com
  • If you have an import file, import using the Import FAB. 

 

Selecting SHARE REPORTS TO SCHOOLS brings up the 'Select schools' window, with no schools selected. 

 

 

Choosing schools from the list will assign the selected reports to those schools (or do nothing if a school already has that particular report). 

 

Selecting REMOVE REPORTS FROM SCHOOLS brings up the 'Select schools' window, with no schools selected.

 

 

Choosing schools from the list will remove the selected reports from those schools (or do nothing if a school does not have that particular report). 

 

Selecting DONE closes the Provincial report card setup and returns the user to the Marks & comments landing page. 

 

 

Ontario Provincial Report Card

The edsembli interface should detect if any of the Ontario provincial report card terms and topics have been defined in Central office. 

 

If provincial terms and topics have not been detected, the following interface will be displayed. 

 

Tabs on the Provincial report card setup window include: 

 

1. MARKS & COMMENTS SETUP (default)

 

2. PROGRESS REPORT - BOARD SPACE

 

 

Comment Setup

1. Progress report subject comment parameters:

  • Collaboration of comments by teachers (all teachers accessing the same space)
  • Distinct entry of comments (non-core teachers see only their comments)
  • Only homeroom teacher enters comments (comment entry only in Language topic)

 

2. Report 1 and Report 2 comment parameters: 

  • Health and Physical Education
    • Collaboration of comments by teachers (all teachers accessing the same space)
    • Distinct entry of comments (non-core teachers see only their comments)
  • History and Geography
    • Collaboration of comments by teachers (all teachers accessing the same space)
    • Distinct entry of comments (non-core teachers see only their comments)
  • Dance, Drama, Music and Visual Arts
    • Collaboration of comments by teachers (all teachers accessing the same space)
    • Distinct entry of comments (non-core teachers see only their comments)

 

3. Kindergarten comment parameters: 

  • Arial 9
  • Arial 10
  • Arial 11
  • Arial 12

 

Selecting DEFINE will define the achievement tables. 

If provincial terms and topics have been detected, the following interface will be displayed.

 

1. Validate secondary setup: check box; default unchecked 

 

2. Validate elementary setup (excluding comment settings below): check box; default unchecked

 

3. Validate progress report subject comment parameters: 

  • Collaboration of comments by teachers (all teachers accessing the same space)
    • Choose if there is a collaborative comment parameter using term ONPRG and topics 1ENG, 1MAT, 1FRN, 1NAT, 1SCI, 1SOC, 1PHY1, 1PHY2, 1ART1, 1ART2, 1ART3, 1ART4, 1OPT
  • Distinct entry of comments (non-core teachers see only their comments)
    • Choose if there is a non-collaborative comment parameter defined for term ONPRG and topic 1MAT and 1SCI
  • Only homeroom teacher enters comments (comment entry only in Language topic)
    • Choose if there is a non-collaborative comment parameter defined for term ONPRG and topic 1ENG, but nothing from 1MAT or 1SCI

 

Radio button options are disabled until the 'Validate progress report' box is checked. 

 

4. Validate Report 1 and Report 2 comment parameters: 

  • Health and Physical Education (default unchecked)
    • Collaboration of comments by teachers (all teachers accessing the same space) - Choose if there is a collaborative comment parameter using term ONRP1 and topics 1PHY1 and 1PHY2
    • Distinct entry of comments (non-core teachers see only their comments) - Choose is there is a non-collaborative comment parameter using term ONRP1 and topic 1PHY1
  • History and Geography (default unchecked)
    • Collaboration of comments by teachers (all teachers accesssing the same space) - Choose if there is a collaborative comment parameter using term ONRP1 and topics 7HST and 7GEO
    • Distinct entry of comments (non-core teachers see only their comments) - Choose if there is a non-collaborative comment parameter using term ONRP1 and topic 7HST
  • Dance, Drama, Music and Visual Arts (default unchecked)
    • Collaboration of comments by teachers (all teachers accesssing the same space) - Choose if there is a collaborative comment parameter using term ONRP1 and topics 1ART1, 1ART2, 1ART3 and 1ART4
    • Distinct entry of comments (non-core teachers see only their comments) - Choose if there is a non-collaborative comment parameter using term ONRP1 and topic 1ART1

5. Kindergarten comment parameters: comment font size; drop down disabled until 'Kindergarten comment parameters' box is checked 

  • Arial 9
  • Arial 10
  • Arial 11
  • Arial 12

Selecting VALIDATE will redefine the achievement tables for ONTARIO PROVINCIAL TABLES.

 

Progress Report - Board Space

 

When selecting the PROGRESS REPORT - BOARD SPACE tab, the interface will look like the following. 

 

 

Selecting anywhere in the box will open a dialogue to upload an image. 

 

 

Selecting anywhere in the box will open a standard file selection window. Only one image file can be selected. 

 

CANCEL closes this window without saving the image. 

 

SAVE stores he selected image file as the Progress report 1 message, closes this window and returns to the BOARD SPACE tab where the new image is now visible. 

 

Select the image to EDIT or DELETE it (using the garbage can icon). 

 

Selecting DELETE will prompt the following confirmation window.
