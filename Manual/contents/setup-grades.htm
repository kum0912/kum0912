Setup > School or Board > Grades


The grades table lists all the grades offered in the board/division, and defines what grades are offered at each school. Since grades may be user-defined, each grade will have the ability to set a reporting grade, which is only needed if the grade is not a recognizable grade for Ministry reporting (e.g. if a school defined a grade as JR1, representing grade 1 students, the ‘Reporting grade’ would be 1, and if the table included grade 2, it would not need a reporting grade recorded.  Most schools will use official grades only, typically J, K, 1 to 12 or M, J, 1 to 12 for French schools, and therefore the Reporting grade is empty.

 

A sort order is included with the grades so that student’s grades may be automatically advanced when scheduling for next school year.  Grades will also have expiry dates so that former grades can still apply to past students but not be available for current students (e.g. grade 13).

 

School access

The board defines the grades, and the school is limited to use only those grades.

 

Choose Setup > Board or Setup > School and select GO on the Grades card.  

 

 

The Setup - Grades page will appear.

 

 

 

The grades table lists all the grades offered in the board/division (as defined by the board), and it shows the school the grades that are offered at the school.  

 

 

Board access

The board defines the grades, and the school is limited to use only those grades.

 

From the main menu select Setup > Board. This will take you to the Board section.  

 

 

Select GO on the card to open the Grades page.

 

 

 

Add grade

To add a grade to the table of grades select the add icon in the Grades header.

 

Grade: Enter up to 3 characters in this required field

 

Title: Enter up to 150 characters in this optional field

 

Sort order: This field is included with the grades so that student’s grades may be automatically advanced when scheduling for next school year.

 

Report as a different grade: This is an optional field that user can record the official ministry grade if the grade is not recognized as an official grade (e.g. grade JR1 could have 1 in the Report as different grade field).

 

Effective/Expirt date: Enter dates or select from the calendar to show when the grade goes into effect and when/if it expires.

 

Select SAVE or CANCEL if you change your mind.

 

 

 

Edit grade

To edit a grade select the edit icon in the Grades header. Using the edit icon will open the first grade in the list. From there you can use the PREVIOUS and NEXT icons to navigate through the grades. To open a specific grade row click and select the Edit menu.

 

 

The edit fields are the same as described in Add grade.

 

 

 

View details

To view grade details row click and select the Details menu.

 

 

You'll see a Details card with a summary of each field.

 

 

Select grades for schools

Once you have set up the table of grades you can now specify the grades are used at each school.  Select the edit icon in the Grades offered at each school then check the grade boxes that apply to the list of schools.

 

Search

Enter as much of the school name as you know and the results will be displayed automatically.

 

 

Export

Use the more menu to export a list of schools and corresponding grades. This will generate a Microsoft Excel file similar to this example.
