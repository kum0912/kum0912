Import student pictures

Overview

Note: The student pictures (you can also do this with staff pictures) have to be named as studentnumber.jpg or ministrynumber.jpg.

For instance if the student number is 00000123456, the file should be 00000123456.jpg

 

Within the import student pictures screen, you must specify how to match the pictures to student records. What you select will depend upon how the photos were labeled (named). You have two options, Student number (this tends to be the most common format used) or Ministry Number.

Next, verify that you wish to Update existing pictures (meaning, this will overwrite existing older photos) and whether or not to include deleted students (students who may have been at your school for picture day and for whom you have a photo but have since left the school).   

 

 

Next, press the icon beside Student, Pictures. This will bring you into the file manager. This is where you will upload your school’s photo file(s) so that you can import into edsembli SIS. Press the plus icon to add the file(s). One added you, press the close to return to Import student pictures.

 

 

Now, press the Import files icon located in the bottom right. This will begin the import process and send a notification to the Task Manager.

Your request will be submitted to the Task Manager. Should you choose, you can move on to other tasks. When the job is finished you will alerted and can return to the task manager to review the task details or import log. It is recommended that you preview the log pdf for any errors. 

 

 

Students now have student profile pictures imported into the student record.

 

Should you need to edit or manually add a student’s profile picture, within the student record, simply click on the picture or the silhouette icon. 

 

Once in the Add/edit profile picture card, press on the picture and then find location/path where the desired picture exists. Press the save button when done.
