Scheduling next year > Schedule classes > Unscheduled classes

An unscheduled class is a class that has not been assigned a placement. A placement defines when the class is taught using the semester/term and day/period patterns defined by the school. If a class is missing either the semester/term or day/period pattern, it is considered an unscheduled class (sometimes also referred to as an unplaced class).

Note: classes that are set up with a ‘special schedule’ are considered scheduled. Classes with multiple schedules (used on rare occasions), will only consider the first schedule to determine whether or not the class is scheduled.

At the beginning of the scheduling process, there will be many unscheduled classes. As you approach the end of the scheduling process, it is important to have no unscheduled classes. The automatic load students process that builds the students timetables will never assign students into these unscheduled class.

Use the Unscheduled classes option to see if there are any classes that are unscheduled. Choose Scheduling next year > Schedule classes.

 

Select GO on the Unscheduled classes card to see this page:

 

Select a class (row click) to access the option ‘Schedule class’. This option will open a new browser window opened to the Master builder – Manual option for this class, where you may schedule the class, teacher and room. If you schedule the class, you can use the refresh icon in the Unscheduled classes tool bar to remove the newly scheduled class(es) from the list.
