Student records >  Student records > Transportation

The transportation option provides a way to record detailed information about how a student gets to and from school. The list of possible ways that a student gets to and from school is defined by the board/division/district (Setup > Transportation > Types) and may include methods such as walk, taxi, public transport, and bus.   Each transportation type is flagged as to whether bus info is needed), and if bus info is needed, the student may have up to 2 inbound buses, 2 outbound buses, and a late bus (if available to the students involved in after school activities).  Before entering bus info in the student's record, each school will have their own list of buses that must be set up (Setup > Transportation > Inbound and Outbound). The board/division/district will also defined a list of Reasons that may be specified for each student to describe why they have the selected transportation type (e.g. lives outside walk limit, or lives inside walk limit).

 

If the student has the same inbound and outbound info, many schools only record the inbound info.

 

If a student has unique pickup or drop off addresses, this information is recorded as part of the student's address information as recorded in the Personal > Contact info area of the student's record.

 

Transportation info may be recorded for the current school year only, and therefore this option does not apply to past students. Transportation info is specific to the school, so students enrolled in multiple schools may have info recorded at each school. Integration with transportation planning applications may be available to automatically update the student's bus info.

 

The Transportation option is available from the student's record.

 

The student's transportation will appear, if previously entered. If no transportation info exists, use the Edit icon on the transportation card. Inbound and Outbound cards will appear if applicable to the student.

Use the Edit icon on any of the cards to access the transportation info which uses a tab navigation to move between GENERAL, INBOUND and OUTBOUND information. In and outbound tabs appear only if the student's in via and out via field values require bus info.

 

Note: When first entering in via and out via fields for the student, the SAVE button must be selected to save the data and then the INBOUND and OUTBOUND tabs will appear, if needed.

As you navigate between the tabs, the info on each tab will be automatically saved. The SAVE button is disabled if there is no info to save, so select the CANCEL button to leave the form.

 

GENERAL tab

The GENERAL tab contains the following fields:

In via: Select from the values in the transportation types table to describe how the student gets to school (e.g. walk, bus).  If this value is setup needed bus info, then the INBOUND tab will appear upon selecting SAVE.

Out via: Select from the values in the transportation types table to describe how the student leaves school (e.g. walk, bus).  If this value is setup needed bus info, then the OUTBOUND tab will appear upon selecting SAVE.

Reason: Select from the values in the reasons table (typically to describe why the student has the selected types).

Comment: An optional note may be entered to comment on the student's transportation to/from school.

 

INBOUND tab

The INBOUND tab contains info about the student's bus to school. The student may have up to 2 bus routes recorded here.

 

The following field are available on the INBOUND tab:

Bus 1 Route/bus: Select the student's first bus the student takes to get to school. Select from the values in the inbound bus table. The route name, number and operator are shown to help select the correct bus.

Bus 1 Stop: Optionally select the stop from the values in the inbound bus table. The stop name, location, type and time are shown to help select the correct stop.

Bus 2 Route/bus: If applicable, select the student's second bus taken to get to school.

Bus 2 Stop: If applicable, select the student's stop for the second bus to school.

Comment: An optional note may be entered to comment on the student's transportation to school.

Last modified: The date this info was last updated.

 

OUTBOUND tab

The OUTBOUND tab contains info about the student's bus home from school at the end of the day. The student may have up to 3 buses recorded here (bus 1, bus 2, and a late bus).

This tab appears if the student's out via transportation type was set up indicating that it needs bus info.  This tab is similar to the inbound tab, except that it uses the outbound table of buses, and it includes the option for a late bus. The 'late bus' is the equivalent to a 3rd bus for the student, using the same list of outbound buses and stops. The outbound comment is another comment designed for a note on the student's transportation from school at the end of the day. The last modified date is a different date specifically for the date the student's outbound info was last updated.
