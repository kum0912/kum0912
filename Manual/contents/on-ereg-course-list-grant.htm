Course lists for grant purposes

The Course list for grant purposes provides a report of Average Daily Enrolment (ADE) calculations for the school or board. This summary provides a quick overview of the ADE totals for all the continuing education registers as well as the Independent Study & e-Learning Register for Day School Pupils. It also provides class by class details of each register (in separate tabs).

 

These ADE calculations will match the summary page generated for each of the following registers:

 

  • Independent Study and e-Learning Register for Day School Pupils
  • Register of Enrolment in Continuing Education Credit Courses
  • Summer School Register
  • Register of Enrolment in Correspondence/Self-Study/e-Learning Courses
  • Register of Enrolment in the International & Indigenous Languages Elementary (IILE) Program
  • Register of Enrolment in the Literacy and Numeracy Program
  • Register of Enrolment in Adult Indigenous Language Instruction

 

The registers above include the student by student details which may be used to verify the summary information in this report.

 

The xml file created by this option can be opened in MS Excel to show the results in spreadsheet format. There is a Summary worksheet (tab) which mirrors the reporting in the Education Finance Information System (EFIS). The summary page is linked to the totals on the individual worksheets (tabs) corresponding to the different enrolment registers.

 

To access this summary report, choose Ontario Reporting > Enrolment registers > Course lists for grant purposes.

 

 

Select GO on the Course lists for grant purposes card and you will see this page:

 

 

The list of classes will default all classes in the school. You may limit the classes using the filters, or select an individual or range of classes to print.

 

Select the Print button located at the bottom of the form. Once the report has been submitted to the task queue, you may leave this application or generate other reports. You will receive notification when your report is ready. Reports are saved in file manager under Provincial > Enrolment registers.

 

Hint: See previously created reports and registers by selecting the folder icon to open file manager at this location.

 

The info generated on the register may also be exported in a CSV file by checking the option shown below. The CSV file will be generated along with the register report, as part of the submission to the task queue.   

 

 

When the report is available, download it locally.  Open MS Excel (or equivalent spreadsheet applicaton) and open the saved XML file.

 

When the report opens, use the tabs at the bottom of the page to jump to different worksheets.  There is one worksheet for each register, and a summary of all the registers.  Refer to the register links (listed earlier) for details on how these values are determined.
