Overview

Configuration options are set up here. Typically, these settings are adjusted based on the type of reporting options that a school may require, or turning on or off various features that impact how Maplewood connectEd works at this school. As part of any new school implementation, Maplewood will assist you in the configuration settings that are needed for your school. As you explore and read about features in Maplewood connectEd, the manual may refer to specific configuration setup options located here.

 

To access the configuration options, select the Menu icon in the left corner, then select Setup, then School setup.

 

  

This will take you to the School setup section where you will find the Configuration card.

 

 

Click the GO button on the Setup - Configuration page, where you will see setup information on separate cards.

 

Each card is described below. Use the Edit icon in the card’s tool bar to make changes to the fields in each card.

Options

The Options card will show the following fields divided into 2 subheadings:  General and Address verification:

 

General

Homeroom internal transfer: Set to yes if you want an internal transfer automatically recorded When a student changes homerooms.  If internal transfers exist, it is possible to determine homeroom enrolment numbers on past dates even if the students have since changed homerooms.

 

Days before recording timetable changes:  The number of school days to wait before recording add/drop dates for timetable changes.  Enter 1 (the default value) to record the actual date the student added or dropped the class starting after the first day of school. If there are 200 school days, and 200 is entered here, the actual dates that the student’s timetable was changed will never be recorded (it will always be the start of the school year). Elementary schools may want to set this to 200. Some schools may want to start recording add/drop dates after the first couple of weeks of school when they know that timetables are in the process of being inputted (e.g. 10 days).  Note:  number of days refers to dates in the school’s calendar (e.g. typically Monday to Friday, including non-school days such as holidays and PA days).

 

As classes are added and dropped from a student’s timetable, the dates when the classes were added and/or dropped are automatically recorded. Every class in the student’s timetable must have a corresponding “add” record.  If a student has been in a class since they started school this year, the add date is typically the first day of school. These add/drop dates are important for secondary schools but are not as important for elementary schools.

 

Timetable Changes apply to All Semesters:  (Available only if the school has multiple semesters)  Set to yes if you do not want timetable add/drops recorded until after the number of days specified into the 2nd, or 3rd, etc semester. For example, if a secondary school has 2 semesters, and 5 days is specified for “Days before recording timetable changes”, then semester 2 timetable changes for the first 5 days in second semester will recorded in the add/drop area as if the student was in the class from the start of the semester (adds will be created, drops will be removed). Changes from the 6th day on will use today’s date to record all class adds and drops.

Address verification

Enable verification: Set to yes if address verification tables have been set up and the school now wants to verify student addresses against the verification tables.

 

Allow streets outside boundary: Set to yes if a student’s address may be permitted even if it is not within this school’s boundary. Set to no if a student’s address must be within this school’s boundary.

 

Allow all addresses: Set to yes if a student’s address may be any address (allow any address even if it is not standardized).

Indigenous and Northern Affairs Canada

Indigenous and Northern Affairs Canada (INAC) has an annual census of eligible students living on reserve. All students residing on-reserve or on Crown land, in receipt of any education assistance from INAC and who meet specific requirements are to be reported on the Nominal Roll. Maplewood can assist with these reporting requirements. If this school reports to INAC, set up these fields correctly by selecting the Edit icon on the card:

 

 

Provincially certified: Set to yes if this school is provincially certified.

 

Serviced cultural education centre: Set to yes if the school is a serviced cultural education centre.

 

School programs offered: Select all the programs that are offered at this school from the following list:

 

Regular

Immersion

Bilingual

Alternative/Outreach

Adult/mature

Technical

Vocational

Training for a semiskilled trade (Quebec only)

Prework training (Quebec only)

 

School delivery methods: Select all the delivery methods offered at this school from the following list:

 

Classroom

Distance education

Virtual (internet)

Home schooled

Blended:  Classroom and virtual (internet)

Blended: Classroom and distance education

 

Native operation: Affairs (Alberta only), or if the Funding type is Native (all provinces except Alberta), this field is enabled to select from the following native operation types:

 

Band operated

Federally operated

Provincially operated

Privately operated
