Snapshots

The snapshot tool provides a way to save and restore a copy of the school’s next year’s timetable. The saved copy of the timetable is referred to as a snapshot.  Snapshots let you to experiment with the timetable knowing that you can restore the timetable if you aren't happy with the results.

 

At any time during the scheduling process, if you are going to make changes which may not be beneficial to the timetable, then take a snapshot of the timetable.

 

The snapshot includes the following information:

 

  • Next year’s classes
  • Next year’s semester/term and day/period sets and patterns
  • Next year’s student timetables

 

When a snapshot is restored, there are some additional notes that may apply in some very rare situations:

 

  • If a course has been removed from the school’s next year courses, and if a class exists in the snapshot for that removed course, that class will not be restored.
  • If a student’s course selection includes the feature where a student is assigned into a specific class section, and if that class does not exist in the snapshot, the special section linked to the student’s course selection will be removed.  The student will still have the course selection, but they may get loaded into any of the class sections because the special section link no longer exists.

 

To access the Snapshot tool choose Scheduling next year > Classes. Select the GO button on the Snapshots card.

 

 

  • CREATE button creates a new snapshot
  • RESTORE button restores a previously created snapshot.

 

Creating a snapshot

Select the CREATE button and choose to create a NEW snapshot, or overwrite an existing snapshot. If you create a NEW snapshot you will need to give it a name up to 20 characters.

 

 

You can save up to 20 snapshots, each; default names are from SNAP 1 to SNAP 20.  

 

If you are creating a new snapshot select NEW from the Snapshot drop down list and enter a name in the Name of snapshot drop down list below.

 

If you want to overwrite an existing snapshot, select an existing snapshot from the Snapshot drop down. Select the CREATE SNAPSHOT button to save the snapshot.  

 

Note: The date of the snapshot will be saved automatically so that the list of snapshots will show both the name and the date.

 

Restoring a snapshot

To overwrite next year’s master timetable with the info in a snapshot, select the RESTORE button on the Snapshots card.  This will show the Restore snapshot form.

 

Select the RESTORE button from the Snapshots card to see this form:

 

 

The most recently created snapshot will be defaulted but you can select a different snapshot from the Snapshot to drop down list. Select the RESTORE button to restore the snapshot.
