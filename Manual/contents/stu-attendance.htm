Student records > Attendance > Attendance

 Overview

The Attendance option in a student’s record contains information related to the student’s attendance. You can enter and modify attendance from the student’s record using this option, but typically this is not where you enter daily attendance. If you are not already familiar with the attendance functionality, see the Attendance section of the manual.

 

The Attendance option contains only information for the current school year, and therefore the option is not applicable to past students who attended the school in previous years.

 

Choose Attendance > Attendance from the sub-menu of the student’s record.

 

 Students attending multiple schools this year

If a student attends more than 1 school in your board/division/district in the current school year, distinct attendance information and attendance notes will be recorded at each school. You can select a different school with attendance records from the School drop down list.

Attendance

The Attendance card provides access to the student’s attendance for this school year in a card titled Attendance.  It will show period attendance or morning/afternoon attendance depending on the student's attendance method. Only the dates with attendance appear (e.g. if the student is present for a day, that date does not appear).  

 

Attendance may be added or edited for any school day (past, current, and future) as explained in Common attendance features  – Attendance (for the school year). Select the row with the attendance to access the Edit option.  Use the Add icon in the tool bar to add a new date.

 

Attendance is entered using attendance codes that are defined by the board/division/district using Setup > Attendance > Attendance codes. This setup includes the letter, the title, reasons (optional), the colour, and info on how the code is interpreted in each of the different attendance views.

 

The School codes view is the default, but you can select the view icon to choose a different view (hovering over the icons will show a tool tip message describing the 3 view icons: School codes view, Official codes view and Class absent/excused view):

Class absent/excused view

This view provides an easy view of how much time the student was not attending classroom instruction. Times when the student is absent represent those times when the student was missing from school. The times when the student is excused represent those times when the student was not in the classroom but was present to school (e.g. sport teams, field trips).

 

 

Exam day attendance

The Exam day attendance card appears If a school's calendar has dates that are set up as exam days. On exam days, regular period attendance does not apply and therefore these dates don't appear in the Attendance card. These dates appear in this card where the school can record morning/afternoon attendance for these students. Select the date to access the morning/afternoon attendance for the exam day.
