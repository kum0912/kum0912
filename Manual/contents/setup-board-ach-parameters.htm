Setup > Marks & comments > Achievement parameters (Board)

Overview

 

Clicking GO will bring the user to the following interface. 

 

 

Setup parameter is defaulted to 'All (view only)' when the main screen is opened. 

 

Additional parameter setup options include: 

 

  • Marks 
  • Comments
  • Collaborative comments
  • Programs

 

A second drop down list titled 'Achievement profile' lists all of the achievement profiles plus 'All terms/topics', which is the default. Single select from the drop down list. 'All terms/topics' means that items are not broken down by achievement profiles, and that all terms and topics appear in the interface. 

 

In the grid, each intersection of term and topic will display the following data:

 

1. Marks - title of the letter grade group assigned to the term/topic intersection, or the work Numeric, or left blank

 

2. Comments - comment dimensions (width x height), measurement units (centimetres or inches), font style and font size of the comment parameter for the topic/term intersection; an asterisk will appear if the comment parameter is part of a collaborative (grouped) comment parameter

 

3. Programs - title of each topic program code that has been assigned to the term/topic intersection

 

Selecting DONE will close the screen without saving or changing anything. 

 

Editing Mark Entry Definitions

Selecting Marks from the 'Setup parameter' drop down will show the following interface: 

 

 

Here there is only one row per topic, which is the Marks (Setup parameter) line. 

 

In each term/topic intersection is a drop down which contains all letter grade groups that have been defined, plus an option called Numeric, and one called Disabled. 

 

The 'more' icon (ellipse) presents the option to 'Copy term'. 

 

 

Drop down list for 'Copy marks settings from term:' - lists all the terms defined in the Central office

 

Drop down list for 'Copy marks settings to term:' - lists all the terms defined in the Central office

 

Note: the same term cannot be picked for both drop down lists. 

 

CANCEL closes this window and returns back to the Marks parameter view grid without changing or saving anything. 

 

COPY will confirm that all mark-entry settings be copied from one term to another. 

 

 

CANCEL will close this window and remain on the Copy term - Marks window. 

 

COPY will copy the mark entry settings from one term to another. 

 

Editing Comment Parameters

Selecting Comments from the 'Setup parameter' drop down will show the following interface: 

 

 

Here there is only one row per topic, which is the Comments (Setup parameter) line. 

 

In each term/topic intersection are fields which contain the following information for comment parameters that have been previously setup: 

 

1. Width - positive numeric value; maximum character count = 8; maximum decimal places = 3

 

2. Height - positive numeric value; maximum character count = 8; maximum decimal places = 3 

 

3. Units - drop down options of Inches and Centimetres 

 

4. Font styles - list of available font styles: Arial, Arial black, Calibri, Comic sans MS, Courier, Courier new, Georgia, Impact, Roman, Tahoma, Times new roman, Verdana 

 

5. Font size - drop down options: 8, 9, 10, 11, 12, 14, 16, 18, 20, 22, 24, 26, 28, 36, 48, 72

 

6. Asterisk - only present if that particular comment parameter is part of a collaborative comment parameter

 

  • Note: '* - Collaborative comment' line will appear only if there is a collaborative comment somewhere on the screen. 

 

The 'more' icon (ellipse) presents the option to 'Copy term'. 


 

Drop down list for 'Copy comment parameters from term:' - lists all the terms defined in the Central office

 

Drop down list for 'Copy comment parameters to term:' - lists all the terms defined in the Central office

 

Note: the same term cannot be picked for both drop down lists. 

 

CANCEL closes this window and returns back to the Comments parameter view grid without changing or saving anything. 

 

COPY will confirm that all comment parameters be copied from one term to another. 

 

 

 

CANCEL will close this window and remain on the Copy term - comment parameters window. 

 

COPY will copy the comment parameters from one term to another. 

 

Entering Collaborative Comment Parameters

Selecting Collaborative comments from the 'Setup parameter' drop down will show the following interface: 

 

 

Note: the Achievement profile drop down does not exist.

 

CANCEL will return to the Marks & comments landing page. 

 

Row-click on the grid will bring up the options DETAILS and EDIT. 

 

DETAILS will bring up a standard Details window that will include all of the fields listed on the grid.

 

Selecting EDIT will show the following interface: 

 

 

1. English/French: Default screen language. This controls whether the user is entering the English or French version of a field that has unique values for each language. Fields with underlines have unique values for each language.

 

2. Identifier: Maximum character count = 5

 

3. Title: Maximum character count = 255

 

4. Term: drop down list showing all terms defined in the central office

 

5. Description: Maximum character count = 255

 

6. Width: positive numeric value; maximum character count = 8; maximum decimal places = 3

 

7. Height: positive numeric value; maximum character count = 8; maximum decimal places = 3 

 

8. Units: drop down options of Inches and Centimetres 

 

9. Font styles: list of available font styles: Arial, Arial black, Calibri, Comic sans MS, Courier, Courier new, Georgia, Impact, Roman, Tahoma, Times new roman, Verdana 

 

10. Font size: drop down options: 8, 9, 10, 11, 12, 14, 16, 18, 20, 22, 24, 26, 28, 36, 48, 72

 

11. Topics card: lists the topics that are currently included in this collaborative comment parameter

 

Selecting the Edit icon on the Topics card will show the following interface:

 

 

Contains a list of all the topics defined at the central office. 

 

A column of check marks indicates whether each topic is included in this collaborative comment parameter.

 

If a topic is already used in another comment parameter (whether regular or collaborative), It will show in this list but the check box with disabled. 

 

CANCEL and SAVE buttons are available at the bottom of the window. 

 

CANCEL will prompt the usual confirmation message:

 

 

Choosing DISCARD CHANGES will close the Edit topics window and return to the Collaborative comment parameter window without making any changes. Choosing SAVE will save any changes and then close the Edit topics window and return to the Collaborative comment parameter window.
