Room master timetable

The Room master timetable option provides access to a listing of rooms along with the classes being taught in each period during a specific semester/term/day.  By default, it will search for the room being used in all semesters, terms, and days, but it may be filtered. Select the Search button to show the rooms’ timetables.  It lists rooms in rows, and periods in columns, with the class assigned to the room appearing under the appropriate period column. Each class includes the course code and section letter, course title, and teacher.

 

To access the Room master timetable choose Scheduling next year > Classes. Select the GO button on the Room master timetable  card.

 

 

You'll see the Room master timetable screen:


 

Select the Semester, Term, and Day you want to see and select the Search icon.

 

 

If you want to print a copy select the Printer icon.
