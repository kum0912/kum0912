Dashboard

The Dashboard is the main page that loads after you log in to edsembli SIS.

 

 

  • Provides access to the main menu by selecting the Main menu icon  in the top left corner of the app bar.
  • Provides easy access to important information such as today’s date, the current period, semester, term and day information.  If the school has recorded daily event notes, they will also appear.
  • Provides valuable information in an easy to read format.  For example, the number of students enrolled in the school today.  
  • Provides quick access to other functions with a single tap/click.  For example, click the students card to go directly to student records.
  • Dashboards will be unique to each user.  Contents of the dashboard will depend upon the user’s security rights.

Dashboard header

Main menu

 

Select the main menu in the upper left corner to open the first level of the menu.

 

 

 

Select menu text to open the corresponding page.

 

 

 

 Select a down arrow to show submenu options.

 

 

 

 

Change school/board

Use the down arrow to see a list of schools/boards. Not all users will have the security rights to perform this function.

 

 

Select a different school or the board/division/district  from the drop down menu.

 

 

 

Session time

The countdown clock shows your remaining session time.

 

 

 

 

Notifications

You'll see a number beside the notifications icon in the main header bar if there are unread notifications. If you have nothing unread there won't be a number. See more information in the Notifications section.

 

 

 

 

Account holder information

Select the account holder icon to see name and status details.

 

 

 

 

 

 

Help

Select the help icon to call up the corresponding help page.

 

 

 

Add your own help

Use the add your help icon to customize special instructions or assistance for a specific page by defining a link to a url containing the help you want the schools to see. Once your help link is defined, all schools in the board/division/district will see the icon on the specific page it was defined for, and selecting the icon will call the URL containing your help content.

 

 

 

More menu

The more menu gives you access to some advanced features.

 

 

 

 

My account: Change the system language and reset your password.

 

 

Change theme: Adjust the colour setting for the site by selecting a primary, accent and background colour. In addition, use the switch to use dark or light theme.

 

 

Sign out: Leave the current session and open a new log in screen.

 

Support information:

edsembli info - this card shows system version information, achievement database version and the school with date and settings.

Setup - this card shows the current and scheduling setup for the school

Help contact - this card shows your support contact at the edsembli help desk and support hours.

Diagnostic information - this card shows any diagnostic information.

 

 

 

About: This cards shows your build date, copyright information and disclaimer notice.

 

 

 

Dashboard main cards

Date card: This card shows the current date, period, semester, term and day. If this is a special day e.g. holiday it will show that as well. Otherwise the display will say Normal day.

 

 

Future days: This card shows days in advance. Use the scrollbar to see 5 upcoming days and associated events. The sample below lets you know there is a scheduled Pizza lunch.

 

 

 

Students card: This card shows the total number of students enrolled in the school you are currently logged into. Use the handy student records icon to open the main student records search page.
