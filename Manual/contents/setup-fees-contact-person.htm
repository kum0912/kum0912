Setup > Fees > Contact person

Overview

 

Clicking GO will bring the user to the following interface. 

 

 

CLOSE button on the bottom bar will return the user to the setup page (landing page with all the cards).

 

 

Adding/Editing the Contact Person

Selecting the Edit (pencil) icon will allow users to add a contact person or edit the information for the current contact person. 

 

 

The Contact person fields are listed in the following order (field names are bold – do not include #s or colon), and the card title is called Edit contact person.

 

1. Surname

 

2. Given name

 

3. Phone

 

4. Ext.

 

5. Email

 

 

Selecting SAVE will save the new entry and return the user to the setup page.

 

Selecting CANCEL will not save the new entry.
