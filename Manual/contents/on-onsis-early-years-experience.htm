OnSIS > October elementary school > Early years experience

 

The Early years experience option appears for the October elementary school submission only. This option is used to create batch files, using the CREATE button. The batch file is uploaded to OnSIS through the OnSIS web site.  Once OnSIS processes the batch file, a results file is created. The results file is downloaded from OnSIS and is then processed using the PROCESS RESULTS button.  

 

This section describes the Early years experience batch file. See the OnSIS Important concepts section for info on Updating data in OnSIS using the ‘Create/Upload/Process’ cycle.

 

The file provides information about the students child care and participation in other activities prior to registering for junior or senior kindergarten, as recorded in the student’s record (under Enrolment > Early years experience). The file specifications are detailed below.

 

Select the CREATE button on the Early years experience card.  

 

 

 

Select the CREATE button to see this form:

  

 

The OnSIS batch file options are detailed below. Prior to selecting the CREATE BATCH FILE option, determine whether other options on the form should be selected.   

 

Purge:  Y/N field. See Important concepts – Purge check boxes for more info.

 

Save results to CSV file (aka Also create CSV file): Y/N field. If set to Yes, the CREATE BATCH FILE option will create the requested batch file and the contents of the batch file will also appear in a comma delimited file (same file name as the XML batch file but with a CSV extension). Some boards/schools generate these csv files to perform data analysis on the information sent to OnSIS.

 

Early years experience batch file  

Default file name: E16_######_EarlyYrsExp_yyyymmdd_hhmm.xml

 

The early years experience batch file contains info for junior and senior kindergarten students who are new to the school this submission period. It is based on the information recorded in the Enrolment > Early years experience option in the student’s record. This info is typically entered by the parent as part of the online registration process.
