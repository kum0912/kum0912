Scheduling next year > Course selections > List course selections

This report provides a list of all students who have course selections.  For each student, it will show the number of preferred course selections along with the actual course codes. Similar information is listed for the student’s alternate course selections.

 

The report may be sorted by any column and may be exported. Sorting based on the preferred requests column will help the school identify students with too many or too few courses.  

 

Select a student for quick access to the course selections area of the student’s record.  

 

To access the list of course selections, choose Scheduling next year > Course selections. Select Go on the List course selections card.

 

 

 The list of course selections will appear similar to this example below:  

 

 

 

Select any of the column headers to change the sort order. For example, sort by the ‘Preferred requests’ column to see the students in ascending or descending order based on the number of preferred course selections they have. A student in grade 10 with 3 preferred courses may indicate a problem. Similarly a student with 25 courses may also be problematic.  

 

Select a student from the list to open that student’s record in a new browser window. It will be opened to the course selections option for easy access to edit the student’s course selections, however you may browse to other options as needed.  For more information see Course selections.  

 

Select the Export icon from the app bar to export the list of course selections in csv format.
