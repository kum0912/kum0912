Ontario > Ontario reporting > Ontario Student Record (OSR) - Indigenous language card 

Overview

This page will be used to access the Indigenous language card. Users must have rights to the Indigenous language card to access the GO button. 

 

 

Selecting GO will present users with the interface shown below. Use the criteria options to select the school(s) needed. 

 

Select school group: 

  • All schools
  • Secondary schools
  • Elementary schools

 

Name of school: enter the name of the school

 

Other criteria: 

  • Lowest grade to Highest grade
  • School classification: 
    • All
    • Continuing Education
    • Education and Community Partnership Program (ECPP)
    • Regular
    • Summer

 

Include inactive schools checkbox

 

 

After users have selected the school(s) needed, users will be able to select the required student(s) using the criteria options. 

 

The 'Select by' option can be changed from the default value of Alphabet using the drop-down menu next to it. 

 

Other options include: 

  • Grade
  • Class 
  • Homeroom
  • Group 
  • Advisor group

 

Include students registered will have all options except Past selected by default. Selecting Past students disables selection methods that do not pertain to past students: 

  • Class
  • Homeroom 
  • Advisor group
  • Excursion

 

Cohort grouping is now an additional option.

 

The Print FAB will generate the Indigenous language card to be sent to the task queue. 

 

 

Selecting CLOSE will bring users back to the Ontario Student Record (OSR) landing page.
