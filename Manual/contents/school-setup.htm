Setup > School > School info


 

Information about your school is set up here. This information is often used in a variety of reports. The board will not see this card.

 

The following information should be checked for accuracy and updated as needed:

 

  • School name, number, director/superintendent, internal board ID
  • Phone number(s)
  • Email address(es)
  • Address
  • Website(s)

 

Select the main menu icon in the left corner, then select Setup > School. This will take you to the School info card.  

 

 

Select GO on the card to open the School info page.

 

 

 

School information

This card displays School name, School number, Internal school ID and Principal.

 

 

 

Edit school information

Select the edit icon to open the tabbed Edit school information page.

 

 

School name: Not editable. The school’s official school name. Contact edsembli if this requires changes.

 

School number: Not editable. The official school number assigned to this school from the province/jurisdiction. This number is used extensively to identify data from this school during federal/provincial/jurisdictional reporting.  Contact edsembli if this requires changes.

 

Principal: Not editable. The principal of the school. One staff member in the school should be assigned the role ‘Principal’ in staff records. The role field is located in the Employment, Teaching history area of staff records.

 

Internal school ID: Enter an ID or use the arrow buttons to navigate to the number you want. This is an optional 7 digit number that can be used by the board/division to identify this school for its own internal use.  Use the Edit icon on the School information card to access this field.

 

Alternate school number: If the school number is not the official number used in federal/provincial/jurisdictional reporting, an alternate school number is needed. Contact Maplewood if this number is needed or is incorrect. If an alternate school number does not exist, the field will not display. If an alternate school number exists, this is the number that will be used for some reporting purposes.

 

Click SAVE when you have finished making changes, or CANCEL if you change your mind.

 

Phone

The Phone card will list phone numbers specific to this school. It works just like student/staff/contact phone numbers where an unlimited number may be recorded, each with a phone type (e.g. School, Fax, Cell). Use the Add icon to add a new number, or select an existing number to access the standard phone number edit form.

 

Note: You can also add a phone number by selecting the add FAB in the bottom right corner of the Edit screen.

 

 

 

Add phone

Select the add icon to open the Add phone form:

 

 

 

Area code: Enter 3 digit required area code.

 

Order: Enter an order number. This is order in which the phone number appears for importance in your record.

 

Phone: Enter the required phone number using this format 123-4567.

 

Type: Select a School/Board from the drop down list.

 

Ext.: Enter a phone extension.

 

Country code: Select a Country code from the drop down list.

 

Note: Enter unlimited characters in this text field.

 

Availability

Day: Select yes or no

 

Night: Select yes or no

 

Unlisted: Select yes or no

 

Texts: Select yes or no

 

Edit phone

Edit phone is accessible in two ways:

 

  1. Select the edit icon from the heading. If you choose this method, a tabbed screen will appear with access to edit all cards. Edit a phone entry by selecting the edit icon.

 

Note: You can add a phone number by selecting the add FAB in the bottom right corner.



  1. On the main card select a phone entry and choose Edit from the menu

 

Note: To call the phone number select Dial from the menu.

 

You will see this Edit phone form.  When you edit a phone number, the fields are the same as described in the add section.

 

 

 

Email

This card displays the email(s) for the board/division and school(s). You can add and edit an email from here.

 

Note: You can also add an email by selecting the add FAB in the bottom right corner of the Edit screen.

 

 

Add email

Select the add icon to open the Add email form:

 

 

 

Email: Enter the required email.

 

Order: Enter an order number. This is order of importance in which the email number appears

 

Type: Select a School/Board from the drop down list.

 

Ext.: Enter a phone extension.

 

Note: Enter unlimited characters in this text field.

 

Availability

Day: Select yes or no

 

Night: Select yes or no

 

Edit email

Edit email is accessible in two ways:

 

  1. Select the edit icon from the heading. If you choose this method, a tabbed screen will appear with access to edit all cards. Edit an email entry by selecting the edit icon.



  2. Select an email entry and choose Edit from the menu

 

Note: To send an email select Email from the menu.

 

 

 

You will see this Edit email form.  When you edit an email, the fields are the same as described in the add section.

 

 

 

 

Address

This card displays the current address. Schools cannot add an address but may edit.

 

 

Edit address

Edit address is accessible in two ways:

 

  1. Select the edit icon from the heading. If you choose this method, a tabbed screen will appear with access to edit all cards. Edit an address entry by selecting the edit icon.



  2. Select the address entry and choose Edit from the menu

 

Notes: To see address details select the Details menu.

To open the address location in Google maps select the Map menu.

 

 

 

You will see this Edit address form.

 

 

You can edit these fields.

 

House #: Enter up to 10 characters.

 

Street: Enter up to 255 characters.

 

Unit: Select a unit from the drop down list.

 

#: is a text field.

 

City: Enter up to 50 characters.

 

Province: Enter a province up to 50 characters.

 

Postal code: Enter a postal code format A1A_1A1.

 

Country: Select from the drop down list

 

PO Box #: Enter up to 10 characters.

 

RR #: Enter a rural route up to 5 characters.

 

Compartment: Enter up to 20 characters.

 

Site: Enter up to 20 characters.

 

Quarter: Select from a drop down list.

 

Section: Enter up to a 5 character section.

 

Township: Enter up to a 50 character township.  

 

Range: Enter up to a 50 character range.

 

Meridian: Enter up to a 10 character meridian.

 

County: Select from the drop down list.

 

Emergency code: Enter up to a 100 character section.

 

Additional info: Enter unlimited text here.

 

Select SAVE after you have finished editing or CANCEL if you change your mind.

 

Website

This card displays the current website for the board/division and school(s). You can add and edit a website from here.

 

Note: You can also add a website by selecting the add FAB in the bottom right corner of the Edit screen.

 

 

 

Add website

Select the add icon to open the Add website form:

 

 

 

You can add the website here by simply entering in the name plus domain, e.g. edsembli.com. The program automatically adds http:// for you..

 

Edit website

Edit website is accessible in two ways:

 

  1. Select the edit icon from the heading. If you choose this method, a tabbed screen will appear with access to edit all cards. Edit a website entry by selecting the edit icon.



  2. Select a website entry and choose Edit from the menu

 

Note: To open the website select Open web site from the menu.

 

 

 

You will see this Edit website form where you can make changes.
