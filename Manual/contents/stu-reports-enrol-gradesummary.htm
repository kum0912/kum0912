Student reports > Enrolment > Grade summary report

 

Grade summary  

The report provides enrolment numbers for each grade in the school on a specific date. This report may run based on the students’ grades for this year or next year. If running the report for next year’s grades, the it will include all students who are still actively enrolled in the school this year or plan to attend next year.

 

To access this report, select the Menu icon in the left corner, then select Student reports, then Enrolment reports. This will take you to the Enrolment reports section where you will find the Grade summary card.

 

Choose Student reports > Enrolment > Grade summary and select THIS YEAR or NEXT YEAR on the Grade summary button.

 

 

The Student reports - Grade summary screen will appear corresponding to your selection. Both This year and Next year screens have the same options.

 

 

Report options - Use these options to specify report settings

 

Date: (Not available if running the report for next year ) The date to be used in the report. It will consider each student’s entry and exit dates to determine whether the student was enrolled in the school on the specified date. Grade internal transfers, if they exist, will be used  to determine each student’s grade on the report date.

 

Include gender breakdown: (Does not appear for elementary schools)  Y/N field to indicate whether to show the gender breakdown for each grade. If set to Y, the total number of students in the grade will appear in the first line and the #males/#females will appear in the second line.

 

Grades included in report: Select the lowest number grade level to appear in the report. Select the highest number grade level to appear in the report. Select the letter grades that are to appear in the report.

 

Separate column for special ed.: Y/N field to indicate whether the special education students are to be counted in a separate column rather than the column corresponding to their assigned grade.

 

Sample report

If the report is run centrally from the board/division, it can include many schools. If it is run from the school, statistical information for that school only is included.

 

 

The report may be formatted to include all grades or just specific grades. Special education students may optionally be counted separately rather than count based on their assigned grades.
