Attendance - Typical daily process

This is our suggested method of taking attendance.

1. Enter attendance from phone calls/notes

2. Enter attendance for a group of students/excursions

3. Attendance in each classroom is entered

4. Check that attendance has been taken for all classrooms

5. Examine unexplained absences/call home

6. Repeat

 

 1. Enter attendance from phone calls/notes

Phone calls/notes regarding student attendance are entered by the school office on an individual student basis by selecting the Individual student card (Attendance > Enter attendance > Individual student):

  

Select the GO button and then enter the student’s name using the standard select student form found in student records.

 

When the student is selected, the following information is available:

 

Edit attendance- today's date: Today’s attendance is editable directly on this card as explained in Common features – Edit individual student’s attendance – morning/afternoon method or Edit individual student’s attendance – by period method. We suggest that you avoid using the attendance code A – Absent for absences where the reason for the absence is known.  Absent is typically reserved for unexplained absences.

 

As soon as this attendance information is entered by the office, it will appear on the teachers’ attendance pages so that they will know the reason for a student’s absence.

 

 2. Enter attendance for a group of students/excursions

The school office enters attendance for groups of students with known attendance such as field trips, sport events, and bus delays. Rather than entering attendance one student at a time, use the Group/excursion card in Enter attendance:

 

Select the GO button and then specify the following information:

  1. Set the date. Today’s date will be defaulted, however, any date or date range may be selected.
  2. Select the group that requires attendance to be entered. It can be:

- all students,

- students in a grade or homeroom or class or bus,

- students assigned a specific defined character, or

- students in a school defined excursion list.

  1. Select the attendance code and when it is to appear (am/pm or period(s)).
  2. Check that you have the desired “Change attendance option” set. Select this icon to see the possible actions that can be taken when you attempt to apply attendance to a student who already has attendance entered.
  1. Select the APPLY button at the bottom of the page.

 

Note: these steps will need to be repeated if the group of students contains students on both methods of attendance. If this situation applies, MORNING/AFTERNOON and BY PERIOD tabs will be visible.

 

See Enter attendance – Group/excursion for detailed information on this feature.

 

 3. Attendance in each classroom is entered

Teachers take attendance as soon as class begins. If teachers do not enter attendance from the classroom, attendance information is typically recorded on a printed class list (see Student reports > Class lists > Class or Homeroom and these reports can include a monthly attendance entry grid). The class lists are delivered to the office for entry.

 

Regardless of who enters the attendance, the process is the same. The school office will use the Homeroom card if entering the information from a list of students in a homeroom, or the Class card if entering information from a list of students in a class. Elementary schools typically enter attendance by homeroom, while secondary schools typically enter attendance by class. Teachers enter attendance using edsembli CONNECT.

 

Notes: 

  • Some secondary schools take class and homeroom attendance because they track whether or not the student shows up for a homeroom period.

  • The Homeroom card will not appear if all students have attendance recorded using the “by period” method, and the school does not take homeroom attendance.

  • Sometimes, schools have students meet for core subjects in the morning and attendance is entered by homeroom. The students go to different classes in the afternoon so afternoon attendance is entered by class rather than by homeroom.

 

Even if all students are present, the teachers should go into the enter attendance area and select the Attendance taken icon.

 

Every effort has been made to make the entry of class/homeroom attendance easy, efficient, and powerful.  See Bulk apply to learn more about the “bulk apply” feature that allows the same attendance information to be applied to many students.

 

 4. Check that attendance has been taken for all classrooms

The office should check to see whether all teachers have taken attendance. Depending on whether attendance is entered by homeroom or by class, you would select the appropriate Homeroom or Class card as if you were entering attendance.

 

Here is an example showing the entry of attendance by class, showing period 1 classes. Note the list of period 1 classes shows classes that have attendance entered (Entered column shows Yes) and classes that have not had attendance entered (Entered column shows No).

  

The list of classes will always display the classes that have not yet had attendance taken today. There is an icon in the toolbar that will hide the classes that have Yes in the Entered column so that you only see those that are not entered. One of the following 2 icons will appear so select the icon to toggle between the 2 icons.

 

List of classes/homerooms IS NOT showing those that have attendance already taken. This is the icon you want on the toolbar when checking to see if all teachers have entered attendance. Your list of classes/homerooms will be empty when all classrooms have had their attendance entered.

 

List of classes/homerooms IS showing those that have attendance already taken along with the classes/homerooms that have not had their attendance entered.

 

Note: Schools that take morning and afternoon attendance  will notice this icon on the toolbar:

 

Reset attendance taken

 

If your school repeats the attendance taking process in the afternoon, select this icon before teachers commence afternoon attendance. It will set the Entered column to No for all homerooms or classes.

 5. Examine unexplained absences/call home

As soon as attendance is entered, the office should examine the absent students, and then follow the appropriate school protocol for determining the reason for the student’s absence. For younger students with unexplained absences, most schools are required to contact the student’s family. Select the Absent students card (Attendance > Call home > Absent students).

 

When the card is selected, it will automatically default to show today’s unexplained absences. An unexplained absence is an attendance entry of Absent with no reason provided. For example, students with the following attendance code examples would be excluded because you know the reason for their absence: ill, ill – cold, ill – flu, team event – football. Although we don’t recommend defining reasons for the Absent attendance code, if a school has defined Absent – ill, students with this code will be excluded because a reason is selected. The entry of student attendance always provides the ability to select an attendance code only or an attendance code with a reason (e.g. Absent or Absent – ill).

 

Here’s an example of the students with unexplained absences for the “by period” attendance method.

  

Use the ‘All absences’ option to see all students with any kind of attendance entry in any period or morning/afternoon. Students who are present all day (so far) are excluded.

 

As you examine the list of students, select a student to see additional information about that student. The following information will be available:

  1. Today’s attendance - so you can quickly enter any new attendance information. If you find out the reason for the student’s absence, change the attendance code from Absent (without a reason) to any other attendance code or reason. This will mean that the student will no longer appear in the list of students with unexplained absences.
  2. Contacts - so you can quickly see who you need to reach to find out why the student is not at school today.
  3. Attendance notes – so you can see notes that have been entered in the past.
  4. Attendance for the school year – so you can see the student’s attendance history for the school year.

Hint: Use the NEXT button at the bottom of the page to go to the next student in the list.

 

 6. Repeat

Repeat steps 3 to 5 for each time attendance is taken (e.g. for period 2, period 3, etc or for the afternoon if required).
