Reset class schedules

Every class has 4 fields to identify where the class fits into the master timetable.  These 4 fields represent the class schedule:

 

  • Semester/term set  (e.g. semestered or full year or term)
  • Semester/term pattern (e.g. sem 1 or sem 2 if set = semestered)
  • Day/period set (e.g. single period or double period)
  • Day/period pattern (e.g. P1 or P2 or P3 or P4 if set = single period)

 

The set and pattern names are user-defined, so the examples provided above may be different for each school. Resetting a pattern means removing the pattern assigned to the class.  

 

Use the Reset class schedules tool to remove the patterns assigned to classes, leaving only the set information in the class schedule.  The set is never removed because it clearly defines how to schedule the class and it is used by the automatic scheduling tools. With the patterns removed, the scheduling tools may be used to reschedule the classes into their best locations in the timetable.   

 

Typically, pattern information is only reset for unlocked patterns (the default setting), however it can reset both locked and unlocked patterns if required for unusual situations. When editing a class, if the pattern is entered, it may be locked. Locking patterns is used when that class must be scheduled into a specific pattern. For example, if a class must be scheduled into semester 1, it should be locked into that pattern. If it is not locked, the automatic scheduling tools may change it to semester 2 if that is a better location in the timetable. Similarly, a class may be locked into a specific day/period pattern.  

 

While resetting the pattern information, the tool can also delete the teacher and room assigned to the class.  If there is no flexibility with respect to who can teach some classes or the room of the class, this information should not be deleted.  The delete teacher and room options are provided for usual situations.   The tool also includes the ability to enter a course code format of classes to exclude (e.g. enter K* to exclude all courses starting with K).

 

Note:  resetting class schedules is typically used upon recommendation of Maplewood, or if you want to start the scheduling of classes all over again and take a different approach. The automatic scheduling tools will re-schedule a class into a better location regardless of whether that class is already assigned into a pattern or the pattern has been reset.   

 

To access the Snapshot tool choose Scheduling next year > Classes. Select the GO button on the Reset class schedules card.

 

 

 

As a precaution, you will be prompted to create a snapshot because this tool has the potential to make significant changes to the timetable. You can make a new snapshot or override an existing one, then select the CREATE SNAPSHOT button or select CANCEL if you don't want to create a snapshot.
