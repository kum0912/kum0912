Class lists

The student reports in the class lists section include those reports where students are formally organized into groups based on the student’s grade, homeroom, advisor group and class. Each school year, the student’s class information is often different. For example, a student has a grade for this school year and a grade for next school year. The student may also have different homerooms, advisor groups, and classes in their timetable. For this reason, each of these reports provides the option as to whether the report is to consider the student’s data for this year or for next year.

 

Class

Students in the same class (e.g. course and section letter) are listed together in this report, with a page break optionally inserted between each new class. A count of the number of students in each class is automatically included.

 

The class teacher and course title are automatically included, however the class schedule (when it’s taught) can be optionally included.  Classes may be sorted in alphabetically order by the class code, or in order by the teacher of the class.

 

Grouped classes are those classes where the teacher is teaching different courses at the same time but each student is enrolled in only one of the courses (e.g. a teacher is teaching a grade 10 and 11 music class together). There is a report setting that allows the grouped classes to be printed together even if they wouldn’t normally print together if their course codes are not in sequential order (e.g. the course codes are very different). When grouped, classes are printed together, it can group all the students in the different courses together in alphabetical order, otherwise they are printed as separate class lists.

 

Class reports also may be filtered by entering a course pattern (e.g. M* for all courses that start with M), section letter, period in a day (e.g. just the period 1 classes) or based on the pattern assigned to the class which defines where it is taught.

 

To create a Class report choose Student reports > Class lists and select This year or Next year on the Class card.

 

 

The Student reports - Class screen will appear (This year or Next year depending on your selection)  with a tabbed menu for report option selection.

 

 

Filter

 

Class - Select your class(es) to include; specify Starts with and Ends with or an individual class.

Include students registered - Select the registered type, both this year and next year. This year only (not returning), Next year only or Withdrawn (no longer enrolled this year).

Class filter - Select from these options:

Pattern - Select from the list.

Section - Select from the drop down list.

Period in day or Specific pattern - Choose one using the radio buttons.

Select Semester, Term, Day and Period

Include - Choose from Student information, Attendance entry grid (then select the month), or the Mark entry grid (1-10).

Sort by- Specify if you want the report sorted by teacher or class.

Other - Specify if you want additional options: Include deleted students, Double-spaced, Include class schedule in title, Page break between classes, Print grouped classes together, and print All students alphabetically.

 

 

Name formats

 

Student name - Specify to print the student’s name using their usual name or full given and middle names. You can also include the student number on the report.

 

 

 

Student information

 

Student name only - Use the toggle switch to specify that you want Student name only. This will disable all other options below.

 

Personal - Here you can specify add additional student data such as Gender, Birthdate, Telephone number, Address, Grade Insurance number, Student picture. Use the toggle to include Homeroom. If selected you can include Number, Designation and Teacher. If you select Teacher, be sure to include the name format by selecting from the drop down list.

 

Enrolment - Specify the enrolment types, Full time/part time, Date of entry, Accumulated credits and Provincial student number.

 

Contact phone - Select contact phone numbers that you want to include: Mother, Father and or Guardian.

 

Defined character - If your school or board has defined characters you can choose to include by checking the boxes.

 

Locker - Specify locker information to include: Locker, Type of locker, Locker bank, Locker serial number, Locker combination and Locker brand.

 

 

Bus

 

Include the student’s bus Inbound and Outbound information on the report, if appropriate.

 

Inbound bus - Check First route, Second route, and Comments.

 

Outbound bus - Check First route, Second route, Late bus and comments as needed.

 

 

Font

 

Customize fonts used in the report for the header/footer, title, student name, and student information.

 

Font - For each area, you may select a unique font name, font style (e.g. regular, bold, italic, etc.), and font size. Fonts installed on the server are available for selection. This is also where the date in the footer may be turned off or set to the operating system or school date.

 

Other - Set additional options for including the date in the footer: Operating system date, Admin system date or none for no date.

You can also save space by checking the Condensed line spacing option. You can left justify student information and even indent to accommodate 3 hole punch.
