Student incident report

To create a Student incident report choose Student reports > Fees and select  RUN on the Student incident report card.

 

 

The Student reports - Student incident report screen will appear with 4 tabs:.

 

INCIDENT tab

In the Print report for area select a From and To date..

 

Students with incidents - Select the students you want included in the report.

 

Summary - Select the summary options that you want included in the report:

Reported by

Date of incident

Time of incident

Police involved

Officer's name

Number of offender(s) involved

Violent incident report required

Weapon involved

Type of weapon

Parent(s) contacted

Parent(s) contacted date and time

Incident note

Location of incident

Timeframe

 

 

INFRACTIONS/CONSEQUENCES tab

Consequence information - Select the options that you want included in the report.

Duration

Effective date

End date

Resolved

Resolved date

Imposed by option has additional option to include the name as well as the job position.

 

 

 

DEMOGRAPHICS tab

Student name - Select the options that you want included in the report. If you select Use name fields then decide on the fields you want:

Surname

Given name

Middle name

Usual name

Legal surname

 

If you do not choose to use name fields then select from the predefined Name formats:

 

 

Personal - Select from these options:

Date of birth

Home phone

Grade

Age

Student number

Parent/guardian

OEN number

Homeroom

 

Address -  Select Use complete address to include all fields. If you do not choose to use complete address then select the fields that you want.

 

 

FORMAT tab

Margins - Select Top, Left, Bottom and Right margins.

 

Other - Select a Date format from the drop down list. The Header can include school and or board information.

 

Copies for - Select who will receive copies:

Student OSR

Parent

Board (director)

Attendance counsellor

Superintendent affiliate

Teachers

 

Additional notes - Select from the Report notes available.
