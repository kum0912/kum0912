Student reports > Enrolment > Age summary report

 

Age summary  

The report provides enrolment numbers for each age in the school on a specific date. This report is based on the students’ ages on the selected report date. If the report is run centrally from the board/division, it can include many schools. If it is run from the school, statistical information for that school only is included. Centrally generated reports will provide totals for secondary schools and elementary schools and overall totals for the board.

 

To create an Age summary report, choose Student reports > Enrolment and select RUN on the Age summary card.

 

The Student reports - Age summary screen will appear.

Report options - Set the following options for your report.

Date as of to be used in the report. It will consider each student’s entry and exit dates to determine whether the student was enrolled in the school on the specified date. The report date must be a date within the school’s current school year calendar.

Include gender breakdown: (Does not appear for elementary schools)  Y/N field to indicate whether to show the gender breakdown for each age. If set to Y, the total number of students in the age will appear in the first line and the #males/#females will appear in the second line.

Ages included in report:  Minimum age displayed:  Select the lowest age to appear in the report. Maximum age displayed:  Select the highest age to appear in the report.
