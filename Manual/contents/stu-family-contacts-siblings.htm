Siblings

 

Overview

Choose Family & contacts > Siblings from the submenu.

 

 

 

By tracking siblings in the same school, you can generate reports for just the oldest or youngest student in each family.  As well, you can share sibling contact information (for the students and their contacts) so that editing the information in one place automatically updates information for the all siblings.

 

Siblings may include other related students as appropriate for the situation (e.g. foster families, billet families, exchange students).

 

You can record siblings' information even if they don’t live in the same household, or share the same exact same contacts, as may be the case in separated families where one child may live with one parent but the other child lives with the other parent.  Each student has a unique set of siblings.

 

Schools may only add siblings that are students within the same school.

 

Siblings

The Siblings card will show the students who are already linked as a sibling to this student.  In this example, Rob Brady’s sibling list shows Susan Brady.

 

 

Edit siblings

To edit a sibling select the Edit icon in the tool bar and the Add/edit siblings page will appear.

 

 

Note: Use the add FAB button in the lower right corner to add a sibling to the list.

 

Select the sibling you want to edit by selecting the sibling's name (e.g. row-click). The Edit sibling form will appear:

 

 

The only field you can edit is relationship (e.g. is this a Sibling or Other relationship). Note: Non-editable fields are noted with a dotted rather than solid line.

 

Once you have made a change select SAVE or CANCEL if you change your mind.

 

If you need to delete the student from the sibling list, select the Delete icon. You will be asked for confirmation.

 

Add siblings

To add a sibling first select the Edit icon in the tool bar.

 

 

The Add/edit siblings page will appear.

 

 

Select the Add FAB button and the Search student page will appear.

 

Enter the name of the sibling to be added (enter as much or as a little of the 'surname, usual name' as you want, then select the search icon (or Enter key) to generate a list of matching students).

 

 

In the Search results area check the box beside the student(s) you're going to add.

 

 

Hint:  if you are going to add all students in the search results, select the check box in the header row (and select it again to deselect all students).

 

After you've checked all of the students you want, use the SELECT button to add students to the list of siblings.

 

Each student will be added with the relation set to Sibling.
