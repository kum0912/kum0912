Overview

As part of the class setup, each school has their own unique a table of rooms. This table lists every room that is used in the school for teaching a class, or is a room where a group of students gather (e.g. homeroom or advisor group).

 

After the initial set up, from year to year, this table may not require changes. This setup area is used under the following circumstances:

 

  • A new room is available for use in the school (i.e. expansion)
  • A room is being renamed (e.g. the physical room was previously called 1 but is now called 1-A)
  • Room timetables need to be printed for posting in the room

 

To access the rooms table , select the Menu icon in the left corner, then select Setup, then Class setup.

 

 

This will take you to the Class setup section where you will find the Rooms card.

 

 

Select either the THIS YEAR or NEXT YEAR button on the card to access the rooms that are available for the appropriate school year.

 

The rooms may be different for each of the school years to allow the school to plan for future changes.

 

Note: The initialize scheduling process automatically copies this year’s rooms over to next year’s rooms, while the year-end roll over process (transfer to current) copies next year’s rooms over to this year’s rooms.

Adding a new room

Use the Add icon in the rooms tool bar to add a new room.

 

 

  

The following fields may be entered for each room:

 

Room number: (Required) The room number should correspond to the school’s facility plans. It should only require changing if the room number changes. For example, if the door to a room has the number 1 on it, but then the same door number gets changed to 1A, then edit room 1 and change it to 1A. Changing the room number will automatically change the room number for all classes (affecting student and teacher timetable), homerooms, and advisor groups that are assigned to that room.

 

Site: Optionally select from Main or Satellite.

Room type: (Required) Select from most appropriate type for the room (e.g. Regular, Gymnasium, Library, Portable, etc).

Default capacity: (Required) Enter a number corresponding to the number of students that can occupy the room. When classes are assigned to this class, they may set a different capacity for the class, but this is the default capacity.

 

Accessible room: Y/N field to indicate if this room is accessible.

 

Room features: Optionally select from the list of room features. The room does not need to be accessible to select from this list.

 

Start date: If a room is to only be used starting on a specific date, enter the date it may start being used.

 

End date: If a room is no longer to be used starting on a specific date, enter the end date.  

 

 

Editing a room

Select (row click) the room from the list to access the Edit option.

 

 

 

Then proceed to change the room number as shown below.

 

 

While editing a room, you can navigate to the next or previous room in the list using the NEXT ROOM and PREVIOUS ROOM buttons at the bottom of page. Selecting any of the buttons at the bottom of the page will save any edits to the room number.

 

Note: Selecting a room from the list provides access to the Details option as well as the Edit option. The Details option is standard on all tables, and in this case, provides no additional information that is not already displayed on the list.

Removing a room

As a safeguard, rooms may not be removed if they are assigned to any classes, homerooms, or advisor groups for the selected school year (this or next year). To unassign students from a room, simply assign the class, homeroom and/or advisor group to the new room, then return to discard ‘old’ room. If a room is to be removed, select the room from the list to access the Edit option. From the edit room tool bar, select the Discard icon.

 

Room timetables

Some schools print a timetable for each room that indicates what classes and teachers will be using the room for each period. To do this, from the Rooms card, select the More menu to access the Print menu. Select the Print option, then select Room timetables.

 

 

The Room timetables screen will appear where you can specify the General layout, Content and Font.

 

 

Once your options have all been set click the Print icon. Now you will have to select a few more options:

 

  • Specify Single room or the room to start with, and then select the room from the drop down list.
  • If you want one timetable per page be sure to check that box.
