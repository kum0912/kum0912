Student tools > Timetables > Clear timetables 

Scheduling next year > Assign students to classes > Clear timetables

This tool provides a quick way to delete all of the classes from the timetables of a group of students. This is much faster than editing each student’s timetables, one at a time.

 

This year’s or next year’s timetables can be completely cleared of classes (e.g. students’ timetables will have no classes when this process is finished).

 

Each student has a dated record of all their timetable adds and drops (for this year’s timetable). The date each class was removed from the student’s timetable is automatically recorded, however, if the drop date is prior to the class start date, no add/drop dates will exist for the class.

 

Attendance and achievement information, where applicable, is retained for dropped classes even though they are no longer part of the student’s timetable.

 

Note: Please be sure that you understand the consequences of completely removing all classes from students’ timetables as there is no undo process.

 

To access this tool, Student tools > Timetables then select THIS YEAR or NEXT YEAR on the Clear timetables card, depending on whether you want to remove all classes from this year’s or next year’s timetables.

The Timetable - Clear timetable main screen will appear:

 

First, the group of students is selected from the standard student filter settings. In the example below, the filter is set to clear the timetables of all grades (8-13).

Include students registered - Select the registered type, both this year and next year. This year only (not returning), Next year only or Withdrawn (no longer enrolled this year).

School year - Select This year or Next year.

Options - Check that this card has the selected setting that you want for this additional field:

 

Adjust course fees where applicable: Select this option if course fees are used in your school. If course fees apply to a class, when the class is successfully deleted, the corresponding course fee will no longer apply if the drop is within the allowable refund period. The refund period is setup for each course.

 

Once the group of students is defined, and the course fee option is set correctly, select the CLEAR TIMETABLES button at the bottom of the page. Once the process is complete, the selected students’ timetables will be empty.
