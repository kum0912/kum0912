Scheduling next year > Schedule classes > Master builder - Automatic

 

Refer to the secondary scheduling overview for suggested steps to schedule the school.

 

Master builder- automatic will schedule classes into the best placement (semester/term/day/period patterns) for each class, however it is limited to the available locations where both the room and the teacher are available, and any other constraints put on the class (e.g. class may be locked into Semester 1). The best location for a class is determined by:

 

  • the location with the fewest number of student conflicts  
  • if there is a tie for fewest conflicts, the location with the lowest balance is selected.  Balance is the nominal number of students in that grade that will be scheduled into classes in that location (e.g. if 50 students requested a course with 2 classes, the nominal class size is 25 students per class).

 

Master builder – Automatic impacts the school’s master timetable only (e.g. the set up of next year’s classes). The students in each class are not updated until you run Load students, and therefore you cannot analyze conflicts arising from the new class schedules until the load students process is done.  

 

Depending on where you are in the scheduling process, first determine which classes you want to have scheduled (e.g. all the grade 12 classes).  Then follow these steps for the group of classes.

 

1. Choose Scheduling next year > Scheduling classes.  

 

Select GO on the Master builder - Automatic card to see this page:

 

 

2.  Set options

Change these 2 options, if needed:

 

Number of passes: It will default 3 passes, but may be set to any value from 1 to 5.  This controls the number of times it will cycle through all the classes, attempting to find a better placement for the class with each successive pass. If there are 10 classes to be scheduled, and 3 passes, the application will analyze class 1 to find the best location to schedule class 1 (class get scheduled), then it will schedule class 2 then 3, then 4 … then 10, at which point it has completed 1 full pass, and then it will repeat the process again for pass 2.  Multiple passes allow the automatic scheduler to re-evaluate the best location based on the adjustments to the other classes in the previous pass.

 

Periods per day a teacher can teach: It will default the number of periods in the school (e.g. if a school is set up with 4 periods per day, it will default 4).  When determining the best location to schedule a teacher, the automatic scheduler will limit the teacher to the selected # of classes.

 

3.  Specify the classes to be automatically scheduled

Select the search method from the list of available methods.  Set any other options, then refer to the hint text in the box.  Click in the box, start typing the search text, and then press the Enter key. The classes will be added to the card title ‘Classes to be scheduled’.

 

Text searches are not case specific (e.g. entering EN or en or En or eN will all find the same results, in this example, the search is for course codes or titles that start with en).  An asterisk in front of the search text (e.g. *10) will be the equivalent to a ‘contains’ search where the course code or title contains 10. Entering * is the equivalent to an ‘all’ search.

 

These are the search methods:

 

Course: In the box, type as much or as little of the course code or course title, then press the Enter key. Optionally include only those matching courses with a specific class section letter (e.g. change ‘Add class sections’ field from ‘All’ to ‘a’ to include only classes with section letter ‘a’). Also, if you want to include only those courses that have a specific # of class sections, enter that number in the ‘# of sections’ field.

 

Class: Typically this method is used to add a specific class section (e.g. MAT10b), but it will add all classes that match the entered text in the box.

 

Course grade: Select the grade from the ‘Filter’ field. Only grades applicable to the school may be selected. Then enter the course code/title to be searched in the box. Matching classes in that grade are added.

 

Department:  Select the department from the ‘Filter’ field.  Then enter the course code/title to be searched in the box. Departments are user-defined. Matching classes in that department are added.

 

Teacher:  Select the teacher from the ‘Filter’ field. Then enter the course code/title to be searched in the box.  Matching classes taught by that teacher are added.

  

Regardless of the search method, matching classes will appear in the card titled Classes to be scheduled:

 

 

Scroll horizontally to see more info about each class, including the delete icon to remove individual classes from the card. The delete icon in the tool bar will remove all classes from the card.

 

4. Select the SCHEDULE button

From the bottom bar, select the SCHEDULE BUTTON.  This will submit the scheduling job to the task queue. You may then select the CLOSE button to leave and await notification that the task has been completed.

 

 

 

5. Continuing scheduling process when task is completed

When you receive notification that the task is completed, and no warnings are generated*, you have just finished step 1 for the group of classes.  Proceed to step 2, Assign students to classes – Load students, as illustrated here:

 

 

* Sometimes, Master builder – automatic will generate a report indicated that there were unscheduled classes.  These are classes that have no available location in the timetable where both the room (or alternate room) and teacher are available.  Use the Master builder – manual option on these classes to determine if there may be changes to accommodate this class in the timetable.  This option will quickly show what other classes are occupying the room(s) and teacher’s time, or whether a constraint is limiting where the class may be scheduled.  Sometimes a different teacher or room may be needed, or a different class needs to be rescheduled to free up a location for this class.
