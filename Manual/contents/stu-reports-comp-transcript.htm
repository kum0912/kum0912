Student reports > Completed courses > School Transcripts

 

School transcripts

The ‘Completed courses’ area of each student’s record provides the ability to print a school transcript for just that student, however, there are times when groups of students require printed transcripts. This report will generate a transcript for each student in the selected group. Depending upon your province, the Print options area will provide settings that control what appears on the transcript (e.g. failed courses, only courses in specific grade ranges, issue date, etc). If you are printing on official transcript paper, there are print offset values that may be set to adjust the report vertically and horizontally on the page. All print options/settings are saved so that they are defaulted the next time you print the report.

 

 

To create a School transcripts report choose Student reports > Completed courses and select RUN on the School transcripts card.

 

 

The Student reports - Transcript screen will appear

 

 

Select a print order from the header drop down list. Alphabet is the default but you can also choose Class, Group, Grade, Excursion, Homeroom, Defined character, and Advisor group. See Student reports common features for more Print order information.

 

Include students registered - Select the registered type, both this year and next year. This year only (not returning), Next year only or Withdrawn (no longer enrolled this year).

 

Print options - Enter the Issuing authority in the text box, and select the Date of issue.

 

Settings - Use the Settings card to specify either Transcript or SHSM record information:

 

Transcript - Selecting Official transcript disables all other options except Indicate with asterisk previous school courses.

Selecting Unofficial transcript enables these options: Select a course code length from the drop down, Include grade 9/10 withdrawn courses, Include grade 9/10 failed courses, Include zero credit courses, Include requirements legend and Indicate with asterisk previous school courses.

 

 

SHSM - Select SHSM record options.

 

General/layout - Select from these options: Include students with empty transcripts, Use French form, and Print on blank paper has additional offset options.

 

 

To print the report, click the printer icon. This will download a pdf copy of the report which you can then print.
