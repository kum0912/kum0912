Student tools > Lockers > Clear lockers

Overview

Locker assignments are decided upon at the school level. Schools can clear/unassign individual or groups of lockers from students.  

 

 

Clicking RUN will bring the user to the following interface. 

 

 

Use SELECT to choose those students whose lockers you would like to clear/unassign. 

 

Use CLOSE to return to the Lockers landing page. 

 

 

Use the various Search Criteria options to search and SELECT those students whose lockers you wish to clear/assign. 

 

 

Once you have selected the intended students, you will be returned to the main Clear Lockers interface where you will see the selected students. 

 

When you have selected all of the students, use CLEAR LOCKERS to run the task. 

 

A standard prompt will confirm that you wish to clear the lockers of the selected students. 

Unassigned Lockers

If lockers have not been assigned to students, clicking RUN will bring up the following interface:

 

"There are no students in any lockers to clear."

 

 

Selecting CLOSE will bring the user back to the Lockers landing page.
