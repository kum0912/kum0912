Student tools > Lockers > Clear locks

Overview

Locker assignments are decided upon at the school level, with some schools choosing to assign school-owned locks to students as well. Schools can clear/unassign individual or groups of locks from students.  

 

Clicking RUN will bring the user to the following interface. 

 

 

Use SELECT to choose those students whose locks you would like to clear/unassign. 

 

Use CLOSE to return to the Lockers landing page. 

 

 

Use the various Search Criteria options to search and SELECT those students whose locks you wish to clear/assign. 

 

 

Once you have selected the intended students, you will be returned to the main Clear Locks interface where you will see the selected students. 

 

When you have selected all of the students, use CLEAR LOCKS to run the task. 

 

A standard prompt will confirm that you wish to clear the locks of the selected students.

 

Unassigned Locks

If locks have not been assigned to students, clicking RUN will bring up the following interface:

 

"There are no students with any school-owned locks to clear."

 

 

Selecting CLOSE will bring the user back to the Lockers landing page.
