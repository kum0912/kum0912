Scheduling next year > Course selections > Invalid prerequisites

After the students course selections are entered, if the school has set up course prerequisites, use the Invalid prerequisite option to see if any student has a course selection with invalid prerequisites.  Although the entry of the course selections will check for prerequisite courses, there may be situations where the course was recorded any ways (e.g. import course selections, office users with security rights to allow courses even if the prerequisites are not met, or prerequisite course information was added after students course selections were entered).   See Scheduling next year > Course selections > Courses for more info on prerequisites.

 

The following checks are performed  to determine whether the student has the prerequisite course (or courses):  

 

  • Student’s completed courses (transcript): If prerequisite course is found, and the mark is 50 or higher or a ‘P’ (for pass), then student has the prerequisite course.  
  • Student’s timetable: If course is in student’s timetable AND not in the student’s completed courses, it will assume the student will successfully complete the course by the end of the school year and therefore the student has met prerequisite course.
  • Student’s course selections:  If the prerequisite course is found in the student’s preferred course selections, then the student has the prerequisite course.

 

To access invalid prerequisites, choose Scheduling next year > Course selections. Select Go on the Invalid prerequisites card.
