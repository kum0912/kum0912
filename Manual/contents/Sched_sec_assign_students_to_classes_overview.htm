Scheduling next year > Assign students to classes - Overview

The Scheduling next year > Assign students to classes option is where you’ll find all the tools to assign students into specific course sections (classes) using next year’s master timetable. The load students tool will create the student’s unique timetable based on the student’s course selections. After the school is done its final run of the load students tool, additional tools exist that will bulk add classes to a group of students. Similar functionality exists for bulk removal of classes.

 

The tools to assign students to classes are:

  1. Load students:  Use Scheduling next year > Assign students to classes > Load students to build student timetables by assigning each student into classes based on the courses the student selected and the master timetable scheduled by the school.  This step is run many times through the scheduling process, and each time, the student’s timetable is cleared and rebuilt again based upon the student’s course selections.
  2. Add classes: Use Scheduling next year > Assign students to classes > Add classes to add one or more classes to the timetables of a group of students. This tool should only be used AFTER the final load students is run. This may be useful if the school adds lunch or study classes. For example, if students have no class assigned in a period, some schools like to assign a study class to the student’s timetable. If these classes exist in the classes table, use this tool to quickly assign these study classes to everyone in the school. You can list all the study classes (e.g. the study class in Semester 1 Period 1, and the study class is S1P2, and S1P3, etc), and when you use the tool to add the classes, it will only add the classes to the students if it doesn’t generate a conflict (e.g. only the students who don’t have a class in the period will be assigned into the study class).
  3. Delete classes: Use Scheduling next year > Assign students to classes > Delete classes to remove one or more classes to the timetables of a group of students. This tool should only be used AFTER the final load students is run. This may be useful if the school used placeholder classes such as a class that occupies all of semester 2 so that the student is forced into their requested classes in semester 1. The tool can remove these fictitious placeholder classes from the students’ timetables.
  4. Clear classes: Use Scheduling next year > Assign students to classes > Clear classes to remove ALL classes from the next year's timetables of a group of students. This tools doesn’t have any real practical use during the scheduling process for a school that is entirely based on unique timetables for each student, however, it may be useful for some schools if combined with the add classes tool (e.g. delete all classes from the timetables of the students in grade 7 and then add a number of classes to all student in homeroom 7A so that everyone has the same set of classes). Please note that this is just an example scenario for this tool. There are better approaches to keeping students together in the same classes (block sequencing in course setup or defining classes in homeroom setup).
