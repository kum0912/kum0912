Scheduling next year > Schedule classes > Master builder - Manual

 

 

The Master builder – Manual tool provides info about a class to help you decide the best placement (location) to schedule the class. A placement defines when the class is taught using the semester/term and day/period patterns defined by the school.    

 

This same info is used in the Master builder – Automatic tool, which is typically used first to automatically schedule a group of classes.  The ‘manual’ scheduling of classes, through this tool, is typically used for these situations:

 

  • Master builder – Automatic generated a report of unscheduled classes (classes where there is no placement available where both the teacher and room are available).
  • Analyzing conflicts shows a class that is not well scheduled (course has lots of conflicts and/or a class has low student enrolment). Note: overloaded classes are a problem, however they are scheduled into a good placement (e.g. don’t try to re-schedule these classes).  The class sections with low enrolment are poorly scheduled.

 

To help understand how this scheduling tool works, a quick examination of the information presented in the Master builder – Manual tool may be beneficial. For each class, the semester/term patterns are listed in columns, and the day/period patterns are in rows. Each intersection of a semester/term column and day/period row represents one possible placement in the master timetable.

 

 

The Master builder – Manual mode presents the info and lets you decide where to schedule the class. As explained later, you can select (click) any placement to access the tools to schedule the class into that placement, or to look at available teachers and rooms in that placement. It will also show the students who would have conflicts if the class were scheduled in that placement.

If the class is currently scheduled in a placement, that current placement will be identified with a dark background colour. A class has a current placement if it has both a semester/term pattern and a day/period pattern.

 

If the class is not scheduled in the ‘best’ placement, that best placement will be identified with a lighter background colour. The best placement for a class is the placement with the lowest # of conflicts and no icons appear (e.g. teacher and room are available in that placement). If more than 1 placement has the fewest conflicts, the placement with the lowest balance is considered the ‘best’.   

 

Every placement will include # Conflicts / # Balance.  The ‘# Conflicts’ is the number of student conflicts that will be generated if the class is scheduled in that placement.  The ‘# Balance’ is the theoretical number of students who will be scheduled into classes in that placement (actual class sizes are not known at this step, so if 100 students have requested a course and there are 4 classes, theoretically each class will have a balance of 25 students). The balance sums the theoretical number of students in classes that are at the same grade level as this class.

 

 To use Master builder – manual, follow these steps for the class to be scheduled.

 

 

1. Choose Scheduling next year > Scheduling classes.  

 

 

Select GO on the Master builder - Manual card.

 

The first class will be automatically presented, but any class may be selected from the Class field.

 

Sometimes there are classes already scheduled by the school such that they cannot be analyzed using Master builder – manual.  If one of these classes are selected, a message will appear reminding you why it may not be scheduled:

 

  • Classes is scheduled by the school using the special schedule option in the setup of the class.  Patterns are needed for the scheduling tools and these classes do not have patterns.
  • Class is scheduled manually using more than 1 schedule (sometimes called multiple occurrences). This is used in special situations, typically when a school has built the timetable by hand (e.g. a full year class that is taught in different periods in each semester).

 

2. Analyze the info  

 

Look at the number of conflicts in each possible placement. Since the automatic scheduling tools have constraints, the ‘current’ or ‘best’ placement may not necessarily be the best placement to reduce student conflicts.  If there is a placement that has a significantly lower number of student conflicts, use the info to determine if there is a way to make that placement work for this class.  Some things that may be considered:

 

  • Assigning a different teacher or room, if the teacher or room are not available in the desired placement
  • Unlocking the semester/term and/or day/period pattern to allow the class to be scheduled into a placement with a lower number of conflicts.  To do this, select the lock icon beside the pattern.   

 

As you analyze this info, remember to consider teacher constraints with respect to the number of periods they may teach each day, and whether a lunch period is required.  

 

Also consider the class info located above the grid, as shown here:

 

 

# of sections represents the # of classes of this course.

 

Student requests is the number of students who have this course as one of their preferred course selections.   

 

Residual conflicts are conflicts that exist prior to scheduling this class. No matter where this class is scheduled, these conflicts will exist because they are conflicts with other courses these students have requested.  It shows the # of single conflicts / # of multiple conflicts.  If this number is high, it might be time to re-analyze the entire school schedule by running load students and then analyzing conflicts to see if there are some other classes that should be rescheduled.

 

3. Change the class schedule/teacher/room, if needed

 

Select any of the available placements (cells in the grid) to access the scheduling tools.  If a pattern is locked, selecting a placement that is outside a locked pattern constraint will generate a warning message. For example, selecting a semester 2 placement when the class is locked into a semester 1 pattern will show this:

 

 

 

 

 

Once a placement is selected, the class info above the grid will be collapsed and the Scheduling tools card will appear as shown above.

 

If the selected placement has student conflicts, a student conflicts card will also appear showing the students with conflicts, for informational purposes only.  Conflict resolution for individual students is done after all the scheduling is done.

 

 

The scheduling tools card has 3 distinct areas.  

 

 

Schedule class: This is where you can check the box to schedule the class in the selected placement. You may also lock/unlock the semester/term and day/period here.

 

Teacher: If no teacher is assigned to the class, this section is expanded and the first available teacher in that placement is shown.  Browse the available teachers and use the teacher timetable icon to see that teacher’s timetable.  Check the box “Set as new teacher for class” if you want to make this the teacher for the class.   If a teacher was previously assigned to the class, this section is collapsed by default but may be expanded to access other available teachers if necessary.

 

Room: If no room is assigned to the class, this section is expanded and the first available room in that placement is shown.  Browse the available rooms and use the room timetable icon to see that room’s timetable.  Check the box “Set as new room for class” if you want to make this the room for the class.   If a room was previously assigned to the class, this section is collapsed by default but may be expanded to access other available rooms if necessary.

 

4. Save the changes to the class

 

When finished with the scheduling tools, remember to select the SAVE button to save changes. If no changes were made, select the X in the tool bar.
