Attendance > Reports > Monthly attendance or

Student reports > Attendance > Monthly attendance

 

 

This report lists students along with their daily attendance for any selected month.  

  

See Attendance reports common features for detailed information on the presentation of the student’s attendance in this report.

 

Choose Attendance > Reports >Monthly attendance or Student reports > Attendance > Monthly attendance.

 

This report is run for the students with the morning/afternoon attendance method by selecting the AM/PM button on the card, and is run for the students with the ‘by period’ attendance method by selecting the BY PERIOD button on the card.

 

 

AM/PM

Select AM/PM and the Student reports - Monthly attendance screen will appear.

 

 

Select a print order from the header drop down list. Alphabet is the default but you can also choose Class, Group, Grade, Excursion, Homeroom, Defined character, and Advisor group. See Student reports common features for more Print order information.

 

Include students registered - Select the registered type, both this year and next year. This year only (not returning), Next year only or Withdrawn (no longer enrolled this year).

 

Attendance codes - Note: This setup option only appears when the report is run for the students with the morning/afternoon attendance method.

 

Use school attendance codes:  This will show the exact attendance code (A –Z) assigned to the student.

 

Use official attendance codes: This will show official codes only (e.g. S – Sick recorded for a student will show as A – Absent). Each school attendance code has an equivalent official code to indicate whether the code is considered Present, Absent, Late, Suspended, or Grant day (for unexpected school closures due to weather, infrastructure issues).

 

Note: By period students may not have this option available if the report is converting the attendance to AM/PM as explained in ‘By period’ attendance method – conversion to AM/PM.

Select from the following options to determine which additional information is included to the right of the absence information:

 

Lates: This option will show the number of lates the student had for the month, and the accumulated number of lates since the start of the school year. If a student is late in the morning and late in the afternoon, it is counted as 2 times late.

Presence as a % of possible days: This option will convert the student’s attendance into percentage of time the student was present at school for the month and the accumulated (Acc.) total for the school year.

  

In the example below, the 1st student was absent 0 days out of 9 possible days (0/9 in the 1st Tot. column), and therefore the student was present 9/9 days or 100% of the time (the 2nd Tot. column). Since the start of the school year, the student was absent 3 out of 40 possible days. Therefore, the student was present 37 of the 40 days (40 -3). This is represented as 93% present (37/40 x 100) in the 2nd Acc. Column.
