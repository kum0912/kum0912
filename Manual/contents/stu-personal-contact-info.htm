Overview

The Personal menu option in a student’s record contains student-specific information, i.e. information that is the same regardless of what school the student attends. Only the school that is the record owner can edit this information.

 

Choose Personal > Contact info from the submenu.

 

 

 

The student’s personal contact info is grouped and presented on separate cards titled Phone, Email, Address, and Contact details.  

 

Note that this contact info menu option is for contacting the student, not the information of the student’s parents/guardians and other contact people associated with the student. That other contact information is located under the Family & contacts menu.

 

The student’s information is grouped and presented on separate cards. The fastest way to make minor changes to the student's contact info is to click the select the info that needs editing (e.g. click on the phone number to access a quick link to go directly into the edit of that phone number).

 

 

 If many changes are needed, it may be easiest to click any of the edit icons on the cards or the edit icon at the top showing the student's name and an edit icon.

 

 

Selecting any of the edit icons calls up a tabbed navigation page where you can easily move between all the different contact info to make changes. The CONTACT DETAILS tab is where you can record a date to show when the information was most recently updated and add records tracking the individual's consent over time to unsolicited communications as per Canadian anti-spam legislation.

 

 

Note: The tabs shown above correspond to the cards on the initial summary screen that appears when you first access the student's contact info.

 

Phone

The Phone card contains the student’s phone information. You can record as many phone numbers as you like.  Each phone is given an order number, i.e. order 1 will appear first on the card.

 

 

Add phone

Select the add icon to see the Add phone form. You can also add a phone number from the tabbed page.

 

Here are the phone fields:

 

Area code:  Area code of the phone number.

 

Order:  The order in which the phone should appear in the list when multiple phone numbers exist. It will default the first available number, but if changed, all subsequent phones will be reordered automatically (e.g. if you add a 3rd phone number and assign it order 2, the previous phone 2 will change to phone 3).

 

Phone: The remaining portion of the phone number (e.g. not including the area code).

 

Type:  Select a description type from the drop down list (e.g. home, cell). Duplicates are permitted (e.g. student may have 2 cell numbers).

 

Ext.:  Extension number, if applicable.

 

Country code:  Country code (if applicable – e.g. overseas phone numbers).

 

Note:  optional note that may be recorded pertaining to this specific phone number.

 

Availability:  Switch between Yes or No for the following attributes that may or may not apply to this phone number:

 

Day (is the phone available to be called during the day)

Night (is the phone available to be called during the night)

Unlisted (is the phone number unlisted)

Texts (can the phone number accept text messages)

Emergency only (should the phone number be used only in emergencies)

 

Edit phone

You can edit a phone number by row clicking a phone number. If you have to edit more than 1 phone number select the edit icon in the header. The tabbed page will appear where you can select individual phone numbers to edit.

 

 

The Edit phone form will appear where you can make changes.

 

 

 

Note: You can also add a phone number by using the add FAB on the tabbed page

 

The fields on the edit form are the same as found on the add form.

 

Dial

If you have phone calling enabled, you can row click the phone number and select the Dial menu option. This will call the phone number.

 

Email

The Email card contains the student’s email information. There is no limit to the number of email addresses that may be recorded. Each email is given an order number, so that the number that is ordered as 1 will appear first on the card.

 

 

Add email

Select the add icon to see the Add email form. You can also add an email from the tabbed edit screen.

 

Here are the email fields:

 

Email:  Email address (e.g. yourname@school.ca)

 

Order:  The order in which the email should appear in the list when multiple emails exist.

 

Type:  Select a description type from the drop down list (e.g. school, personal). Duplicates are permitted (e.g. student may have 2 school emails).

 

Note:  optional note that may be recorded pertaining to this specific email.

 

Availability:  Switch between Yes or No for the following attributes that may or may not apply to this email:

 

Day (is the email address available during the day)

Night (is the email address available during the night)

 

 

Edit email

You can edit an email address by row clicking. If you have to edit more than 1 email address select the edit icon in the header. The tabbed page will appear where you can select individual email addresses to edit.

 

 

 

The Edit email form will appear where you can make changes.

 

 

 

Note: You can also add a phone number by using the add FAB on the tabbed page.

 

The fields on the edit form are the same as found on the add form.

 

Send email

If you have emailing enabled, you can row click the email and select the Email menu option. This will call up your default email program.

 

 

 

Address

The Address card contains the student’s address information. A student may have an unlimited number of addresses. A student may also have 3 different types of addresses for the following purposes:

 

  • Mailing/Physical address: This is the address where the student lives while attending school. This address type can have unique mailing and physical address information.

 

In most situations, the mailing and physical address is the same, or they can be entered as if they were the same even though the mail may be delivered to a different location such as a post office box (e.g. 123 Main Street, PO Box 3).

 

In some situations, the mailing and physical addresses cannot be combined because they require unique cities or provinces (e.g. mail is delivered to PO BOX 3, Mapletown, but the physical location of the house is 24241 Centre Rd, Silvertown).

 

  • Permanent address: This is the student’s home address outside of the school year. This address type is needed for those students who live somewhere else to attend school, and their permanent address may be in a different city, province, or country. Permanent address may use the standard address fields, or it may be switched to use the international address format which is 4 lines of text to allow the entry of an address in a format that is appropriate for the location.

 

  • Transportation address: This is the student’s bus pickup and/or drop off address(es). These addresses are only needed if the location where the student is picked up or dropped off is different from the student’s home (physical) address.

 

 

Each address has an effective date, and that date is used to determine whether the address is a past address, a current address, or a future address. If you add an address and make it effective today, it will be considered a current address. If you enter a future effective date, a nightly process is run automatically to check for addresses that need to change from future to current and current to past.

 

A student may have only 1 address for each type that is effective today.

 

 

When a student moves to a new address, be sure to use the Add icon to create a new address. See Add new address.

 

Select an address on the card to see a menu of options such as locating the address on a map or viewing all the details of the distinct address fields (there may be more address fields beyond what is summarized on the card), or edit the address (to fix a mistake).

 

 

 

Edit address

You can edit an address in one of two ways:

 

a. Row click an address from the Address card on the main Contact info page.

 

 

 

b. Select the edit icon to call up the tabbed Contact info page.

 

 

Each address has a Correct icon (inverted pencil with an eraser) rather than the regular Edit icon. This is to reinforce the idea that edits to the address are for corrections to mistakes in the original entry of the address. Edits are automatically applied to all people in the student’s contacts and siblings who share the address.

 

The Floating Action Button can also be selected to add a new address.

More commonly, a new address will be a manual entry of the new address information.

 

 

The student’s address has a number of distinct fields so that exact information can be obtained, especially important for the school’s transportation department. The fields may vary by province, and fields may be left blank if they don’t apply to the student.

 

There is a field called “First line of mailing” and it allows you to enter anything you would want to appear on the first line of a mailing label (e.g. Mr. & Mrs. Smith or To the Parent(s)/Guardian(s) of Jane Smith). This field does not apply to transportation addresses.

Transportation address type

Transportation address type is only available to students who are flagged as being transported to and/or from the school via a bus. For example, if a student is arriving to school via a bus, but is getting a drive home from a parent, the student will not have access to a drop off address.

 

Transportation address is only needed if the student is being picked up or dropped off somewhere other than the student’s house. If however, a school wants to record the student’s physical address as their transportation address, the text ‘(locked)’ will appear to disable the editing of the address fields.

Contact details

The Contact details card is for the tracking of the student’s consent to receive unsolicited electronic communication. This tracking is required as part of the Canadian anti-spam legislation (CASL).

 

Each time consent information is changed, a new record is added to track the consent type (Expressed yes, expressed no, implied consent, or unknown), along with a note (e.g. to describe how you were informed of the consent type). Each record is automatically assigned today’s date. Only one entry per date is allowed and therefore adding two records on the same date will only save the most recent record. For audit purposes, consent information recorded on previous dates may not be edited or removed.

 

 

 When a person expresses no to receiving unsolicited electronic communication, an antispam icon will appear on that person’s email card.
