Setup > Fees > Settings

Overview

 

Clicking GO will bring the user to the following interface. 

 

 

CLOSE button on the bottom bar will return the user to the setup page (landing page with all the cards).

 

 

Add/Edit the Settings

Selecting the Edit (pencil) icon will allow users to add settings or edit the information for the current settings.

 

 

The Settings fields are listed in the following order (field names are bold – do not include #s or colon), and the card title is called Edit settings.

 

1. Account limit

 

2. Use due dates: check box

 

3. Use course title if adding course fees/Use course cose if adding course fees: radio buttons

 

4. Invoice prefix: Maximum character count = 15

 

5. Starting invoice number*: Maximum character count = 25

 

6. Invoice postfix: Maximum character count = 5 

 

7. Invoice next number: Maximum character count = 5

 

8. Invoice padding: Maximum character count = 5

 

9. Receipt prefix: Maximum character count = 15

 

10. Starting receipt number*: Maximum character count = 25

 

11. Receipt postfix: Maximum character count = 5 

 

12. Receipt next number

 

13. Receipt padding

 

14. Starting credit memo number*: Maximum character count = 15

 

15. Include student number on student reports: check box

 

16. Include Ministry number on student reports: check box

 

 

Selecting SAVE will save the new entry and return the user to the setup page.

 

Selecting CANCEL will not save the new entry.
