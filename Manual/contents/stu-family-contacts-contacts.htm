Overview

The Contacts option in a student’s record contains student-specific information, i.e. information that is the same regardless of what school the student attends. Only the school that is the record owner can edit this information.

 

Choose Family & contacts > Contact info from the submenu.

 

 

This Contacts menu option is for the contact information of the student’s parents/guardians and other contact people associated with the student, not the contact information for the student. The student’s contact information is located under the Personal > Contact information menu.

 

A student may have an unlimited number of contacts, with each contact having its own contact type (a school/centrally defined list of types that describe the relationship of the contact such as Mother, Father, Doctor, etc). A student may have multiple contacts with the same type (e.g. 2 Mothers, 3 Emergency contacts). The Custody icon will appear beside those contacts with custody notes.  The contact’s custody note will be available as a tooltip when you hover over the icon.

 

 

The 'Lives with student' icon will use the student’s address to determine whether a contact shares the same address. This is automatically determined.

 

 

 

Contacts are given an order number, so that the number that is ordered as 1 will appear first on the contact list when the list is defaulted to All (contact order) as shown here:

 

 

 

 

 

Use the Add floating action button (below the list of contacts) to add a new contact. It will open the same form as editing an existing contact (explained later). When a contact is added, you may create a new contact (if the contact isn't already in the school's data for another sibling), or you may pick a contact that already exists for a related student (e.g. the contact was previously added for a sibling).

 

To see detailed contact information on the right side of the page, select a contact from the list of contacts on the left (e.g. Father in the example above).

 

You can flag a contact as the student’s legal guardian, custodian, non-custodial parent, emergency contact, and/or school closure contact. Emergency and school closure contacts are also listed in preferential order so that the school knows who to contact first in the event of an emergency or an unexpected school closure. The list of contacts may be changed to a subset of contacts by selecting the drop down arrow shown below:

 

 

You can customize the order of contact display by selecting from the drop down list.:

 

 

 

The contact’s information is grouped and presented on separate cards. The first card shows a summary.

 

 

Selecting the edit icon calls up a tabbed navigation page to edit personal information like name, marital status, contact order, contact type and picture. You can also record a date to show when the information was most recently updated.

 

 

Photos for contacts are optional but if you have one to upload, click the placeholder image and select the file from your computer/drive to upload.

 

Phone

The Phone card contains the student’s phone information. You can record as many phone numbers as you like.  Each phone is given an order number, i.e. order 1 will appear first on the card.

 

 

Add phone

Select the add icon to see the Add phone form. You can also add a phone number from the tabbed edit screen.

 

Here are the phone fields:

 

Area code:  Area code of the phone number.

 

Order:  The order in which the phone should appear in the list when multiple phone numbers exist. It will default the first available number, but if changed, all subsequent phones will be reordered automatically (e.g. if you add a 3rd phone number and assign it order 2, the previous phone 2 will change to phone 3).

 

Phone: The remaining portion of the phone number (e.g. not including the area code).

 

Type:  Select a description type from the drop down list (e.g. home, cell). Duplicates are permitted (e.g. student may have 2 cell numbers).

 

Ext.:  Extension number, if applicable.

 

Country code:  Country code (if applicable – e.g. overseas phone numbers).

 

Note:  optional note that may be recorded pertaining to this specific phone number.

 

Availability:  Switch between Yes or No for the following attributes that may or may not apply to this phone number:

 

Day (is the phone available to be called during the day)

Night (is the phone available to be called during the night)

Unlisted (is the phone number unlisted)

Texts (can the phone number accept text messages)

Emergency only (should the phone number be used only in emergencies)

 

Edit phone

You can edit a phone number by row clicking a phone number. If you have to edit more than 1 phone number select the edit icon in the header. The tabbed page will appear where you can select individual phone numbers to edit.

 

 

The Edit phone form will appear where you can make changes.

 

 

 

Note: You can also add a phone number by using the add FAB.

 

The fields on the edit form are the same as found on the add form.

 

Dial

If you have phone calling enabled, you can row click the phone number and select the Dial menu option. This will call the phone number.

 

 

 

Email

The Email card contains the contact’s email information. You can record as many email addresses as you need. Each email is given an order number, so that the number that is ordered as 1 will appear first on the card.

 

 

Add email

Select the add icon to see the Add email form. You can also add an email from the tabbed edit screen.

 

Here are the email fields:

 

Email:  Email address (e.g. yourname@school.ca)

 

Order:  The order in which the email should appear in the list when multiple emails exist.

 

Type:  Select a description type from the drop down list (e.g. school, personal). Duplicates are permitted (e.g. student may have 2 school emails).

 

Note:  optional note that may be recorded pertaining to this specific email.

 

Availability:  Switch between Yes or No for the following attributes that may or may not apply to this email:

 

Day (is the email address available during the day)

Night (is the email address available during the night)

 

 

Edit email

You can edit an email address by row clicking. If you have to edit more than 1 email address, select the edit icon in the header. The tabbed page will appear where you can select individual email addresses to edit.

 

 

The Edit email form will appear where you can make changes.

 

 

Note: You can also add a phone number by using the add FAB on the tabbed page.

 

The fields on the edit form are the same as found on the add form.

 

Send email

If you have emailing enabled, you can row click the email and select the Email menu option. This will call up your default email program.

 

 

Address

The Address card contains the contact’s address information. A contact may have an unlimited number of addresses. Each address has an effective date, and that date is used to determine whether the address is a past address, a current address, or a future address. A contact may have only 1 address that is effective today (1 current address).

 

 

When a contact moves to a new address, it is important to use the Add icon to create a new address. See Add new address.

 

Select an address on the card to see a menu of options such as locating the address on a map or viewing all the details of the distinct address fields (there may be more address fields beyond what is summarized on the card), or edit the address (to fix a mistake).

 

 

 

Edit address

You can edit an address in one of two ways:

 

a. Row click an address from the Address card on the main Contact info page.

 

 

 

b. Select the edit icon to call up the tabbed Contact info page.

 

 

Each address has a Correct icon (inverted pencil with an eraser) rather than the regular Edit icon. This is to reinforce the idea that edits to the address are for corrections to mistakes in the original entry of the address. Edits are automatically applied to all contacts and siblings who share the address.

 

 You can add a new address here by selecting the add FAB.

 

 

Add new address

Adding a new address is broken down into 3 steps.

 

Step 1 requires you to first select whether the address is going to be a new address that you will manually enter or if the address was already entered for someone else (in which case you will be selecting the address from the list of people and their addresses (the people are limited to this person's contacts). Select NEXT to continue.

 

 

Step 2 is the actual entry of the address (if manually entering a new address) or selecting the address from a list of recorded addresses used by the other contacts). Address that match the current address for the address type are excluded from the list. More commonly, a new address will be a manual entry of the new address information.

Contact details

The Contact details card allows a contact to be flagged as an emergency contact or school closure contact for the student. When either of these flags are set to Yes, the order in which the contact is to be contacted is set.

 

The contact details section also tracks the contact’s consent to receive unsolicited electronic communication. This tracking is required as part of the Canadian anti-spam legislation (CASL).

 

Each time consent information is changed, a new record is added to track the consent type (Expressed yes, expressed no, implied consent, or unknown), along with a note (e.g. to describe how you were informed of the consent type). Each record is automatically assigned today’s date. Only one entry per date is allowed and therefore adding two records on the same date will only save the most recent record. For audit purposes, consent information recorded on previous dates may not be edited or removed.

Custody

The Custody card is for recording the contact’s custodial situation with the student.

 

 

 

Edit custody

You can edit Custody details by selecting the edit icon on the main card. The Edit contact info form will appear:

 

 

For those contacts that are parents, guardians or custodians, you can fill in these fields:

 

Legal guardian: References to Parent(s)/Guardian(s) will use this flag to consider this contact as a parent.

Custodian: If the student is not living with a parent or legal guardian, a minor must be cared for by a responsible adult in Canada. This person is known as a custodian. The custodian flag is typically reserved for those contacts with which the student is living with when they live away from home, however a school or board/division may provide specific instructions on when this flag should be set to Yes.

Non-custodial parent: This option appears if the legal guardian flag is set to No.  This non-custodial parent flag is to be set to yes if  the contact is the student’s parent but has only limited access to the student. Setting this flag to Yes will allow you to select whether the parent has access to the edsembli SIS parental site, copies of the student’s report cards, or access to other student reports.

Note: A note specifically about this contact. When a note is entered here, a Custody icon will appear beside this contact. Hovering over the icon will show the note.
