Setup > Board > Student numbers


The board can define number banks to be used for each school, and enable automatic student number assignment . This student number definition is then used to generate student numbers automatically from Add new students, and transfer students from this board.  

 

By limiting student number set up to the board, we eliminate possible number overlap between the schools (creating issues when new students are added with the same student number).  There will be one bank for the entire board. .

 

School access

The board defines the numbers, and the school can only see the information.

 

Choose Setup > Board and select GO on the Student numbers card.  

 

 

The Setup - Student numbers page will appear.

 

 

 

Here you can view student numbers information (as defined by the board), shows the Length of student number, Lowest allowed number and Highest allowed number.  

 

Board access

The board defines the student number information.

 

Select the main menu icon in the left corner, then select Setup > Board. This will take you to the Board info section.  

 

 

Select GO on the Student numbers card to open the Student numbers page.

 

 

 

 

Length of student number: Select a required number between 4 and 12 from the drop down list.

Lowest allowed number: Enter a lowest number in this required field, if Assign automatically check box has been selected.

The number defined as the lowest number must contain less than or equal to the number of characters defined as the length of student number.  If the lowest value is less than the “length of student number” value, then the automatically generated number will be padded with leading zeros to achieve the required length,

Highest allowed number: Enter a highest number in this required field, if Assign automatically check box has been selected.

The number defined as the highest number must contain less than or equal to number of characters defined in the length of student number. If the highest value is less than the “length of student number” value, then the automatically generated number will be padded with leading zeros to achieve the required length.  

Assign automatically: This check box tells the program to assign student numbers automatically.

Do not reuse previously assigned numbers: This check box makes sure no previously assigned numbers are used.
