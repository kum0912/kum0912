Overview

The Class info option in a student’s record contains information related to the courses, classes and teachers involved in the student’s education during the current school year (this year) or the upcoming school year (next year).

 

Past students do not have access to the Class info menu, however their historical class information, if applicable, is available in student record options Achievement >  Completed courses or Previous years > Marks & Comments or Previous years > Attendance details. Past students have a grade last attended which is editable by selecting that field on the profile area.

 

Choose Class info > Classes. This option provides a single location where you can see all the classes in which the student is enrolled in either this year or next year.

 

 

A page will appear on the right that shows you cards for This Year and Next year.

 

This Year vs Next Year tabs

As shown below, a tab strip will appear with up to 2 options:

 

 

THIS YEAR tab will appear if the student is enrolled in ANY school in the board/division/district during the current school year.

 

NEXT YEAR will appear if the student will be enrolled in ANY school in the board/division/district in the upcoming school year. The student’s School year status at each school determines whether or not a student is considered as attending that school for this year and/or next year. The school year status field is found in the Enrolment > Student enrolment option on the School enrolment card. These are the School year statuses you'll see:

 

  • This year and next year: student is considered as enrolled in the school on the THIS YEAR and the NEXT YEAR tabs.

  • This year only: student is considered as enrolled in the school on the THIS YEAR tab.

  • Next year only: student is considered as enrolled in the school on the NEXT YEAR tab.

  • Withdrawn: although the student is no longer enrolled in the school, the THIS YEAR tab is available to see the classes that were completed or dropped, and the student’s grade info just prior to leaving the school.

 

Use the tab strip to navigate between the student’s classes for this year or next year. When both options exist for the student, it will default THIS YEAR.

Students concurrently enrolled (more than 1 school)

For the selected school year (this or next), if the student is enrolled in more than 1 school in that school year, then the select schools drop-down will appear as shown here:

 

  

It will default “All schools” to provide the most accurate picture of the student’s classes and workload because the student is attending multiple schools concurrently (at the same time).

 

In addition to the “All schools” option, the drop-down list will also include the individual school names where the student is/has/will be enrolled in for the selected school year. Select an individual school to show the classes considering only the selected school.

 

The Classes option includes these cards:

 

  • Current location
  • Grade info: (Another card will appear for the grade info for other schools, if applicable.)
  • Course workload

 

Current location

The current location card provides the student’s current location right now. It will use today’s timetable to determine where the student is located. The current computer time will be compared to the school’s period times to determine the current period. Each period in the school’s setup will be available from a tab strip. The current period will be defaulted, showing the start and end times of the period (e.g. Period 1 (8:15 AM - 9:34 AM).

 

If the student’s timetable contains at least one class in a period today, the Current location card is available to indicate where the student may be located right now by showing the class, room and teacher information for the student’s class during the current period (based on the current time compared to the school’s period times). If the student is not assigned a class in the current period, the words “Student is not in any class at this time” will appear.

 

For the selected period, you will see the class code, title of the course, room, teacher(s) and school information so you can locate the student easily, as shown in the example below.

 

 

If the student is taking a class at another school, the background will be shaded as shown in this example:

 

 

 

If the other school’s periods extend before or after this school’s period times, the tab strip will include ‘Other’ if the student is taking classes in these other times.

 

If the current time does not correspond to any specific period, the next available period will be the default (or period 1 if outside of the school day). Use the tab strip to navigate to other periods.

 

If the student does not have any classes today (based on the school’s calendar date), the Current location card will be not be available. Only classes scheduled into actual periods will be considered (period X or floater classes are excluded).

 

This card is only available if the student is attending school in the current school year, so it will be available only when accessing the THIS YEAR tab.

 

Grade info

A Grade info card will appear for this school to show the student’s grade at the school, the homeroom designation/number/teacher (if applicable), advisor group designation/number/teacher (if applicable), attendance type (full time, part time, etc), and attendance method. Grade info exists for this year and/or next year (e.g. the student can have a grade for this year which is distinct from the grade for next year, as well as distinct homerooms, and distinct advisor groups). Attendance type is only available for THIS YEAR.

 

Every student will have at least one Grade info card. If the student is enrolled at this school for the selected school year (this or next), the first card will be titled Grade info – This School Name as shown here.

 

 

If the student is enrolled at another school (or schools) for the selected school year, you'll see a Grade info – other school(s) card. This information can't be edited.

 

 

You can edit grade info at this school by selecting the Edit icon on the tool bar of the card.

 

 

Grade: The student’s grade at the school. Grade is a mandatory field. Changes to a student’s grade will be automatically recorded as an internal transfer. The internal transfer icon  on the tool bar can be used to view/edit grade internal transfers.

 

Each school has a table of grades that controls the grades available to be assigned to the students in the school. Each grade in the table is a number or a letter (or 2 letters). Letter grades can have a title as well (e.g. letter grade K may have the title Kindergarten).

 

Note: the board/division/district is responsible for the set up of grades using Setup > Board > Grades. This is also where they define the Ministry equivalent grade if needed (e.g. Alberta schools may have grades J, K, M, E, C, EC, and P which are the equivalent of grade EC as required by the province).

 

Homeroom: The student’s homeroom number, designation (a 4 digit code assigned by the school), and the homeroom teacher(s). Homeroom is an optional field (“not assigned” will appear if the student is not assigned to a homeroom).

 

Homerooms are set up by the school using Setup > Class info > Homerooms and tools exist to help automate the assignment of student homerooms for each new school year (Student tools > Homerooms)..

 

Changes to a student’s homeroom will be automatically recorded as an internal transfer. The internal transfer icon on the tool bar can be used to view/edit grade internal transfers.

 

Changes to a student’s homeroom may trigger the automatic reassignment of classes in a student’s timetable, if the homeroom is setup with class associations. This is a feature that many elementary schools enable and is described in the setup of homerooms.

 

Advisor group: The student’s advisor group name, teacher(s), and room number.  Advisor groups are similar to homerooms where a room and teacher(s) is/are assigned to the advisor group. The name of the group can be anything the school wants. Some schools use advisor groups to indicate the guidance counsellor assigned to the student. Some schools have students meet regularly in groups where the students meet with an advisory teacher to discuss general school related topics.

 

Advisor group is an optional field (“not assigned” will appear if the student is not assigned to an advisor group). They are set up by the school using Setup > Class info > Advisors and tools exist to help automate the assignment of student advisors for each new school year (Student tools > Advisors).

 

Attendance type: A field to indicate the student’s attendance at the school as to whether or not it is full time, part time, independent study, or half time*. Half time types include:

 

Half time AM: Student attends every day but for the morning only.

 

Half time PM: Student attends every day but for the afternoon only.

 

Half time alternate days: Student attends both the morning and the afternoon, but does not attend school every day. Students with this attendance type will belong to one of four possible alternate day schedule groups (e.g. Half Time Alt. Days 1, Half Time Alt. Days 2, Half Time Alt. Days 3, or Half Time Alt. Days 4). The school’s calendar is set up to indicate which school days the students with schedule number 1 attend, which days the students with schedule number 2 attend, etc. For example, the students with schedule 1 may attend every Monday, Wednesday and alternate Fridays, while the schedule 2 students attend every Tuesday, Thursday, and the opposite Fridays.

 

* Half-time is only available for schools offering pre-grade 1 (kindergarten) programs, based on the school’s table of grades.

 

Ontario schools only: The attendance type field is also located in the Enrolment register info option in the student's record. Refer to this section for additional info about this field.

 

Attendance method:  A field to indicate how many times a day a student’s daily attendance is recorded. Elementary students typically have morning/afternoon attendance recorded. Secondary students typically have attendance recorded for each period where the student is expected to show up to a class. A school with kindergarten to grade 12 students may have students with either of the 2 possible attendance methods:  Morning/afternoon or By period. The allowable attendance methods for each school is part of the school's configuration (Setup > School > School configuration).

 

 

Editing grades, homerooms and advisor groups for many students

Rather than editing students individually through student records, there are tools that will assist schools when many students require edits to their grades, homerooms, or advisor groups. For example, as the school prepares for a new school year, the students’ next year’s grades are automatically advanced (see Scheduling next year > Prepare for next year > Initialize next year). You can automatically assign homerooms and advisor groups to students using a number of different assignment options provided within the Student tools option.

Course workload

The course workload area provides a view of the student’s workload with respect to the courses they are taking this year or next year. If a student has any classes in their timetable, they will have one or more course workload cards. Depending upon the student’s timetable and the time in the school year, course workload cards may exist for the current classes the student is taking, the upcoming classes in the next semester/term, the completed classes, the dropped classes, and the unscheduled classes for next year. Each of these card types are explained in detail further below.

 

A course workload card will appear to provide a quick view of the student’s current course workload as of today’s date in the school’s calendar. Additional cards will appear where applicable to show the student’s upcoming workload (for courses that start in a future term/semester), the course workload that is now completed, and course workload for courses that the student has dropped.

 

Note: Course workload information is for informational purposes only. If you need to make timetable changes to the classes, use the student record option Class info > Timetable which presents the student’s classes in a more traditional timetable format along with all the tools to assist in making effective timetable changes.

 

Here is an example workload card for a student’s current classes:

 

 

For each class you'll see this information:

 

Class: This is the class from the student’s timetable. Class (e.g. MAT9Ab) is a combination of the course code (e.g. MAT9A in all caps) and the section letter which is a lower case letter a to z (e.g. b in this example). In the school’s classes table (e.g. the master timetable), a unique section letter is used for each unique group of students that are taught the course.

 

Title: The title of the course.

 

Teacher: The teacher or teachers assigned to the class.

 

Room: The room number where the class is taught. If a class is taught in multiple rooms, where the room changes on different days, the room will appear as M. Select the class to access the details option to see class details including specific room information.

 

Start date: The start date of the class based on the semester/term assigned to the class. If the class is continuous entry, each student has a start date which is independent from the class start date, in which case you will see the student’s unique start date.

 

End date: The end date of the class based on the last day in semester/term assigned to the class. If the class is continuous entry, each student has their own end date which is independent from the class end date, in which case the student’s unique end date is shown. End date appears when the class is completed.

 

Drop date: If the student dropped (withdrew) from the class prior to completing the class, the dropped date will appear rather than the end date. Dropped dates are recorded automatically when a class is removed from a student’s timetable. Dropped dates are also automatically recorded as the student’s exit/retirement date when a student’s Maplewood status changes to withdrawn.

 

Semester(s): The semester(s) in which this class is taught. This field will be hidden if the school does not offer semesters, or if the student is only taking classes that are scheduled for the full year.

 

Term(s): The term(s) in which this class is taught. This field will be hidden if the school does not split semesters into terms, or if the student is only taking classes that are scheduled for all terms. For example, if school has 2 semesters and 2 terms, some classes may be offered in semester 1, term 1 and other classes in semester 1, term 2. If a student is not taking any of the single term classes, the term column will be hidden.

 

School: The school in which this class is taught. This is beneficial if a student is taking classes in more than 1 school, since the workload cards consider other schools as well.

 

Each class on the workload card may be selected to see more details about the class (e.g. info about the class setup).

 

 

Flex, CTS, and continuous entry classes)

Some classes may include an icon to provide a visual indication that the class may be one of these unique class types. The icons, explained below, will provide a tool tip when the icon is hovered, and may provide additional functionality when the icon is selected as described here:

 

Flex class (Alberta only): The class is considered a flex class, which is a unique class type available for those schools offering flex periods. The flex class is scheduled into the student’s timetable and typically all students have the same flex period. Students can pick what they want to take each flex period from a wide range of offerings created by the teachers in the school (e.g. one teacher may offer extra math help, one teacher may offer badminton in the gym, and another teacher may offer a meditation class). The teacher offerings are unique for each time a flex class is held throughout the school year, and each time the student can choose something different to take. The flex class in the course workload card is really a placeholder class. The icon indicates that this class has unique flex class functionality in edsembli CONNECT for teachers and students.

CTS class (Alberta only): The class is considered the master class of a Career and Technology Studies course or a module class. Typically, the CTS master class is scheduled into the timetable in a specific period, and the student takes many modules as part of their course. The CTS module classes are not scheduled into a period.

Although only the master class appears in the course workload cards, hovering over the icon will indicate whether it is a CTS master or CTS module.

Continuous entry class: The class is considered a continuous entry class, which means the student may start and complete the course at their own pace. The student will have their own start and end times.

Here is the detailed information on each of the possible workload cards:

 

Course workload – current

The ‘current' workload card represents the classes that the student is taking right now based on the school’s current date. If a class is scheduled in the semester/term of the current date, it will be considered current.

 

If the class is continuous entry, each student has their own start and end dates which are independent from the class start and end dates. If the student’s unique start and end dates indicate that the student is still taking the class on the school’s current date, it is considered a current class.

 

Course workload – upcoming

The ‘upcoming’ workload card represents the classes that the student will be taking in the future. Based on the school’s current date, if a class is scheduled in a future semester/term, it will be considered upcoming.

 

If looking at next year’s workload (e.g. NEXT YEAR tab is selected), all classes that are currently scheduled into the student’s next year's  timetable will be considered future and therefore will appear on the upcoming workload card. Any courses that the student has requested for next year but are not yet scheduled into the student’s timetable will appear on the ‘unscheduled’ workload card described below.

 

Course workload – completed

The ‘completed’ workload card represents the classes that the student has finished. Based on the school’s current date, if a class is scheduled in a past semester/term, it will be considered completed. If the student dropped the class or withdrew from the school prior to the class’ end date, it will appear on the ‘dropped’ card as it is not considered completed.

 

If the class is continuous entry, each student has a unique start and end date independent from the class start and end dates. If the student’s unique start and end dates indicate that the student has taken the course in the past (end date is prior to the school’s current date), it is considered a completed class.

 

 

Course workload – dropped

The ‘dropped’ workload card represents the classes that the student started but did not finish. If the student dropped the class or withdrew from the school prior to the class’ end date, you'll see it here.

 

 

Course workload – unscheduled

The ‘unscheduled’ workload card may only apply when looking at next year’s workload (NEXT YEAR tab is selected). This card shows the courses the student has requested for the next school year
(as recorded in Class info > Course selections), but the school has not yet assigned the student into a class.
