Achievement > Marks & comments > Class marks & comments

Overview

 

The Class marks & comments option provides access to the mark and comment information for students in a class.  Teachers record marks and comments.  At this time, this information is ‘view only’ for the office users at this time, however edit functionality will be included in a later release.  

To access this option, select the Menu icon in the left corner, then select Achievement, then Marks & comments.

 

This will take you to the Marks and comments section where you will find the Class marks & comments card.

 

 

Select the GO button to access the Class marks & comments interface.

The presentation of the marks and comments is driven by the report card setup, and report card setups can vary drastically between different schools and grade levels.  The board/division or school office is responsible for this setup so the terms and topics (columns) should be familiar.

Each class has its own set of terms, and each term is given a unique name. For example, the first report card period might be called Term 1, or November, or Report 1.  Each class also has its own set of topics (columns) and if needed, subtopics.  For example, an elementary school may have a topic called English with subtopics corresponding to curricular outcomes or generalized into subtopics such as Reading, Writing, Visual & Oral Communication.  Many high schools classes may simply have one topic called Grade or Mark.

The features of Class marks & comments is best illustrated through examples.

The last class you viewed will be defaulted, and the class, title, and teacher(s) will appear in the tool bar as shown here:  

Use the class filter to change to a different class.  If easier, change the teacher filter first to reduce the list of classes to just those classes assigned to that specific teacher.

 

Use the term filter to change the display to show the students’ marks and comments for a different report card term, and use the topic filter to reduce the topics/subtopics displayed .

 

If the teacher has set up markbooks to record classroom assessments for a topic, and if you have access to this markbook information, use the date filters to limit the markbook information to just the entries in the date range.  This date range will be defaulted to the last date range used and it will be retained as you move between classes.

 

 

If markbook information is available, then an additional + or – icon will appear beside the topic (called Mark in this example).

Scrolling to the left will present more markbook information, and you may expand/collapse sections as needed.

Example 2 – Topics, subtopics and comments

This example is showing 2 topics. The second topic (Language Arts 5) has subtopics corresponding to curricular outcomes.  When a topic has sub-topics, there is no mark/comment information at the topic level, but marks and comments are available for the subtopics.  

 

This setup has a topic specifically for the comment (called Subject Comment). The comment icon will appear if the teacher has entered a comment:

 

Select the comment icon to see the comment, formatted according to the dimensions defined for the report card.

  

If a comment is too long, the comment icon will include an exclamation mark, and the comment text would appear in red:

 

The setup also defines what type of marks are allowed for the term/topic.  Marks may be numeric and/or letters.  When letters are used, hover over the letter to see the meaning of that letter as shown here:

 

Mark cells that are grey, indicate that the cell is locked to prevent the teachers from making further changes to the marks for that term.  

 

The mark cell will appear with a white background if the term is unlocked (available for mark/comment entry).  No mark will appear if the teacher has not yet entered marks.

 

Example 3 -  Topic Program Codes (e.g., IEP, IPP, ESL etc)

Topic program codes are part of the report card setup to identify students with unique or alternate programs such as IEP or IPP or AP, ESL or EAL, or any other codes that the school/board/division requires.  If the program code applies to the student, it will appear in the column with the topic/subtopic where it applies.  In this example, the topic program code IEP has been setup as a possible code that can be applied to students in the language arts 5 topic.  This student has been flagged as having the IEP code for term 1:

  

More options

There are additional options that may be used to change the display of the class mark information.  Use the More icon in the class’ card to access these additional features.

 

This will present the additional options as shown here:

 

Select ‘Mark display’ to see formatting options for the marks.  The student’s marks will re-format automatically as you change between the different options.  Click anywhere outside the more options area to close it.

If the board/division/school has defined colours for each letter grade, and letter grades exist for the student, the ‘Display letter colour coding’ will cause the mark cells to turn the appropriate colour when a specific letter grade is recorded.
