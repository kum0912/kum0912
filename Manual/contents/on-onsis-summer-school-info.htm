OnSIS > Summer school > Summer school info

 

 

The Summer school info option appears for schools that are configured as summer schools. This configuration is done using Setup > School > Configuration. This option is used to report all classes that appear on the continuing education registers that may apply to summer schools:

 

  • Summer School Register
  • Register of Enrolment in Correspondence/Self-Study/E-Learning Courses
  • Register of Enrolment in the International & Indigenous Languages Elementary (IILE) Program
  • Register of Enrolment in the Literacy and Numeracy Program
  • Register of Enrolment in Adult Indigenous Language Instruction

 

This option is used to create batch files, using the CREATE button. The batch file is uploaded to OnSIS through the OnSIS web site.  Once OnSIS processes the batch file, a results file is created. The results file is downloaded from OnSIS and is then processed using the PROCESS RESULTS button.  

 

This section describes the creation of the batch files. See the OnSIS Important concepts section for info on Updating data in OnSIS using the ‘Create/Upload/Process’ cycle.  

 

There are 3 OnSIS batch files available for summer schools. Follow the links below for the file specifications:

Select the CREATE button on the Summer school info card to see this form:

 

  

The OnSIS batch file options are detailed below. Prior to selecting the CREATE BATCH FILE option, determine whether other options on the form should be selected.   

 

Purge:  Y/N field. See Important concepts – Purge check boxes for more info.

 

Save results to CSV file (aka Also create CSV file): Y/N field. If set to Yes, the CREATE BATCH FILE option will create the requested batch file and the contents of the batch file will also appear in a comma delimited file (same file name as the XML batch file but with a CSV extension). Some boards/schools generate these csv files to perform data analysis on the information sent to OnSIS.

 

Adult day school – Exclude attendance totals: Y/N field. Report card attendance data is reported to OnSIS for publicly funded secondary schools. If this summer school is considered an adult day school, then set this option to Y. If this is set to Yes, the following information is omitted from the OnSIS batch file: Total classes, Classes Missed, Times Late, IEP and ESL/ELD info.  

 

 

 

Student class enrolment batch file

 

OnSIS is collecting enrolment information about the students enrolled in summer schools,  along with information regarding any credits attempted/earned during the academic school year.   Failed courses in the students' transcript are collected by OnSIS, along with withdrawn courses that are entered in the transcript as per full disclosure requirements. Any course not completed where the student did attend class, and therefore an ADE was generated will also be included in the OnSIS submission (if the ADE was generated within the submission period)

 

The setup of the school and the student’s timetables needs to be accurate for this batch file. Check the following:

 

  • Calendar start and end dates of each semester/term are accurate (Setup > School > Calendar).
  • Classes have accurate schedules (semester/term/day/period info) and start/end dates (Setup > Class info > Classes).
  • Students timetables are accurate (student’s record > Class info > Timetable). The timetable should include all of the student’s classes. The date the student started each class, and if applicable, the date they withdrew from a class should be accurate. Hint: From the student’s record, us the Class info > Classes option to see the student’s start/end dates for each class.  If they need editing, from the timetable option, use the More icon to access the Register option (register of add/drop dates).
  • Students should have a graduation requirement assigned (student’s record > Achievement > Graduation requirements). This is used to determine whether a completed course is compulsory for graduation.

 

The students' transcripts (completed courses) must be updated with the final marks and credits earned (see Add completed courses for info on this process). The info in the student’s completed courses (student record > Achievement > Completed courses) is where the final marks and earned credit information is recorded for use in this file.

 

This batch file will create the Student class enrolment batch file just like any regular secondary school, however it does not include the 'other credit' fields that are included in regular school batch files. Follow the link below for detailed file specifications from the School info section for regular schools.

Non-Credit ADE - Indigenous & International Language Elementary (IILE) batch file

This file includes those students taking classes that appear on the official Register of Enrolment in the International & Indigenous Languages Elementary (IILE) Program. If a class is to appear in this batch file, the setup of the class (Setup > Class info > Classes) requires the following value in the course offering type field:  

 

  • International & Indigenous Languages Elementary (IILE)

 

Please see the enrolment register instructions for International & Indigenous Language Elementary (IILE) Program for detailed instructions on how to set up these classes and students. It also includes info on how ADE is calculated.

 

This batch file will create the Non-Credit ADE - Indigenous & Interantional Language Elementary (IILE) batch file just like a night/continuing education school.  Follow the link below for detailed file specifications from the ConEd/Night school info section.

Non-credit ADE / Other programs batch file  

This file includes those students taking classes that appear on the official Register of Enrolment in the Literacy and Numeracy Program and Register of Enrolment in Adult Indigenous Language Instruction.  

 

If a class is to appear in this batch file, the setup of the class (Setup > Class info > Classes) requires one of the following values in the course offering type field:  

 

  • Adult Indigenous Language Instruction
  • Literacy and Numeracy for Parents/Guardians
  • 7/8 Literacy & Numeracy – Adult
  • 7/8 Literacy & Numeracy – Con Ed
  • 7/8 Literacy & Numeracy – Fully High Credit
  • 7/8 Remedial Literacy & Numeracy
  • 9/10 Literacy & Numeracy – Adult
  • 9/10 Literacy & Numeracy – Con Ed.
  • 9/10 Literacy & Numeracy – Fully High Credit
  • 9/10 Remedial Non-Credit Literacy & Numeracy

 

Please see the enrolment register instructions for detailed info on how to set up these classes and students (and parents enrolled in literacy & numeracy programs). It also includes info on how ADE is calculated.

 

This batch file will create the Non-Credit ADE - Other programs batch file just like a night/continuing education school.  Follow the link below for detailed file specifications from the ConEd/Night school info section.

 

Non-credit ADE / Other programs
