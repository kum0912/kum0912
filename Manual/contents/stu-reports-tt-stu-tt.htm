Student timetable

The student’s timetable may consider the student’s data for this year or for next year. A page break is automatically inserted between each student. You can print the students’ timetables for the whole school alphabetically, or by grade, homeroom, or a number of additional orders. Please note, the sort order is only determining who is printed, as the same timetable is printed for the student regardless of the sort order.

 

There are a number of setup options presented on the initial page when this option is selected. These options include selecting who is to be printed, as well as whether the timetable will print portrait or landscape, semesters print down or across, what information is to print for each block in the timetable and what semester(s) and term(s) are to be printed.

 

To create an Student timetable report choose Student reports > Timetable and select THIS YEAR or NEXT YEAR on the Student timetable card.

 

 

You can also choose Scheduling next year > Print timetables > and select GO on the Student timetables card.

 

 

The Student reports - Timetables screen (this year or next year depending on what you selected) will appear.

 

 

 

Select a print order from the header drop down list. Alphabet is the default but you can also choose Class, Group, Grade, Excursion, Homeroom, Defined character, and Advisor group. See Student reports common features for more Print order information.

 

Include students registered - Select the registered type, both this year and next year. This year only (not returning), Next year only or Withdrawn (no longer enrolled this year).

 

Semester and term - select the Semester and term from the drop down lists.

 

 

OPTIONS tab

 

 

General/layout - Select Portrait or landscape and enter the number of copies to print. For Semesters and Terms specify if you want information printed across or down.

 

Contents -  You can include a Class title. If you check this option, you will need to choose With class or In legend.

In the Other section you can choose these additional options: Print course number, Print date in footer, Print grid lines, Print teacher name, Break on semester and Print adds and drops.

If you choose to include Homeroom information then select a Before period from the drop down, and a Start and End time as options.

 

Font - For each area, you may select a unique font name, font style (e.g. regular, bold, italic, etc.), and font size. Fonts installed on the server are available for selection. This is also where the date in the footer may be turned off or set to the operating system or school date.

 

Student information

 

 

Personal - For Surname options select Given or Usual name. Under Other you can select any of the following information: Gender, Grade, Homeroom, Advisor group and Telephone number.

 

ID numbers - Select student number options as required.

 

Locker - Specify locker information to include: Locker, Type of locker, Locker bank, Locker serial number, Locker combination and Locker brand.
