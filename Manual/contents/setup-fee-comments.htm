Setup > Fees > Comments

 

The various areas within the fees functionality include the ability to optionally record comments.  Comments are available when recording student fees, payments and refunds, and when recording other school funds. Comments can always be entered uniquely for each transaction, but, if the school sets up a list of comments, you can select a comment from this list. The selected comment will copy to the comment field so that the user make additional edits if needed  (e.g. comment “Payment received by:     Thank you” may be copied then the user can add their name to the comment).

To access the list of comments, select the Menu icon in the left corner, then select either:

Setup, then Fees setup, or

Fees, then Setup.

Editing a comment

Select the comment from the list to access the Edit option.

 

Then proceed to edit the comment as shown below:

 

Select the SAVE button to save the changes.  While editing a comment, you can navigate to the next or previous comment in the list using the NEXT  COMMENT and PREVIOUS COMMENT buttons at the bottom of the page.  Selecting any of the navigation buttons at the bottom of the page will save any edits to the comment.

Note:  Selecting a comment from the list provides access to the Details option as well as the Edit option.  The Details option is standard on all tables, providing a method to view all the fields in the edit form in one convenient display.  

Removing a comment

If a comment is to be removed, select the comment from the list to access the Edit option.  From the Edit comment tool bar, select the Discard icon.
