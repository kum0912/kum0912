Void fees

The Void option can be used to delete fees. Voided fees will not appear on the student’s record or any reports. edsembli keeps an audit trail, and records this in the database for audit purposes.

There are two ways that you can void a student fee:

  1. Row click and select Edit fee.



    From the edit form that appears, use the Discard icon in the tool bar to remove the fee:


     
  2. Alternately, select the FEE TRANSACTIONS tab. Row click and select Edit fee.

From the edit form that appears, use the Discard icon in the tool bar to remove the fee:

Note: If the fee has a payment, when the fee is voided, the payment will be converted to a payment credit which can be refunded or applied to other fees.  See Payment credits for details.

 

Before you void a fee you will be asked to confirm by selecting PROCEED.
