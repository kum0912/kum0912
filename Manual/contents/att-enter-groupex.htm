Attendance > Enter attendance > Group/excursion

Overview

Rather than entering known attendance one student at a time, if a group of students have the same attendance information, the Group/excursion option may be used to bulk apply the attendance to the students in the group. The Group/excursion option may be useful for the following situations:

 

  • field trips
  • bus cancellations or delays
  • sport teams that are excused from class for games
  • school closures for part of the day

 

From the main menu, choose Attendance then Enter attendance.  This will take you to the Enter attendance section of the Attendance page where you will find the Group/excursion card:

 

 

Click GO to access the Group/excursion page.

Group/excursion

Follow these steps to apply attendance to a group of students:

 

  1. Check that the date is set correctly. It will default today’s date, but a date range may be set.
  2. Choose the group of students from the Group drop down list as shown here:

 

Note:  if a date range is entered, the Change attendance options icon will be disabled, as the ability to check for previous attendance is not available and it will overwrite any previously entered attendance automatically.  For this reason, an extra dialogue message will appear warning that previous attendance entries will be replaced.  The user can decide whether to proceed with replacing the attendance or cancel.

 

 

The following groups are available:

 

Excursion group: This is the group to select if the school has excursion groups (school defined lists of students (e.g. students in band, choir, basketball, science club, etc.)). When this group is selected, an excursion field is available to select the desired excursion list. See Setup > Attendance> Excursion lists for details on creating a unique group of students.

 

All students: Use this for attendance applying to all students attending school on the date(s).

 

Grade: Use this for attendance applying to students in a specific grade. A grade field is available to select the desired grade.

 

Class: Use this for attendance applying to students in a specific class. A class field is available to select the desired class.

 

Homeroom: Use this for attendance applying to students in a specific homeroom. Select the applicable homeroom.

 

Defined character: Use this for attendance applying to students with a specific value for a specific defined character. Two fields are available:  a field to select the defined character to check (students have 4 defined characters) and a field to select the value that the student must have. Defined characters are part of the Flags area in a student’s record.

 

Class: Use this for attendance applying to students in a specific class. A class field is available to select the desired class.

 

Bus: Use this for attendance applying to students on a specific bus. A bus field is available to select the desired bus route.

-if default attendance code is needed for both the morning and afternoon, check the box beside the whole day option and then select the attendance code.

Check that you have the “Change attendance options” set correctly. This provides some control over whether or not the new attendance gets applied to students automatically. For example, if a student has a planned vacation, you may not want to overwrite the previously entered vacation attendance code with something else.

 

It will retain your preferred setting so this step may be skipped if you know that it is set correctly.

 

Select the Change attendance options icon to access the possible options:

  

 

 

Select one option corresponding to how you would like the application to apply the bulk attendance entries when the APPLY button is selected in step 5. The options are explained below.

 

Change all students automatically: The student’s attendance will be changed to the attendance specified in the bulk apply area as soon as the APPLY button is selected.

 

Don’t change if previous attendance: If the student’s attendance is currently present (e.g. no previous attendance has been entered), the student’s attendance will be changed to the attendance specified in the bulk apply area as soon as the APPLY button is selected. If the student has previous attendance, the student will not be changed.

 

Stop if previous attendance & let me decide: This is the recommended setting. If the student has no previous attendance, the student’s attendance will be changed to the attendance specified in the bulk apply area as soon as the APPLY button is selected. If the student has previous attendance, you can decide whether or not to apply the bulk attendance changes. This message will appear when attempting to apply the attendance to students with previous attendance:

 

Stop at everyone & let me decide: As soon as the APPLY button is selected in step iv, the dialogue message (as shown above) will appear regardless of whether or not the student has previous attendance.

 

  1. Click the APPLY button at the bottom of the page to apply the attendance as indicated in the previous steps. Depending upon the setting selected in step 3, you may be prompted to determine whether or not the attendance is to be applied to the student.

 

When all students are processed, the following toast message will appear:

 

image showing toast - Group/excursion attendance was successfully applied.

 

Important note: these steps will need to be repeated if the group of students contains students with both methods of attendance. If this situation applies, MORNING/AFTERNOON and BY PERIOD tabs will be visible. For example, if a K to 12 school has to apply an attendance code to all students, they will see the tabs. First they follow the steps with the focus on the MORNING/AFTERNOON tab and then select the APPLY button. Then they will follow the steps with the focus on the BY PERIOD tab and then select the APPLY button again.
