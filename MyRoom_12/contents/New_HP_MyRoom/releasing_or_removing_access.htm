Releasing or removing access to SharedRooms

Release access to a room  

 

As a user, you can release access to specific rooms in the list that you do not use.  

 

Right click on the room and select Release room access.

   
Click OK to release access to the room.

   
The room is removed from your list of available rooms.

 

 

Removing group access for a single user or all users

As the room group administrator, you can remove group access for a single user or all users associated with a group.

 

  • Right click on the group and select Delete group access.
  • Enter the email address of the user to be removed and click OK.
  • Repeat as needed.

 

You can also apply this to the whole group by ticking Remove access for all users and clicking OK.
