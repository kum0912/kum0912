Timers

Clocks that count down or up depending on how they are configured, timers are used for breakouts and lab exercises. Timers are available as part of the HPE MyRoom University and are useful for training sessions and collaborative meetings. You can use timers in HPE MyRoom to keep your meeting running smoothly and on time. Once created, timers can be dragged from one room to another to create a copy.

 

Create a timer  

 

Right click in the Content area or click on the Open content file menu icon in the Content header. Click on Add and choose Timer.

 

 

Name your timer so that you can easily locate and start it in the Content area.

 

The Message is what will show while the Timer is running. NOTE: Character limits for the message are 320 characters including spaces for a center timer and 100 characters including spaces for a left timer.

 

Enter the number of minutes for your timer.

 

You can choose for the timer to count up rather than down, for instance if you want to show minutes elapsed.

 

If you tick Auto start, the timer will start as soon as you choose it in the Content panel.

 

Set to display it in either the center area or the left panel. You can run one timer in the center area while another runs in the left panel. Click OK to save your timer.

 

 
Your timer now appears in the Content area.

 

 
You can return to the Timer setup window by right clicking the timer you wish to change and choosing Properties. You can modify all options except the display location of the timer at this time.

 

 
When you are ready to run your timer, click on it in the Content area. If it is set to Auto start, it will automatically display and start running. If you did not choose this option, use the Start button in the top right hand corner to start your timer.

 

 
Once running, the Start button changes to a Stop button. You can also close the Timer window using the icon to the right of the panel .
