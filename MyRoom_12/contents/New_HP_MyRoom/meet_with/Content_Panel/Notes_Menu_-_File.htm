Notes menu - File

The File menu is used by presenters to print and export Notes files. As a participant, you can download a note that a presenter has placed in a Shared folder by right clicking on it and choosing Export.

 

The Notes menus appear for everyone except participants as soon as a note is selected in the content panel.

 

 
File menu

 

 

Choosing Print... brings up the regular Print dialog box. Choose your printer, define your print options such as Page Range and Number of copies and click on the Print button to print the note.

 

 
The Print Preview... option allows you to see what the printed notes page will look like. Use the Close button in the top leftright hand corner to close the preview window.

 

 

To save the notes and use the information outside of HPE MyRoom, export the note in one of the formats provided. Click on Export HTML... to export in HTML format. To change the location of the saved file, click on Browse Folders, change to your desired directory and click Save. You can open the saved HTML page in any browser by navigating to the folder where it was saved and double clicking on the file.

 

 
Click on Export TXT (No formatting) to save the note as an unformatted text file. To change the location of the saved file, click on Browse Folders, change to your desired directory and click Save.

 

 
The file can be opened using other word processing applications such as Notepad.

 

 
Choose Export PDF... to create a PDF file of the Notes. To change the location of the saved file, click on Browse Folders, change to your desired directory and click Save. Your last save location is remembered.

 

 
The exported PDF file can then be opened using Adobe® Acrobat®.

 

 
Choose Close to close the note and return to the default page.
