Start page

You can set a default page so that until a presenter enters the room and chooses a content item, anyone who enters will see the start page.

 

NOTE: The following content items cannot be set as a start page: Audio/Video messages, Folders, Surveys (when set to display in left panel), Timers (when set to display in left panel), Web tours (URL type), left viewing objects, and files (objects that do not show in center stage).

 

Set a start page

 

Right click on the content item you want to use as your start page and choose Set as start page.

 

The following content items can be set as the start page:

 

Notes

Survey (when set to display in center stage)

Table

Timer (when set to display in center stage)

Task List

Whiteboard

JPG and PNG files

 

 

 

 

   
Clear start page

 

After you have set a start page, Clear start page becomes active. To change your start page, right click in the Content area and choose Clear start page. You are ready to set a new start page.
