Tables

Use tables to keep track of information. Any presenter or participant can enter information into the table.

 

Create a table  

 

Right click in the Content area or click on the Open content file menu icon in the Content header. Click on Add and select Table.

 

 
  • Your table is created with a default title of Table Mon-dd-yyyy.
  • Click the table title in the content panel to display it in the center area.

 

 

Your table has a default size of 150 rows x 10 columns and a maximum size of 5,000 rows x 5,000 columns. Column headings by default are numbers. You can change each column heading by right clicking the heading and choosing Edit Column from the menu.

 

 

Change the header to the text you want. Select the format of the column from the drop-down list, either Text (default value), Number, Choice or Date. Click OK.

 

If you have existing data in your columns and you change the type of the column, you will see a message warning you that existing data may not match your new type. Click Ok to proceed or Cancel to cancel the operation.

 

 
Setting the type to Choice opens this window. Type in your choices, up to 10, and click OK to confirm or Cancel to cancel the operation.

 

 
Your header is now changed and the column type is set. Repeat the process for as many columns as you need.

 

 

Both presenters and participants can enter and change data in table cells. You enter data or edit it by selecting the cell and typing in it for text and numeric data or tabbing or double clicking to bring up the Choice and Date drop-down menus.  

 

Tables can be exported by right clicking on the table name in the Content panel and choosing Export. You can import a table into another HPE MyRoom by right clicking in the Content panel or clicking the Open content file menu icon and choosing Import. Navigate to your saved Table location and click Upload.

 

You can also drag and drop a table directly from one MyRoom room to another.

   

You can copy and paste from Excel, Word, email and Notepad into your table.

 

You can also right click in the Content area, choose Import and select a CSV file which will be converted into a table in your content list. Drag and drop from another location will also convert a CSV file into a table.

 

 
 

 

Column widths

 

When you copy/paste into a table in MyRoom, the text entered into the columns wraps to the default column width. You can expand the column in the regular way in the table header for yourself only, it does not affect how others see the table. When you leave the room and come back, your changes revert to the default column size. This default size is constant but the amount of text you can enter depends on your resolution.

 

When you import a CSV file, the columns are sized to the size of the cells you are importing.

 

NOTE: change the format of any numbers to plain numbers without commas before saving as CSV for import to MyRoom.
