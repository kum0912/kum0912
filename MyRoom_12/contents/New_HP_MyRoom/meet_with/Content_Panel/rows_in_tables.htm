Cells, rows and columns in tables

Once your table is set up, you can modify the display of your cells, rows and columns as well as the information in those cells.

 

Cells and rows  

 

Use the right click menu in the table to sort, filter, add rows, set cell color or text color as you can see in the menu at right.  

 

Cut, copy and paste as regular using the right click menu in a row.  

 

You can insert a single row above the current row in the right click menu. Position your  cursor in the row where you want to add a row or select the row by clicking in the row header on the left hand side, right click and select Insert row.  

 

Selecting Append rows will add 50 additional rows to the bottom of your table.  

 

To delete a row, select the row by clicking in the row header on the left hand side and right click or right click on a cell in the row to be deleted and select Delete row. You can only delete one row at a time.  

 

Clear cells by selecting the cells, right click and select Clear cell(s).

 

 

Use this right click menu to highlight and organize your data, for instance with cell color and text color. Select the cells to change, right click on one of them and select Background color or Font color.

 

You can select non-contiguous cells by holding down the CTRL key as you click each cell.

 

 

Different people in a meeting can highlight or change their own rows to differentiate themselves, as in the example at right.

 

You can see who changed the data in a cell by right clicking the cell and selecting Last modified.

 

 

Filter data

 

You can filter your rows of data in the table by right clicking on the criteria to filter and selecting Filter. At right, the data has been filtered by the Range 11-15.

 

Changes made in the filtered view are stored in the main table. Click on Clear filter to display all the data again.

 

NOTE: When you filter your data, it is a local operation and is not shared with others in the room.

 

 

 

 

Columns  

 

When you right click on the column header, the right click menu is specific to columns.  

 

You can insert a single column to the left of the current column. Position your  cursor in the column header to the right of where you want to add a column, right click and select Insert column.  

 

Selecting Append column will add 25 additional columns to the right of your table.  

 

To delete a column, right click on a column header of the column or of a single cell in the column to be deleted and select Delete column.

   
Each person can enlarge and reduce the type size of the table they see on the screen using the Zoom in/Zoom out button in the table header section. Individual choice does not affect others' view of the table.
 

 

Sort data  

 

You can sort the data on the basis of the column you have chosen. Right click on the column header and select Sort. All rows are sorted on the basis of that column, depending on the data type, either alphabetically or numerically or by date. The data at right is sorted on the ID data.  

 

Click on the same column header to reverse the sort order. The sorted column is marked with an up or down arrowhead to indicate the sorting direction.  

 

Right click on the same header and select Clear Sort to return the data to the original order.  

 

NOTE: When you sort your data, it is a local operation and is not shared with others in the room.

 

 

Sum and Average data  

 

Numerical columns can be summed and averaged. Right click the column header and select Average or Sum. The result is displayed as in the example at right.  

 

Average and Sum are only available for columns with a number type.  

 

NOTE: When you Sum or Average your data, it is a local operation and is not shared with others in the room.
