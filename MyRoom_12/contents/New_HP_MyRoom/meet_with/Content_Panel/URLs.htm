URLs

You can place a URL in your Content panel and either step through it as a presenter or allow your participants to navigate it themselves.

 

Create a URL  

 

Right click in the Content area or click on the Open content file menu icon in the Content header. Click on Add and choose URL.

 

 

Name your URL so that you can easily locate and start it in the Content area.

 

Enter the web address URL you want to launch with this item.

 

Choose to either launch the URL automatically when you select the item  on your participant windows so it can be navigated by them individually or automatically as a Web Tour where the presenter launches the browser for everyone and can control the navigation in the web browser.

 

Click OK to save your URL.

 

 

 
Your URL is now listed in the Content area.

 

 
Click on the URL item in your Content area to see the options for running the URL. You can launch the URL only for yourself, launch it for all where each person will navigate the URL themselves or launch it as a Web tour where you control the navigation on all the screens.

 

 

If a URL is set up or launched as a Web Tour, once the presenter chooses the URL in the Content area, a Shared region window opens on each participant's screen showing the browser window of the presenter. The presenter can then navigate in the browser as in the example at right and all participants see the results.  

 

NOTE: For Mac and Windows using Safari, share your desktop or region to show a URL rather than using a Web Tour URL.   

 

If you have set up the URL link as an Independent link, clicking on it will launch the link in a web browser on each participant's screen so they can navigate independently.
