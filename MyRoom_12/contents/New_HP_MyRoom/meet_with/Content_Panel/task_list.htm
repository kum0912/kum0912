Task list

As a presenter with the MyRoom Assist subscription for your account, you can create a Task list for step-by-step instructions when you are supporting clients or giving a demonstration. Each person in the room can navigate through the Task list independently once the presenter launches the Task list by clicking it in the Content panel.

 

Create a Task list

 

Right click in the Content area or click on the Open content file menu icon in the Content header. Click on Add and choose Task list.

 

 

A Task list is created with the generic name Task List Mon-DD-YYYY.

 

Click on the Task list you've created in the Content panel.

 

Your new blank Task list is displayed.

 

Click on the edit button to begin creating your task list.

 

 

The Edit task list window is displayed in the center viewing area.

Click the Add step button to begin your list.

 

 

The first Task appears.

 

 

Enter the title and description for this step.

 

Click the Add task button to continue to the next task in your list.

 

 

Continue until you have completed all your tasks in the list.

 

NOTE: The description can contain URLs like www.hpe.com.

 

NOTE: You can use HTML code to add interest to your task list.

 

 

You can add images to each task by dragging the image to the space provided, Drop image here (optional)

 

 

Click on the Back button when you have finished editing your task list.

 

 

When you select the Task list in the Content panel, this is what those on a computer see.

Use the Next task and Previous task buttons to move through the tasks. Each person in the room moves through the tasks independently.

 

 

Once the Task list has been selected in the Content panel, this is what a VRG user sees on the device.

Use the Next task and Previous task buttons to move through the tasks. Each person in the room moves through the tasks independently.
