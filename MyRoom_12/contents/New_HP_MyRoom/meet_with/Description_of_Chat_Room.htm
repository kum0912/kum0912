Description of chat room

The HPE MyRoom chat room can be used for meetings or chats, allowing collaboration, written chat, file transfer and sharing, use of web cameras and microphones as well as desktop sharing. It is a dynamic and powerful tool for small meetings and groups to use. The chat appears in the Center viewing area until a content object is selected. By default, the People and Content panels are not open: go to the View menu and click People panel to display the People panel and Content panel to display the Content panel, as below.

Call icon

Available by default. Click to launch audio communication with another person in the Chat room. Asks permissions to start audio. See the Start a chat with audio page for details.

Audio/Video controls

Once a call is answered or there are more than two people in a chat, these three icons appear. Your speakers are enabled by default. The microphone is on by default in a call but disabled by default with more than one other person. The webcam is disabled by default when you enter a chat. See the Start a chat with audio page for more details.

Settings icon

Adjust personal settings in HPE MyRoom account. See the Settings page for more information.
Message entry area Once the Chat panel is displayed, after the first message is typed, this message disappears. There are a maximum of 10,000 chat entries allowed before chat history must be deleted.
