Remote control and Windows 8

In Windows 8, with User Account Control on, the remote Operator/Agent cannot run Computer Tools, Administrative Tools, Task Manager and Device Manager. The windows are open but the Operator/Agent cannot navigate a tree, switch tabs, etc. To resolve this issue, the client must run the MyRoom standard client as an administrator or run the pre-installed MyRoom version on a controlled system.

 

  • To run as administrator, start MyRoom by right clicking on the desktop icon.
  • Choose Run as administrator.

 

 
  • Click Yes when the UAC dialog appears.

 

 
If the client is on a controlled machine, the client's IT department should install MyRoom as administrator on machines that will use it before the Operator/Agent starts a session or meeting with the client.
