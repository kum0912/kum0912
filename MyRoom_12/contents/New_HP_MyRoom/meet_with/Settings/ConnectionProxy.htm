Connection

If you use a firewall or proxy, enter a user name and password in the Connection window. In that case, untick the box next to Let MyRoom find its own connection method and additional information will be requested.   If necessary, enter the Proxy address and Port required to access the Internet. If these are unknown, consult the local Network Administrators.   Ticking the box next to Let MyRoom find its own connection method again will remove these fields from view.   Click on Save to save any changes.   Close the window to stop making changes.
