Extensions

This tab was available for some versions of MyRoom previous to 12.7 but the installation is no longer available. This page is provided as a help for those with versions previous to 12.7.

 

This tab was only available for 64-bit Windows OS and did not display for 32-bit operating systems.

 

Users of versions prior to 12.6 who do not have the MP4 Recorder extention will not be able to install it. Users of  12.6 with the extension installed will still be able to use it but if it becomes uninstalled, you will need to upgrade to version 12.7 or higher to generate MP4 recordings.

 

Click Install to begin installation of the MP4 Recorder extension.

    
 

 
Click Uninstall if you want to remove the extension.
