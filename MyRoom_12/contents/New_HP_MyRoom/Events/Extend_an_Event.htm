Extend an event

As an event ends, any Presenter can extend the event for 15 minutes, unless there is a conflicting event scheduled in the room. Large Events can only be extended once.

 

Extend an event  

 

Near the end of an event, a warning message pops up for every Presenter in the event. To extend the event by 15 minutes, click Yes.

 

 

A confirmation message regarding the extension of the event appears for all Presenters. Click OK to close the window. Note that if any Presenter clicks Yes, a confirmation window appears to all other Presenters.

 

 
For standard events, this warning message will continue to appear every 15 minutes, as long as there is no conflicting event already scheduled in the same room, until no one clicks Yes. Then the event ends.   Note that for Large Events, the event can only be extended once for 15 minutes.
