Modify an event

Modifications of Event name, Start time and End time can be made without affecting the keys already created for the event.

 

Modify an event  

 

To modify an event, right click on the name of the room in the Rooms tab and choose Events or click on the room in the Rooms tab and click the Events button at the bottom of the tab.

 

 
The Events window opens with the name of the room you have chosen in the title bar. Click on an event to display the details of the event. The current details for the event are displayed. Make any modifications that you want to Event name, Teleconference or other information, Start time and End time and click Save to save these changes.
