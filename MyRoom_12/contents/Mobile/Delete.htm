Eraser

You can use the Edit button to work with the displayed content item.

 

Tap the Edit button to display the menu. Tap the pointer and select the annotation you want to erase.

 

 

 

 

 

 

 

The Eraser deletes one selected annotation at a time. To delete all annotations at once, use the Delete option explained here.

 

Tap the eraser. The selected annotation disappears immediately from the content area.

 

NOTE: Be careful when using the eraser. There is no confirmation message and you cannot undo the deletion.
