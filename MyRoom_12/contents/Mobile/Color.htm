Color palette

You can use the Edit button to work with the displayed content item.

 

Tap the Edit button to display the menu. Tap the Color palette to choose a new color for your tools.

 

 

 

 

 

You see the color palette available. Scroll down to see all colors and tap the color you want. The Color button background changes to the color you have chosen.

 

 

The next annotation you create will use the new color you have chosen.

 

 

Tap the Color palette button again to close the color menu.
