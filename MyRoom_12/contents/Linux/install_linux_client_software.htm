Install Ubuntu client software

  1. The Download page automatically presents the Linux download window. Click the link for your package.

   
  1. Your download will start. Once complete, click on the Download progress arrow and choose the package to open it.

 
   
  1. Choose Save File to save and then launch using Ubuntu Software Center (in this case) or Open with... to start installing right away.

   
  1. Click on Install or Upgrade to begin package installation.

   
  1. Authenticate with your password.

   
  1. Once installation has finished, you can launch HPE MyRoom from Dash Home.

   
More detail for Ubuntu installs

Installing .DEB on Ubuntu (prior to 17.04)

 

#Download .deb package from MyRoom Download page

 

a) #Install with dpkg -i command

sudo dpkg -i myroom_<version>_amd64.deb

#Force dependencies to resolve, subsequently completing the install

sudo apt-get -f install

 

OR

 

b) #Install with gdebi command

    sudo gdebi  myroom_<version>_amd64.deb

  

Installing .DEB on Ubuntu 17.04

 

#Install the libpng12-0 version from Xenial

#Edit the sources list

Sudo nano /etc/apt/sources.list

#Append the following line to the end of the file; save with CTRL+O, exit with CTRL+X

deb https://us.archive.ubuntu.com/ubuntu xenial main

#Update Apt with new repository

sudo apt-get update

#Continue with install

#Download .deb package from MyRoom Download page

sudo dpkg -i myroom_<version>_amd64.deb

#Force dependencies to resolve, subsequently completing the install

sudo apt-get -f install
   
Missing libraries errors

If you get a missing libraries error while processing 'hpmyroom', run 'apt-get -f install' to resolve all missing dependencies.
NOTE: PulseAudio is required for Linux systems using MyRoom version 10.9 and higher. Go to the Sound settings on your Linux system. Explicitly select your audio device in both the Output and Input tabs. Verify your microphone volume is not set too low or muted. Test your device in Sound settings. Then proceed to test and adjust your settings in MyRoom as here.
