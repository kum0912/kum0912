Install RHEL client software

test

  1. The Download page automatically presents the Linux download window. Click the link for your package.

   
  1. Choose Save File to save and then launch using Package Installer (in this case) or Open with... to start installing right away.

   
  1. Click on Install to begin package installation.

   
  1. Authenticate with your password.

   
  1. Once installation has finished, you can launch HPE MyRoom from the Applications menu list.

   
More detail for RHEL 7, CentOS 7 and Fedora installs

• Installing the .rpm on RHEL 7

# install EPEL

sudo subscription-manager repos --enable=rhel-7-server-optional-rpms

sudo subscription-manager repos --enable=rhel-7-server-extras-rpms

sudo yum install -y https://download.fedoraproject.org/pub/epel/7/x86_64/e/epel-release-7-9.noarch.rpm

 

#Download .rpm package from the MyRoom Download page

# Install MyRoom

sudo yum install -y hpmyroom<version>.x86_64.rpm

 

• Installing the .rpm on CentOS 7

# Install EPEL

sudo yum install -y epel-release

 

#Download .rpm package from the MyRoom Download page

# Install MyRoom

sudo yum install -y hpmyroom<version>.x86_64.rpm

 

• Installing the .rpm on Fedora

sudo dnf install -y hpmyroom<version>.x86_64.rpm

 

Installing the .rpm and dependencies manually

sudo rpm -Uvh \

  https://dl.fedoraproject.org/pub/epel/7/x86_64/l/libva-1.2.1-3.el7.x86_64.rpm \

  https://dl.fedoraproject.org/pub/epel/7/x86_64/l/libvdpau-0.9-1.el7.x86_64.rpm \

  https://mirror.centos.org/centos/7/os/x86_64/Packages/libpng12-1.2.50-10.el7.x86_64.rpm \

  /downloadfiles/hpmyroom<version>.x86_64.rpm
   
Missing libraries errors
  1. If you get a missing libraries error while processing 'hpmyroom', run 'yum whatprovides */<libraryname>' for each individual library to find a package that contains the library.
  2. Install the package by running 'yum install <packagename.rpm>'  (where 'packagename.rpm' is an installer name).
   
NOTE: PulseAudio is required for Linux systems using MyRoom version 10.9 and higher. Go to the Sound settings on your Linux system. Explicitly select your audio device in both the Output and Input tabs. Verify your microphone volume is not set too low or muted. Test your device in Sound settings. Then proceed to test and adjust your settings in MyRoom as here.
