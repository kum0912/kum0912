Linux 64 bit installation

For a first time installation on a new Linux 64 bit machine, please follow the steps below according to your distribution.

 

Ubuntu

 

 
  1. Navigate to /Downloads directory in a terminal window.
  2. Run 'dpkg -i<packagename.deb>' (where 'packagename.deb' is an installer name).
  3. If you get a missing libraries error while processing 'hpmyroom', run 'apt-get -f install' to resolve all missing dependencies.
   
Once installation has finished, you can now launch HPE MyRoom from Dash Home.

   

RHEL
  1. Navigate to /Downloads directory in a terminal window.
  2. Run 'yum install <packagename.rpm>' (where 'packagename.rpm' is an installer name)
  3. If you get a missing libraries error while processing 'hpmyroom', run 'yum whatprovides */<libraryname>' for each individual library to find a package that contains the library.
  4. Install the package by running 'yum install <packagename>'.
   

Once installation has finished, you can now launch HPE MyRoom.
